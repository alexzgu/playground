# Casella & Berger material — legacy, unverified text

This directory contains two unpaginated plain-text derivatives covering nominal
Chapters 1–10 and a contents extract covering Chapters 1–5. The exact source PDF
and edition are not present in the repository, so these files cannot currently be
certified as faithful page-by-page transcripts.

Known audit findings:

- `Ch1to5.txt` begins with an exact duplicate of its first 423 lines.
- Neither chapter file has source-page markers.
- Figures generally survive only as captions or references, without a visual
  transcription.
- Literal `||` extraction artifacts occur in both chapter files.
- The Chapter 8 heading is malformed/indented, and the contents extract stops at
  Chapter 5.

Treat these files as legacy reference text, not verified curriculum transcripts.
To complete this material, provide the exact Casella–Berger edition PDF; it can
then be fingerprinted, rendered, transcribed page by page, and checked with the
same image-plus-text audit used for the other books.
