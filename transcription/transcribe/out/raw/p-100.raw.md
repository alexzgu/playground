CLAIM c1: The linear and quadratic Taylor coefficients of $y\nu - n\log(1+e^{\nu})$ about $\hat{\nu}$ are $y - ne^{\hat\nu}/(1+e^{\hat\nu})$ and $-\tfrac12 n e^{\hat\nu}/(1+e^{\hat\nu})^2$.
```python
from sympy import *

nu, nuhat = symbols('nu nuhat', real=True)
y, n = symbols('y n', positive=True)

f = y*nu - n*log(1 + exp(nu))

d1 = diff(f, nu).subs(nu, nuhat)
d2 = diff(f, nu, 2).subs(nu, nuhat)

assert simplify(d1 - (y - n*exp(nuhat)/(1 + exp(nuhat)))) == 0
assert simplify(Rational(1, 2)*d2 - (-Rational(1, 2)*n*exp(nuhat)/(1 + exp(nuhat))**2)) == 0
```

CLAIM c2: Completing the square: $\frac{1}{s}(\nu-\hat\nu)^2+\frac{1}{d}(\nu-\theta)^2 = (\frac1s+\frac1d)\left(\nu-\frac{\hat\nu/s+\theta/d}{1/s+1/d}\right)^2+\frac{(1/s)(1/d)}{1/s+1/d}(\theta-\hat\nu)^2$.
```python
from sympy import *

nu, nuhat, theta = symbols('nu nuhat theta', real=True)
s, d = symbols('s d', positive=True)   # s = sigmahat^2, d = delta^2

lhs = (nu - nuhat)**2/s + (nu - theta)**2/d
A = 1/s + 1/d
rhs = A*(nu - (nuhat/s + theta/d)/A)**2 + ((1/s)*(1/d)/A)*(theta - nuhat)**2

assert simplify(expand(lhs - rhs)) == 0
```

CLAIM c4: With $\lambda=d/(d+s)$: $\frac{1}{s}(\nu-\hat\nu)^2+\frac{1}{d}(\nu-\theta)^2=\frac{1}{\lambda s}\big(\nu-(\lambda\hat\nu+(1-\lambda)\theta)\big)^2+\frac{\lambda}{d}(\theta-\hat\nu)^2$.
```python
from sympy import *

nu, nuhat, theta = symbols('nu nuhat theta', real=True)
s, d = symbols('s d', positive=True)

lam = d/(d + s)
lhs = (nu - nuhat)**2/s + (nu - theta)**2/d
rhs = (nu - (lam*nuhat + (1 - lam)*theta))**2/(lam*s) + (lam/d)*(theta - nuhat)**2

assert simplify(expand(lhs - rhs)) == 0
```

CLAIM c5: The conditional variance $\lambda\hat\sigma^2$ equals the stated $(1-\lambda)\delta^2$, and the stated mean matches the precision-weighted mean.
```python
from sympy import *

nuhat, theta = symbols('nuhat theta', real=True)
s, d = symbols('s d', positive=True)

lam = d/(d + s)

# variance: 1/(lam*s) is the exponent precision  ->  variance lam*s
assert simplify(lam*s - (1 - lam)*d) == 0

# mean: precision-weighted mean equals lam*nuhat + (1-lam)*theta
A = 1/s + 1/d
assert simplify((nuhat/s + theta/d)/A - (lam*nuhat + (1 - lam)*theta)) == 0
```
