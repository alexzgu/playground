**8.3.3 More examples**

1. Bayesian Predictive Inference

$$\underbrace{y_1, \dots, y_n}_{y_s}\ \underbrace{y_{n+1}, \dots, y_N}_{y_{ns}} \quad \sim (\mu, \sigma^2)$$

*(The booklet writes $y_s$ and $y_{ns}$ with under-tildes to indicate vectors.)*

$$p(\mu, \sigma^2) \quad \propto \quad \frac{1}{\sigma^2}$$

Finite population of size $N$ is a (random) sample from an infinite population (super population). Here $y_1, \dots, y_n$ are a random sample from a finite population. Interest:

$$\bar{Y} = \sum_{i=1}^{N} \frac{y_i}{N}\ \text{ finite population mean} = f\bar{y}_s + (1-f)\bar{y}_{ns}$$

> ⏳ VERIFY[c3]: with $f = n/N$, $\bar{y}_s = \sum_{i=1}^n y_i/n$ and $\bar{y}_{ns} = \sum_{i=n+1}^N y_i/(N-n)$, the identity $\sum_{i=1}^N y_i/N = f\bar{y}_s + (1-f)\bar{y}_{ns}$ holds.

where $f = \frac{n}{N}$, $\bar{y}_s = \sum_{i=1}^{n} \frac{y_i}{n}, \bar{y}_{ns} = \sum_{i=n+1}^{N} \frac{y_i}{N-n}$. Then,

$$\frac{\bar{Y} - \bar{y}_s}{s\sqrt{(1-f)/n}}\ \Big|\ \bar{y}_s \quad \sim \quad t_{n-1}$$

> ⏳ VERIFY[c4]: the scale is consistent — $\operatorname{Var}(\bar{Y} - \bar{y}_s \mid \mu, \sigma^2\text{-free predictive}) = (1-f)^2\sigma^2\left(\frac{1}{N-n} + \frac{1}{n}\right) = \frac{(1-f)\sigma^2}{n}$ when $f = n/N$.

and the $100(1-\alpha)\%$ H.P.D. interval for $\bar{Y}$ is

$$\bar{y}_s \pm s\sqrt{\frac{1-f}{n}}\, t_{n-1,\frac{\alpha}{2}}.$$

Remark: This is an approximate interval used in survey sampling.

CLAIM c1: For iid normal $y_1,\dots,y_{n+1}$, $\operatorname{Var}(y_{n+1}-\bar{y}) = \sigma^2(1+1/n)$.
```python
from sympy import *

n = symbols('n', positive=True, integer=True)
sigma = symbols('sigma', positive=True)

# Var(y_{n+1} - ybar) = Var(y_{n+1}) + Var(ybar), independent (n+1 not in the sample mean)
var_new = sigma**2
var_ybar = sigma**2 / n
lhs = var_new + var_ybar
rhs = sigma**2 * (1 + Rational(1, 1)/n)
assert simplify(lhs - rhs) == 0

# sanity: explicit finite-n check via covariance of ybar with y_{n+1} (=0)
for nn in [1, 2, 5, 10]:
    assert simplify((sigma**2 + sigma**2/nn) - sigma**2*(1 + Rational(1, nn))) == 0
```

CLAIM c2: The Student-$t$ pivot $Z/\sqrt{V/(n-1)}$ reduces to $(y_{n+1}-\bar y)/(s\sqrt{1+1/n})$.
```python
from sympy import *

n = symbols('n', positive=True, integer=True)
sigma, s, d = symbols('sigma s d', positive=True)  # d = y_{n+1} - ybar

Z = d / (sigma * sqrt(1 + 1/n))          # ~ N(0,1)
V = (n - 1) * s**2 / sigma**2            # ~ chi^2_{n-1}

pivot = Z / sqrt(V / (n - 1))
target = d / (s * sqrt(1 + 1/n))

assert simplify(pivot - target) == 0     # sigma cancels exactly
```

CLAIM c3: $\bar{Y} = f\bar{y}_s + (1-f)\bar{y}_{ns}$ with $f=n/N$.
```python
from sympy import *

# exact check over several (n, N) pairs with symbolic y_i
for N, n in [(2, 1), (5, 2), (7, 3), (10, 4)]:
    y = symbols(f'y1:{N+1}', real=True)
    Ybar = Add(*y) / N
    ybar_s = Add(*y[:n]) / n
    ybar_ns = Add(*y[n:]) / (N - n)
    f = Rational(n, N)
    assert simplify(Ybar - (f*ybar_s + (1 - f)*ybar_ns)) == 0
```

CLAIM c4: $(1-f)^2\left(\frac{1}{N-n}+\frac{1}{n}\right) = \frac{1-f}{n}$ when $f=n/N$.
```python
from sympy import *

# encode N > n explicitly: N = n + m with m > 0, so sqrt/Abs simplifications close
n, m = symbols('n m', positive=True)
N = n + m
f = n / N

lhs = (1 - f)**2 * (1/(N - n) + 1/n)
rhs = (1 - f) / n

assert simplify(lhs - rhs) == 0

# and the resulting HPD half-width scale matches the printed s*sqrt((1-f)/n)
sigma = symbols('sigma', positive=True)
assert simplify(sqrt(sigma**2 * lhs) - sigma * sqrt((1 - f)/n)) == 0

# numeric spot checks with concrete N > n
for Nv, nv in [(10, 3), (100, 25), (7, 6)]:

    assert (1 - fv)**2 * (Rational(1, Nv - nv) + Rational(1, nv)) == (1 - fv) / nv
```
