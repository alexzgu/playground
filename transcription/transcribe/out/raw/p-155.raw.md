CLAIM c1: $\pi(\alpha)=1/(1+\alpha)^2$ on $\alpha>0$ integrates to exactly 1 (a proper prior).
```python
from sympy import *

alpha = symbols('alpha', positive=True)
pi_alpha = 1/(1 + alpha)**2

total = integrate(pi_alpha, (alpha, 0, oo))
assert simplify(total - 1) == 0

# proper => nonnegative on the stated support
assert simplify(pi_alpha.subs(alpha, Rational(1, 3))) > 0
```

CLAIM c2: $\pi(\mu,\tau)=1/(\tau+1)^2$ on $0<\mu<1,\ \tau>0$ is a proper joint density integrating to 1, and factors into independent Uniform(0,1) and $(\tau+1)^{-2}$ marginals.
```python
from sympy import *

mu = symbols('mu', positive=True)
tau = symbols('tau', positive=True)
joint = 1/(tau + 1)**2

# proper: integrates to exactly 1 over 0<mu<1, tau>0
total = integrate(integrate(joint, (mu, 0, 1)), (tau, 0, oo))
assert simplify(total - 1) == 0

# marginals
m_mu = integrate(joint, (tau, 0, oo))          # marginal of mu
m_tau = integrate(joint, (mu, 0, 1))           # marginal of tau
assert simplify(m_mu - 1) == 0                 # Uniform(0,1)
assert simplify(m_tau - 1/(tau + 1)**2) == 0

# independence: joint == product of marginals
assert simplify(joint - m_mu*m_tau) == 0
```
