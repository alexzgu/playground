**5.3 Marginal Likelihood**

Rank marginal likelihoods for different models (Bayes factor later)

$$x \mid \theta \;\sim\; f(x \mid \theta)$$

$$\pi(\theta) \quad \text{proper prior}$$

$$f(x) = \int f(x \mid \theta) \pi(\theta) d\theta$$

Example(1):

$$x_1, ..., x_n \mid \mu, \sigma^2 \overset{iid}{\sim} N(\mu, \sigma^2)$$

$$\text{Prior for } \theta, \pi(\theta) \text{ is proper.}$$

$$f(x) = \int f(x \mid \theta) \pi(\theta) d\theta \;\text{ is the marginal likelihood.}$$

Computation is done using Monte Carlo methods

$$\widehat{f(x)} = \frac{1}{M} \sum_{h=1}^{n} f(x \mid \theta_h) \quad \text{[sic: the upper limit should be } M\text{, matching the } \tfrac{1}{M} \text{ factor]}$$

> ⏳ VERIFY[c1]: the Monte Carlo estimator is unbiased for the marginal likelihood, i.e. $\int f(x\mid\theta)\pi(\theta)\,d\theta = E_\pi[f(x\mid\theta)]$ — checked on the conjugate normal example $x\mid\theta\sim N(\theta,1)$, $\theta\sim N(0,1)$, where $f(x)$ is the $N(0,2)$ density.

In general,

$$f(x) = \int f(x \mid \theta)\pi(\theta) d\theta = \frac{\displaystyle\int f(x\mid\theta)\frac{\pi(\theta)}{g(\theta)} g(\theta) d\theta}{\displaystyle\int \frac{\pi(\theta)}{g(\theta)} g(\theta) d\theta}, g(\theta) \text{ is a proposal density.}$$

> ⏳ VERIFY[c2]: the importance-sampling identity holds — the ratio of the two integrals equals $\int f(x\mid\theta)\pi(\theta)d\theta$, the denominator being $1$ because $\pi$ is proper.

So

$$\widehat{f(x)} = \frac{\frac{1}{M}\sum_{h=1}^{M} f(x \mid \theta_h)\frac{\pi(\theta_h)}{g(\theta_h)}}{\frac{1}{M}\sum_{h=1}^{M} \frac{\pi(\theta_h)}{g(\theta_h)}} = \sum_{h=1}^{M} w_h f(x \mid \theta_h)$$

> ⏳ VERIFY[c3]: the self-normalized form is an identity with $w_h = \frac{\pi(\theta_h)/g(\theta_h)}{\sum_{k=1}^{M}\pi(\theta_k)/g(\theta_k)}$, and these weights sum to $1$.

But another method can be done with the basic marginal likelihood identity.

$$\pi(\theta \mid x) = \frac{f(x \mid \theta)\pi(\theta)}{f(x)}$$

> ⏳ VERIFY[c4]: the basic marginal likelihood identity — with $f(x)=\int f(x\mid\theta)\pi(\theta)d\theta$, the stated $\pi(\theta\mid x)$ is a proper density and equals the exact posterior ($N(x/2, 1/2)$ in the conjugate normal example).

===CLAIMS===

CLAIM c1: The Monte Carlo estimator averages $f(x\mid\theta_h)$ over prior draws, and $E_\pi[f(x\mid\theta)] = f(x) = \int f(x\mid\theta)\pi(\theta)d\theta$; for $x\mid\theta\sim N(\theta,1)$, $\theta\sim N(0,1)$ this marginal is the $N(0,2)$ density.
```python
from sympy import *

x, th = symbols('x theta', real=True)

def npdf(v, m, s2):
    return exp(-(v - m)**2 / (2*s2)) / sqrt(2*pi*s2)

lik   = npdf(x, th, 1)      # f(x | theta)
prior = npdf(th, 0, 1)      # pi(theta)

marg = integrate(lik * prior, (th, -oo, oo))
marg = simplify(marg)

# The marginal likelihood is the N(0, 2) density evaluated at x.
assert simplify(marg - npdf(x, 0, 2)) == 0

# Sanity: at x = 0 this is 1/(2*sqrt(pi)).
assert simplify(marg.subs(x, 0) - 1/(2*sqrt(pi))) == 0
```

CLAIM c2: Importance-sampling identity: $\int f(x\mid\theta)\pi(\theta)d\theta = \dfrac{\int f(x\mid\theta)\frac{\pi(\theta)}{g(\theta)}g(\theta)d\theta}{\int \frac{\pi(\theta)}{g(\theta)}g(\theta)d\theta}$, with the denominator equal to 1 since $\pi$ is proper.
```python
from sympy import *

x, th = symbols('x theta', real=True)

def npdf(v, m, s2):
    return exp(-(v - m)**2 / (2*s2)) / sqrt(2*pi*s2)

lik   = npdf(x, th, 1)      # f(x | theta)
prior = npdf(th, 0, 1)      # pi(theta), proper
prop  = npdf(th, 0, 4)      # g(theta), proposal density

denom = integrate((prior/prop) * prop, (th, -oo, oo))
assert simplify(denom - 1) == 0          # pi is proper

numer = integrate(lik * (prior/prop) * prop, (th, -oo, oo))
lhs   = integrate(lik * prior, (th, -oo, oo))

assert simplify(numer/denom - lhs) == 0
```

CLAIM c3: The self-normalized estimator equals $\sum_h w_h f(x\mid\theta_h)$ with $w_h$ the normalized importance ratios, and $\sum_h w_h = 1$.
```python
from sympy import *

M = 3
f  = symbols('f1:4', positive=True)      # f(x | theta_h)
pi_ = symbols('p1:4', positive=True)     # pi(theta_h)
g  = symbols('g1:4', positive=True)      # g(theta_h)

r = [pi_[h]/g[h] for h in range(M)]      # importance ratios

lhs = (Rational(1, M) * sum(f[h]*r[h] for h in range(M))) / \
      (Rational(1, M) * sum(r[h] for h in range(M)))

w = [r[h] / sum(r[k] for k in range(M)) for h in range(M)]
rhs = sum(w[h]*f[h] for h in range(M))

assert simplify(lhs - rhs) == 0
assert simplify(sum(w) - 1) == 0
```

CLAIM c4: The basic marginal likelihood identity $\pi(\theta\mid x) = f(x\mid\theta)\pi(\theta)/f(x)$ yields a proper density; in the conjugate normal example it is $N(x/2, 1/2)$.
```python
from sympy import *

x, th = symbols('x theta', real=True)

def npdf(v, m, s2):
    return exp(-(v - m)**2 / (2*s2)) / sqrt(2*pi*s2)

lik   = npdf(x, th, 1)
prior = npdf(th, 0, 1)
fx    = npdf(x, 0, 2)                     # = int lik * prior d(theta), see c1

post = simplify(lik * prior / fx)

# Equals the N(x/2, 1/2) density in theta.
assert simplify(post - npdf(th, x/2, Rational(1, 2))) == 0

# It is a proper density: integrates to 1 for every x.
assert simplify(integrate(post, (th, -oo, oo)) - 1) == 0
```
