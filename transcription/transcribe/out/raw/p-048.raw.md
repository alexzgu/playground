CLAIM c1: Jeffreys' prior for the rate-parametrized Gamma(α,β) equals (1/β)·sqrt(α ψ'(α) + 1) as printed on the page.
```python
from sympy import *

a, b, y = symbols('alpha beta y', positive=True)

# Booklet's Gamma(alpha, beta) density (rate parametrization)
logf = a*log(b) + (a - 1)*log(y) - b*y - loggamma(a)

# Fisher information: both second derivatives below are free of y,
# so the expectation is the negative second derivative itself.
I11 = simplify(-diff(logf, a, 2))          # psi'(alpha)
I12 = simplify(-diff(diff(logf, a), b))    # -1/beta
I22 = simplify(-diff(logf, b, 2))          # alpha/beta**2

assert simplify(I11 - polygamma(1, a)) == 0
assert simplify(I12 + 1/b) == 0
assert simplify(I22 - a/b**2) == 0

detI = simplify(Matrix([[I11, I12], [I12, I22]]).det())

# The page asserts pi(alpha,beta) \propto (1/beta)*sqrt(alpha*psi'(alpha)+1),
# i.e. det I == ((1/beta)*sqrt(alpha*psi'(alpha)+1))**2.
printed = (sqrt(a*polygamma(1, a) + 1)/b)**2
assert simplify(detI - printed) == 0
```

CLAIM c2: the product over i collapses to β^(nα−1) g^(n(α−1)) e^(−nβȳ)/Γ(α)^n · 1/(1+α)^2 with g the geometric mean and ȳ the arithmetic mean.
```python
from sympy import *

a, b = symbols('alpha beta', positive=True)

n = 3
ys = [Rational(1, 2), Rational(3, 2), Rational(5, 2)]
ybar = Rational(sum(ys))/n
prod_y = Mul(*ys)                      # g**n, so g**(n*(a-1)) == prod_y**(a-1)

lhs = Mul(*[b**a * yi**(a - 1) * exp(-b*yi) / gamma(a) for yi in ys])
lhs = lhs * (1/b) * 1/(1 + a)**2

rhs = b**(n*a - 1) * prod_y**(a - 1) * exp(-n*b*ybar) / gamma(a)**n * 1/(1 + a)**2

assert simplify(lhs - rhs) == 0
```
