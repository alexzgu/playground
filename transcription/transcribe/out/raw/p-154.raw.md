Suppose there are k (random) distinct values $\mu_1^*, ..., \mu_k^*$.
As the process goes on k will vary, so that k is a random variable.

Antoniak(1974) showed that:

$$P(k|\alpha,l) = c_l(k)\alpha^k \frac{\tau(\alpha)}{\tau(\alpha+l)}, \quad 1 \le k \le l$$

*(The booklet writes the gamma function as $\tau(\cdot)$ throughout this passage; $\tau(\alpha)/\tau(\alpha+l)$ is $\Gamma(\alpha)/\Gamma(\alpha+l)$.)*

> ⏳ VERIFY[c1]: with $c_l(k)=|s(l,k)|$ the unsigned Stirling numbers of the first kind, $P(k\mid\alpha,l)=c_l(k)\alpha^k\Gamma(\alpha)/\Gamma(\alpha+l)$ is a valid pmf on $1\le k\le l$.

$c_l(k)$ are the absolute value of the stirling [sic] number of the first kind obtained by equality:

$$\prod_{k=0}^{l-1}(x-k) = \sum_{k=0}^{l} c_l(k)x^k$$

CLAIM c1: With $c_l(k)=|s(l,k)|$, the Antoniak formula $P(k|\alpha,l)=c_l(k)\alpha^k\Gamma(\alpha)/\Gamma(\alpha+l)$ sums to 1 over $1\le k\le l$ and vanishes at $k=0$.
```python
from sympy import symbols, gamma, simplify, Abs
from sympy.functions.combinatorial.numbers import stirling

a = symbols('alpha', positive=True)

for l in range(1, 8):
    # c_l(k) = unsigned Stirling numbers of the first kind
    c = [Abs(stirling(l, k, kind=1, signed=True)) for k in range(l + 1)]
    # k = 0 term is zero for l >= 1, so support really is 1 <= k <= l
    assert c[0] == 0
    total = sum(c[k] * a**k for k in range(1, l + 1)) * gamma(a) / gamma(a + l)
    assert simplify(total - 1) == 0
    # each probability is nonnegative
    for k in range(1, l + 1):
        assert c[k] > 0
```

CLAIM c2: The unsigned Stirling numbers come from the rising factorial; the product as printed with $(x-k)$ yields the signed Stirling numbers.
```python
from sympy import symbols, expand, Poly, Abs
from sympy.functions.combinatorial.numbers import stirling

x = symbols('x')

for l in range(1, 9):
    signed = [stirling(l, k, kind=1, signed=True) for k in range(l + 1)]

    # printed form: falling factorial -> SIGNED Stirling numbers
    falling = expand(1)
    for k in range(0, l):
        falling *= (x - k)
    assert Poly(expand(falling), x).all_coeffs()[::-1] == signed

    # corrected form: rising factorial -> ABSOLUTE values c_l(k)
    rising = expand(1)
    for k in range(0, l):
        rising *= (x + k)
    assert Poly(expand(rising), x).all_coeffs()[::-1] == [Abs(s) for s in signed]

    # the two agree only when all signs are +, i.e. l <= 1
    if l >= 2:
        assert signed != [Abs(s) for s in signed]
```

CLAIM c3: $\alpha^k\Gamma(\alpha)/\Gamma(\alpha+l)$ is log-concave in $\ln\alpha$ for $\alpha>0$ (second derivative $\le 0$; identically 0 when $l=1$).
```python
import math
from scipy.special import psi, polygamma

# g(t) = k*t + logGamma(e^t) - logGamma(e^t + l);  the k*t term is linear,
# so g''(t) = a*(psi(a)-psi(a+l)) + a^2*(psi'(a)-psi'(a+l)) with a = e^t.
def g2(t, l):
    a = math.exp(t)
    return a * (psi(a) - psi(a + l)) + a**2 * (float(polygamma(1, a)) - float(polygamma(1, a + l)))

tol = 1e-9  # double-precision cancellation guard

for l in range(1, 9):
    for i in range(-120, 121):
        t = i / 10.0          # alpha = e^t ranges over ~e^-12 .. e^12
        assert g2(t, l) <= tol, (l, t, g2(t, l))

# l = 1 is the boundary case: alpha^k * Gamma(a)/Gamma(a+1) = alpha^(k-1) is
# log-linear, so the second derivative is exactly zero (concave, not strictly).
for i in range(-50, 51):
    assert abs(g2(i / 10.0, 1)) < tol

# for l >= 2 the concavity is strict (bounded away from 0 on this grid)
for l in range(2, 9):
    for i in range(-50, 51):
        assert g2(i / 10.0, l) < -1e-8, (l, i / 10.0, g2(i / 10.0, l))
```
