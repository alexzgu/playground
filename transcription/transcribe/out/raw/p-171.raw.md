CLAIM c1: Integrating the Gamma kernel in $\phi_i$ yields $\Gamma(a_i)\,b_i^{-a_i}$ with $a_i = n_i(\alpha-1/\gamma)+1/\gamma$, matching the exponent printed on $\sum_j y_{ij}^\gamma$.
```python
from sympy import *

phi = symbols('phi', positive=True)
alpha, gamma_ = symbols('alpha gamma', positive=True)
n_i = symbols('n_i', positive=True)
b = symbols('b', positive=True)  # b = sum_j y_ij^gamma

a = n_i*(alpha - 1/gamma_) + 1/gamma_   # shape stated by the booklet

# generic Gamma-kernel integral, with a symbolic positive shape
A = symbols('A', positive=True)
I = integrate(phi**(A-1)*exp(-b*phi), (phi, 0, oo))
assert simplify(I - gamma(A)*b**(-A)) == 0

# therefore the power of b appearing after integration is exactly the shape a
power_of_b = -a
assert simplify(power_of_b + (n_i*(alpha - 1/gamma_) + 1/gamma_)) == 0

# decisive numeric instance: n_i=2, alpha=3, gamma=2  ->  a = 2*(3-1/2)+1/2 = 11/2
a_num = a.subs({n_i: Integer(2), alpha: Integer(3), gamma_: Integer(2)})
assert a_num == Rational(11, 2)
I_num = integrate(phi**(a_num-1)*exp(-b*phi), (phi, 0, oo))
assert simplify(I_num - gamma(Rational(11,2))*b**(Rational(-11,2))) == 0
```

CLAIM c2: The reparametrization $\delta_1=1/(1+\alpha)$, $\delta_2=1/(1+\gamma)$ has Jacobian $(1+\alpha)^{-2}(1+\gamma)^{-2}$ in reverse, so those two prior factors are exactly absorbed and $\pi(\delta_1,\delta_2\mid y)$ equals the bracketed kernel with no leftover factor.
```python
from sympy import *

d1, d2 = symbols('delta_1 delta_2', positive=True)

alpha_of_d1 = (1 - d1)/d1      # inverse of delta_1 = 1/(1+alpha)
gamma_of_d2 = (1 - d2)/d2      # inverse of delta_2 = 1/(1+gamma)

# inverses are genuine
assert simplify(1/(1 + alpha_of_d1) - d1) == 0
assert simplify(1/(1 + gamma_of_d2) - d2) == 0

jac = Abs(diff(alpha_of_d1, d1)) * Abs(diff(gamma_of_d2, d2))
assert simplify(jac - 1/(d1**2 * d2**2)) == 0

# the prior factors (1+alpha)^-2 (1+gamma)^-2 times the Jacobian is exactly 1
prior_factors = (1 + alpha_of_d1)**(-2) * (1 + gamma_of_d2)**(-2)
assert simplify(prior_factors * jac - 1) == 0
```
