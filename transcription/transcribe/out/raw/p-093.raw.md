**Two More Extremes**

These are related to the sampling variance $\sigma_i^2/n_i$, and hence the sample size $n_i$

&nbsp;&nbsp;&nbsp;&nbsp;a. Suppose $\sigma_i^2/n_i \to 0$. Then $\lambda_i \to 1$, and $E\left(\mu_i \mid \boldsymbol{y}\right) \to \bar{y}_i$

&nbsp;&nbsp;&nbsp;&nbsp;b. Suppose $\sigma_i^2/n_i \to \infty$. Then $\lambda_i \to 0$, and $E\left(\mu_i \mid \boldsymbol{y}\right) \to \hat{\theta} = E\left(\mu_i \mid \boldsymbol{y}\right) \to \hat{\theta} = \frac{\sum_{j \neq i} \lambda_j y_j}{\sum_{j \neq i} \lambda_j}$ $\text{[sic: the fragment ``}E(\mu_i\mid\boldsymbol{y}) \to \hat\theta =\text{'' is printed twice]}$

> ⏳ VERIFY[c1]: the two limiting cases — $\sigma_i^2/n_i \to 0$ gives $\lambda_i \to 1$ and posterior mean $\to \bar y_i$; $\sigma_i^2/n_i \to \infty$ gives $\lambda_i \to 0$, posterior mean $\to \hat\theta$, and $\hat\theta$ reduces to the leave-one-out pooled estimator $\sum_{j\neq i}\lambda_j y_j / \sum_{j\neq i}\lambda_j$.

What does (a) say?

&nbsp;&nbsp;&nbsp;&nbsp;If the $i^{th}$ cluster is very homogenous or the sample size is large, there is no need to pool with other clusters.

(b) is remarkable!! What does it say?

&nbsp;&nbsp;&nbsp;&nbsp;It says that if the $i^{th}$ cluster is very heterogenous or the sample size is too small, then use the pooled estimator formed by the other clusters.

**9.2.2 Gaining Strength**

Recall $\mu_i \mid \boldsymbol{y} \sim N\left(\lambda_i \bar{y}_i + \left(1-\lambda_i\right)\hat{\theta},\; \left(1-\lambda_i\right)^2 \nu^2 + \left(1-\lambda_i\right)\delta^2\right)$

Suppose we have only the $i^{th}$ area:

$$y_{ij} \mid \mu_i \overset{iid}{\sim} N\left(\mu_i, \sigma_i^2\right)$$
$$p\left(\mu_i\right) = 1$$

Then,

$$\mu_i \mid \boldsymbol{y}_i \sim N\left(\bar{y}_i, \frac{\sigma_i^2}{m_i}\right)$$

> ⏳ VERIFY[c3]: under $y_{ij}\mid\mu_i \overset{iid}{\sim} N(\mu_i,\sigma_i^2)$, $j=1,\dots,m_i$, with the flat prior $p(\mu_i)=1$, the posterior is $N(\bar y_i, \sigma_i^2/m_i)$.

NOTE: $\left(1-\lambda_i\right)^2 \nu^2 + \left(1-\lambda_i\right)\delta^2 = \frac{\sigma_i^2}{m_i}\left\{\frac{\lambda_i\left(1 + \sum_{j \neq i} \lambda_j\right)}{\lambda_i + \sum_{j \neq i} \lambda_j}\right\} < \frac{\sigma_i^2}{m_i}$

> ⏳ VERIFY[c2]: with $\lambda_i = \delta^2/\left(\delta^2 + \sigma_i^2/m_i\right)$ and $\nu^2 = \delta^2/\sum_j \lambda_j$, the identity $\left(1-\lambda_i\right)^2\nu^2 + \left(1-\lambda_i\right)\delta^2 = \frac{\sigma_i^2}{m_i}\cdot\frac{\lambda_i(1+\sum_{j\neq i}\lambda_j)}{\lambda_i + \sum_{j\neq i}\lambda_j}$ holds exactly, and the bracketed factor is strictly less than $1$.

Remark: There will be a further gain in precision if we allow the $\sigma_i^2$ to share an effect, or if the $\sigma_i^2$ could be taken equal.

CLAIM c1: The two extreme cases of the shrinkage factor and the posterior mean, and the leave-one-out reduction of $\hat\theta$.
```python
from sympy import *

# v = sigma_i^2 / n_i, the sampling variance of the i-th cluster mean
v, d2 = symbols('v delta2', positive=True)
lam_i = d2 / (d2 + v)

# (a) v -> 0  =>  lambda_i -> 1 ;  (b) v -> oo  =>  lambda_i -> 0
assert limit(lam_i, v, 0) == 1
assert limit(lam_i, v, oo) == 0

ybar, thetahat = symbols('ybar thetahat', real=True)
post_mean = lam_i * ybar + (1 - lam_i) * thetahat

# (a) posterior mean -> ybar_i ; (b) posterior mean -> theta-hat
assert simplify(limit(post_mean, v, 0) - ybar) == 0
assert simplify(limit(post_mean, v, oo) - thetahat) == 0

# theta-hat = sum_j lam_j y_j / sum_j lam_j ; as lam_i -> 0 the i-th term drops out,
# leaving the pooled estimator formed by the OTHER clusters.
li = symbols('li', positive=True)
l2, l3 = symbols('l2 l3', positive=True)
y1, y2, y3 = symbols('y1 y2 y3', real=True)
theta_hat = (li*y1 + l2*y2 + l3*y3) / (li + l2 + l3)
assert simplify(limit(theta_hat, li, 0) - (l2*y2 + l3*y3)/(l2 + l3)) == 0
```

CLAIM c2: The NOTE identity for the posterior variance, and the strict inequality $< \sigma_i^2/m_i$.
```python
from sympy import *

# v = sigma_i^2 / m_i ;  S = sum_{j != i} lambda_j  (a sum of positive shrinkages)
v, d2, S = symbols('v delta2 S', positive=True)

lam_i = d2 / (d2 + v)          # shrinkage factor for cluster i
nu2   = d2 / (lam_i + S)       # nu^2 = delta^2 / sum_j lambda_j

lhs = (1 - lam_i)**2 * nu2 + (1 - lam_i) * d2
rhs = v * (lam_i * (1 + S) / (lam_i + S))

# the identity, exactly
assert simplify(lhs - rhs) == 0

# strict inequality: v - rhs equals v^2 * S / ((delta^2 + v) * (lambda_i + S)),
# a quotient of strictly positive quantities, hence rhs < v = sigma_i^2/m_i.
gap = v**2 * S / ((d2 + v) * (lam_i + S))
assert simplify((v - rhs) - gap) == 0
assert gap.is_positive is True
```

CLAIM c3: Flat prior on $\mu_i$ with an iid normal sample gives posterior $N(\bar y_i,\ \sigma_i^2/m_i)$.
```python
from sympy import *

mu = symbols('mu', real=True)
s2 = symbols('sigma2', positive=True)

m = 4  # arbitrary sample size m_i
ys = symbols('y1 y2 y3 y4', real=True)
ybar = Add(*ys) / m

# log posterior kernel with p(mu) = 1
log_post = -Add(*[(y - mu)**2 for y in ys]) / (2*s2)

# claimed kernel: N(ybar, sigma^2/m)
log_target = -(mu - ybar)**2 / (2 * (s2/m))

# the two differ only by a constant free of mu  <=>  same distribution in mu
assert simplify(diff(log_post - log_target, mu)) == 0
```
