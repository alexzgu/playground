===CLAIMS===

CLAIM c1: Gamma(α,1): first log-derivative is (α−1)/x − 1, second is −(α−1)/x², so logconcave iff α ≥ 1.
```python
from sympy import *

x = symbols('x', positive=True)
a = symbols('alpha', positive=True)

f = x**(a-1) * exp(-x) / gamma(a)
logf = log(x)*(a-1) - x - log(gamma(a))

# the booklet's stated log-density
assert simplify(expand_log(log(f), force=True) - logf) == 0

d1 = diff(logf, x)
d2 = diff(logf, x, 2)

assert simplify(d1 - ((a - 1)/x - 1)) == 0
assert simplify(d2 - (-(a - 1)/x**2)) == 0

# second derivative <= 0 on x>0 exactly when alpha >= 1
assert d2.subs({a: Rational(3, 2), x: Rational(1, 2)}) < 0   # alpha > 1 -> logconcave
assert d2.subs({a: Rational(1, 2), x: Rational(1, 2)}) > 0   # alpha < 1 -> not logconcave
```

CLAIM c2: Standard Cauchy is not logconcave: (log f)'' = 2(x²−1)/(1+x²)² > 0 for |x| > 1.
```python
from sympy import *

x = symbols('x', real=True)
f = 1 / (pi * (1 + x**2))
d2 = diff(log(f), x, 2)

assert simplify(d2 - 2*(x**2 - 1)/(1 + x**2)**2) == 0

# strictly positive somewhere -> log f is not concave
assert d2.subs(x, 2) == Rational(6, 25)
assert d2.subs(x, 2) > 0
assert d2.subs(x, -2) > 0
# (and negative near 0, so it is genuinely not concave anywhere-globally)
assert d2.subs(x, 0) < 0
```

CLAIM c3: Standard lognormal (log X ~ N(0,1)) is not logconcave: (log f)'' = log(x)/x² > 0 for x > 1.
```python
from sympy import *

x = symbols('x', positive=True)
f = 1/(x*sqrt(2*pi)) * exp(-log(x)**2 / 2)

logf = simplify(expand_log(log(f), force=True))
d2 = simplify(diff(logf, x, 2))

assert simplify(d2 - log(x)/x**2) == 0

assert d2.subs(x, E) == exp(-2)      # log(e)/e^2 = 1/e^2 > 0
assert d2.subs(x, E) > 0
assert d2.subs(x, Rational(1, 2)) < 0
```

CLAIM c4: N(μ,σ²) is logconcave: (log f)'' = −1/σ² < 0.
```python
from sympy import *

x, mu = symbols('x mu', real=True)
sigma = symbols('sigma', positive=True)

f = 1/(sigma*sqrt(2*pi)) * exp(-(x - mu)**2 / (2*sigma**2))
d2 = diff(log(f), x, 2)

assert simplify(d2 - (-1/sigma**2)) == 0
assert d2.subs(sigma, 2) < 0
```

CLAIM c5: Beta(α,β) with α,β > 1 is logconcave: (log f)'' = −(α−1)/x² − (β−1)/(1−x)² < 0 on (0,1).
```python
from sympy import *

x = symbols('x', positive=True)
a, b = symbols('alpha beta', positive=True)

# density on (0,1); the beta() normalizing constant is free of x
f = x**(a-1) * (1-x)**(b-1) / beta(a, b)
logf = (a-1)*log(x) + (b-1)*log(1-x) - log(beta(a, b))

d2 = diff(logf, x, 2)
assert simplify(d2 - (-(a-1)/x**2 - (b-1)/(1-x)**2)) == 0

# strictly negative on (0,1) whenever alpha, beta > 1
for av, bv in [(Rational(3,2), Rational(3,2)), (2, 5), (Rational(11,10), 3)]:
    for xv in [Rational(1,4), Rational(1,2), Rational(3,4)]:
        assert d2.subs({a: av, b: bv, x: xv}) < 0

# and it fails outside that range: alpha < 1 makes (log f)'' > 0 near x = 0
# (at x=1/2 the beta-term still dominates, so test close to 0 where the
# positive -(alpha-1)/x^2 term blows up)
assert d2.subs({a: Rational(1,2), b: 2, x: Rational(1,10)}) > 0
```
