## **13.1 Hierarchical Multinomial-Dirichlet Model**

If we have groups of counts

$$\underset{k\times 1}{n_i} \Big| \underset{k\times 1}{\theta_i} \sim multi(n_i, \theta_i)_{k\times 1} \tag{13.1}$$

$$\theta_{ij} > 0 \qquad \sum_{j=1}^{k}\theta_{ij} = 1$$

$$\theta_i \mid \mu, \tau \sim Dir(\mu, \tau) \tag{13.2}$$

$$\pi(\mu, \tau) = \frac{(k-1)!}{(\tau+1)^2} \tag{13.3}$$

> ⏳ VERIFY[c3]: $\pi(\mu,\tau) = (k-1)!/(\tau+1)^2$ is a proper joint density — it integrates to $1$ over $\mu$ on the $(k-1)$-dimensional simplex and $\tau \in (0,\infty)$.

CLAIM c1: For the four-cell uniform Dirichlet, $1/Dir(1,1,1,1) = \Gamma(4)/\Gamma(1) = 3! = 6$.
```python
from sympy import gamma, factorial, prod, Rational, simplify

def Dir(a):
    return prod([gamma(x) for x in a]) / gamma(sum(a))

a = [1, 1, 1, 1]
lhs = 1 / Dir(a)
mid = gamma(4) / gamma(1)
rhs = factorial(3)

assert simplify(lhs - mid) == 0
assert simplify(mid - rhs) == 0
assert simplify(lhs - 6) == 0
```

CLAIM c2: The simplex integral of the multinomial kernel equals the Dirichlet normalizing constant with parameters shifted by one: $\int \prod \theta_{ij}^{n_{ij}} d\theta = Dir(n_{11}+1,\dots,n_{22}+1)$.
```python
from sympy import symbols, gamma, prod, integrate, simplify, Rational

def Dir(a):
    return prod([gamma(x) for x in a]) / gamma(sum(a))

t11, t12, t21 = symbols('t11 t12 t21', positive=True)

# Check the identity for several exact integer exponent vectors.
for n in [(1, 2, 0, 1), (0, 0, 0, 0), (2, 1, 3, 0), (1, 1, 1, 1)]:
    n11, n12, n21, n22 = n
    t22 = 1 - t11 - t12 - t21
    integrand = t11**n11 * t12**n12 * t21**n21 * t22**n22
    # integrate over the 3-dim simplex (t22 determined by the constraint)
    I = integrate(integrand, (t21, 0, 1 - t11 - t12))
    I = integrate(I, (t12, 0, 1 - t11))
    I = integrate(I, (t11, 0, 1))
    rhs = Dir([n11 + 1, n12 + 1, n21 + 1, n22 + 1])
    assert simplify(I - rhs) == 0, n

# spot value: n = (1,2,0,1) -> Dir(2,3,1,2) = 1/2520
assert simplify(Dir([2, 3, 1, 2]) - Rational(1, 2520)) == 0
```

tau = symbols('tau', positive=True)

# tau-part: integral of (tau+1)^{-2} over (0, oo) = 1
tau_int = integrate(1 / (tau + 1)**2, (tau, 0, oo))
assert simplify(tau_int - 1) == 0
