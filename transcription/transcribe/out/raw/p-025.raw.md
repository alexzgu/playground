**3.2 Prior Construction** *(correction: the printed heading reads "Constraction" [sic]; a handwritten "u" with an insertion caret is inserted over the "a", giving "Construction")*

Two types of prior distributions:
a. Subjective: proper, conjugate.
b. Objective: proper, improper (noninformative).

Objective priors are constructed based on the likelihood function (also conjugate). *(correction: the printed word reads "construted" [sic]; a handwritten "c" with an insertion caret is inserted between "u" and "t")*
Objective priors are two types:
a. Jeffreys' prior.
b. Shrinkage prior.

**Example**

$y_1, ..., y_n \mid \mu \overset{iid}{\sim} N(\mu, \sigma^2)$ *(The booklet writes $\mu$ with under-tildes to indicate vectors.)*

$\mu \sim N(\theta, \delta^2)$ – subjective, also conjugate (prior and posterior are in the same family).

$\mu \mid y \sim N(\lambda \bar{y} + (1-\lambda)\theta, (1-\lambda)\delta^2)$

> ⏳ VERIFY[c2]: with $\lambda = \frac{n\delta^2}{\sigma^2 + n\delta^2}$, the normal–normal posterior is $N(\lambda\bar{y} + (1-\lambda)\theta,\ (1-\lambda)\delta^2)$.

We say that $\mu \sim N(\theta, \delta^2)$ is a conjugate prior.

**Example**

$X_1, ..., X_n \mid \theta \overset{iid}{\sim} Bernoulli(\theta)$

$\theta \sim Beta(\alpha, \beta)$

$\theta \mid x \sim Beta(s + \alpha, n - s + \beta)$

> ⏳ VERIFY[c3]: Beta–Bernoulli conjugacy, with $s = \sum x_i$, gives posterior $Beta(s+\alpha,\ n-s+\beta)$.

**Example**

$X_1, ..., X_n \mid \theta \overset{iid}{\sim} f(x \mid \theta) = \theta e^{-\theta x}, x \geq 0$

$\theta \sim Gamma(\alpha, \beta)$

$$\pi(\theta) = \frac{\beta^\alpha \theta^{\alpha-1}e^{-\beta\theta}}{\Gamma(\alpha)}$$

$\theta \mid x \sim Gamma(n + \alpha, n\bar{x} + \beta), \theta > 0$

> ⏳ VERIFY[c4]: Gamma–exponential conjugacy gives posterior $Gamma(n+\alpha,\ n\bar{x}+\beta)$.

**Why do we use conjugate priors?**

1. Easy to interpret.
2. Easy computations.
3. Very useful today because they are used as building blocks for more complicated models (e.g., hierarchical Bayesian models).
4. Easy to update.

from scipy.integrate import quad

phi = lambda z: math.exp(-z**2/2)/math.sqrt(2*math.pi)

lhs = quad(lambda z: math.sqrt(abs(z))*phi(z), a, b, points=[0.0])[0]

CLAIM c2: Normal-normal conjugacy: posterior is N(lambda*ybar + (1-lambda)*theta, (1-lambda)*delta^2) with lambda = n*delta^2/(sigma^2 + n*delta^2).
```python
from sympy import symbols, exp, sqrt, pi, integrate, simplify, expand, Rational, oo, solve, Poly

mu, theta, ybar = symbols('mu theta ybar', real=True)
sigma, delta = symbols('sigma delta', positive=True)
n = symbols('n', positive=True)

lam = n*delta**2/(sigma**2 + n*delta**2)
post_mean = lam*ybar + (1 - lam)*theta
post_var = (1 - lam)*delta**2

# log posterior kernel (up to constants in mu):
#   -n(ybar-mu)^2/(2 sigma^2) - (mu-theta)^2/(2 delta^2)
# claimed kernel: -(mu - post_mean)^2/(2 post_var)
kernel = -n*(ybar - mu)**2/(2*sigma**2) - (mu - theta)**2/(2*delta**2)
claimed = -(mu - post_mean)**2/(2*post_var)

diff = expand(simplify(kernel - claimed))
# must be free of mu (i.e. differ only by a constant in mu)
assert simplify(diff.diff(mu)) == 0

# and the quadratic coefficient matches => same variance
assert simplify(kernel.diff(mu, 2) - claimed.diff(mu, 2)) == 0

# exact numeric cross-check: n=3, sigma=1, delta=2, theta=1, ybar=5
subs = {n: 3, sigma: 1, delta: 2, theta: 1, ybar: 5}
assert simplify(post_var.subs(subs) - Rational(4, 13)) == 0
assert simplify(post_mean.subs(subs) - Rational(61, 13)) == 0
```

CLAIM c3: Beta-Bernoulli conjugacy: prior Beta(alpha,beta) with s successes in n trials gives posterior Beta(s+alpha, n-s+beta).
```python
from sympy import symbols, simplify, gamma, integrate, Rational, expand

th = symbols('theta', positive=True)
al, be = symbols('alpha beta', positive=True)
n, s = symbols('n s', positive=True)

# unnormalized posterior = theta^s (1-theta)^(n-s) * theta^(alpha-1) (1-theta)^(beta-1)
unnorm = th**s * (1 - th)**(n - s) * th**(al - 1) * (1 - th)**(be - 1)
# claimed Beta(s+alpha, n-s+beta) kernel
claimed = th**(s + al - 1) * (1 - th)**(n - s + be - 1)
assert simplify(unnorm - claimed) == 0

# exact normalized check with n=5, s=2, alpha=3, beta=4
vals = {n: 5, s: 2, al: 3, be: 4}
A, B = Rational(2 + 3), Rational(5 - 2 + 4)   # posterior params 5, 7
beta_pdf = gamma(A + B)/(gamma(A)*gamma(B)) * th**(A - 1)*(1 - th)**(B - 1)
u = unnorm.subs(vals)
Z = integrate(u, (th, 0, 1))
assert simplify(u/Z - beta_pdf) == 0
```

CLAIM c4: Gamma-exponential conjugacy: prior Gamma(alpha,beta), n iid Exp(theta) observations give posterior Gamma(n+alpha, n*xbar+beta).
```python
from sympy import symbols, exp, simplify, gamma, integrate, Rational, oo

th = symbols('theta', positive=True)
al, be = symbols('alpha beta', positive=True)
n, xbar = symbols('n xbar', positive=True)

# likelihood theta^n exp(-theta * n*xbar) times prior kernel theta^(alpha-1) exp(-beta theta)
unnorm = th**n * exp(-th*n*xbar) * th**(al - 1) * exp(-be*th)
# claimed Gamma(n+alpha, n*xbar+beta) kernel (rate parameterization, matching pi(theta))
claimed = th**(n + al - 1) * exp(-(n*xbar + be)*th)
assert simplify(unnorm - claimed) == 0

# exact normalized check: n=4, xbar=1/2, alpha=3, beta=2  =>  Gamma(7, 4)
vals = {n: 4, xbar: Rational(1, 2), al: 3, be: 2}
A, R = Rational(7), Rational(4)
gam_pdf = R**A/gamma(A) * th**(A - 1) * exp(-R*th)
u = unnorm.subs(vals)
Z = integrate(u, (th, 0, oo))
assert simplify(Z - gamma(A)/R**A) == 0
assert simplify(u/Z - gam_pdf) == 0
```
