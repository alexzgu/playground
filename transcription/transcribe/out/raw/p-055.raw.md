## 5.10 Importance Sampling

We are now interested in

$$E = \int_{-\infty}^{\infty} h(x)f(x)dx,$$

But now it is not easy to draw a sample from $f(x)$.
Suppose, however, we can get samples from $g(x)$. That is,

$$X_1,...,X_M \overset{iid}{\sim} g(x)$$

Then,

$$E = \int_{-\infty}^{\infty} h(x)\frac{f(x)}{g(x)}g(x)dx,$$

and one MC estimator is

$$\widetilde{E}_M^{(1)} = M^{-1}\sum_{i=1}^{M} h(x_i)\lfloor\frac{f(x_i)}{g(x_i)}\rfloor$$

*[sic: the brackets around the weight are set as $\lfloor\;\rfloor$ in the scan; ordinary square brackets are intended.]*

> ⏳ VERIFY[c3]: the importance-sampling identity and the unbiasedness of $\widetilde{E}_M^{(1)}$ — $E_g\!\left[h(X)\frac{f(X)}{g(X)}\right] = \int h(x)f(x)dx$.

CLAIM c1: For the Cauchy(0,1) prior and N(θ,1) likelihood, both integrals defining E(θ|x) converge and the denominator is strictly positive, so E(θ|x) < ∞.
```python
import math
from scipy.integrate import quad

def check(x):
    phi = lambda t: math.exp(-0.5 * (x - t) ** 2) / math.sqrt(2 * math.pi)
    num_abs, e1 = quad(lambda t: abs(t) / (1 + t * t) * phi(t), -math.inf, math.inf)
    num, e2 = quad(lambda t: t / (1 + t * t) * phi(t), -math.inf, math.inf)
    den, e3 = quad(lambda t: 1.0 / (1 + t * t) * phi(t), -math.inf, math.inf)
    # absolute convergence of the numerator => the posterior mean is well defined
    # (quad's reported error estimate is ~1e-8 here; 1e-6 is ample for a
    # finiteness/positivity claim)
    assert math.isfinite(num_abs) and e1 < 1e-6
    assert math.isfinite(num) and e2 < 1e-6
    # denominator is a strictly positive normalizing constant
    assert den > 0 and math.isfinite(den) and e3 < 1e-6
    val = num / den
    assert math.isfinite(val) and abs(val) <= num_abs / den + 1e-12
    return val

for x in (-3.0, -1.0, 0.0, 1.0, 2.5, 10.0):
    check(x)

# at x = 0 the integrand of the numerator is odd, so the posterior mean is exactly 0
assert abs(check(0.0)) < 1e-12
```

CLAIM c3: Importance sampling identity / unbiasedness: E_g[h(X) f(X)/g(X)] = int h(x) f(x) dx.
```python
from sympy import symbols, exp, integrate, simplify, oo, pi, sqrt, Rational

x = symbols('x', real=True)

# Case 1: f = Exp(1), g = Exp(2), h(x) = x on (0, oo)
f = exp(-x)
g = 2 * exp(-2 * x)
h = x
target = integrate(h * f, (x, 0, oo))
is_expect = integrate(h * (f / g) * g, (x, 0, oo))
assert simplify(target - 1) == 0
assert simplify(is_expect - target) == 0

# Case 2: f = N(0,1), g = N(0,2) (heavier proposal), h(x) = x**2, on the whole line
f2 = exp(-x**2 / 2) / sqrt(2 * pi)
g2 = exp(-x**2 / 8) / (2 * sqrt(2 * pi))
h2 = x**2
target2 = integrate(h2 * f2, (x, -oo, oo))
is_expect2 = integrate(h2 * (f2 / g2) * g2, (x, -oo, oo))
assert simplify(target2 - 1) == 0          # Var of N(0,1)
assert simplify(is_expect2 - target2) == 0

# the estimator is unbiased: E[Ehat] = M^{-1} * M * E_g[h f / g] = target
Mn = 7
assert simplify(Rational(1, Mn) * Mn * is_expect - target) == 0
```
