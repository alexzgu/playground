CLAIM c1: The marginal density $\pi(\theta_1\mid y)=\int \pi(\theta_1\mid\theta_2,y)\pi(\theta_2\mid y)\,d\theta_2$ holds; verified exactly for a bivariate normal posterior with correlation $\rho$.
```python
from sympy import *

t1, t2 = symbols('theta1 theta2', real=True)
rho = Rational(1, 2)          # exact correlation
s = sqrt(1 - rho**2)

# joint standard bivariate normal posterior pi(t1,t2)
joint = (1/(2*pi*s)) * exp(-(t1**2 - 2*rho*t1*t2 + t2**2)/(2*(1-rho**2)))

# marginal of theta2 and the conditional pi(t1 | t2)
marg2 = (1/sqrt(2*pi)) * exp(-t2**2/2)
cond  = (1/(sqrt(2*pi)*s)) * exp(-(t1 - rho*t2)**2/(2*(1-rho**2)))

# the conditional times the marginal must reconstruct the joint
assert simplify(cond*marg2 - joint) == 0

# and the mixture integral must equal the true marginal of theta1
mixture = integrate(cond*marg2, (t2, -oo, oo))
true_marg1 = (1/sqrt(2*pi)) * exp(-t1**2/2)
assert simplify(mixture - true_marg1) == 0
```

CLAIM c2: The Rao-Blackwellized estimator has MSE no larger than the un-conditioned one: $\operatorname{Var}(E[\theta_1\mid\theta_2]) \le \operatorname{Var}(\theta_1)$, with both unbiased for $E(\theta_1)$.
```python
from sympy import *

rho, s1, s2 = symbols('rho sigma1 sigma2', real=True, positive=True)
t2 = symbols('theta2', real=True)

# bivariate normal: E[theta1 | theta2] = rho*(s1/s2)*theta2, Var(theta1) = s1**2
g = rho*(s1/s2)*t2

# theta2 ~ N(0, s2**2): the RB estimator is unbiased for E(theta1) = 0
dens2 = (1/(s2*sqrt(2*pi))) * exp(-t2**2/(2*s2**2))
assert simplify(integrate(g*dens2, (t2, -oo, oo))) == 0

# Var(E[theta1|theta2]) = rho**2 * s1**2
var_rb = simplify(integrate(g**2*dens2, (t2, -oo, oo)))
assert simplify(var_rb - rho**2*s1**2) == 0

# variance reduction is nonnegative for every admissible rho in [0,1]
reduction = simplify(s1**2 - var_rb)          # = s1**2 * (1 - rho**2)
assert simplify(reduction - s1**2*(1 - rho**2)) == 0
for r in [Rational(0), Rational(1,3), Rational(1,2), Rational(9,10), Rational(1)]:
    assert reduction.subs({rho: r, s1: 1, s2: 1}) >= 0
```

CLAIM c3: Law of iterated expectation $E(\theta_1\mid y)=\int E(\theta_1\mid\theta_2,y)\,\pi(\theta_2\mid y)\,d\theta_2$, verified exactly for the bivariate normal posterior.
```python
from sympy import *

t1, t2 = symbols('theta1 theta2', real=True)
rho = Rational(1, 3)
mu1, mu2 = Rational(2), Rational(-1)
s1, s2 = Integer(1), Integer(1)
s = sqrt(1 - rho**2)

cond  = (1/(sqrt(2*pi)*s)) * exp(-(t1 - (mu1 + rho*(t2 - mu2)))**2/(2*(1-rho**2)))
marg2 = (1/sqrt(2*pi)) * exp(-(t2 - mu2)**2/2)

# E(theta1 | theta2) from the conditional density
cond_mean = simplify(integrate(t1*cond, (t1, -oo, oo)))
assert simplify(cond_mean - (mu1 + rho*(t2 - mu2))) == 0

# averaging the conditional mean over pi(theta2) recovers E(theta1) = mu1
iterated = simplify(integrate(cond_mean*marg2, (t2, -oo, oo)))
assert simplify(iterated - mu1) == 0
```

CLAIM c4: The printed summand $\frac{1}{\sqrt{2\pi\sigma^2}}\exp\{-\frac{1}{2\sigma^2}[\theta_1-a]^2\}$ is the $N(a,\sigma^2)$ density: it integrates to 1, has mean $a$ and variance $\sigma^2$.

from sympy import *

t1 = symbols('theta1', real=True)
a = symbols('a', real=True)
