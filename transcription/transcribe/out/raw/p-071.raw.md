- **Method I: Numerical**

  Let

  $$F(\theta|r) = \int_{-\infty}^{\theta} f(t|y)dt$$

  $\text{[sic: the conditioning symbol should be } y\text{, i.e. } F(\theta\mid y)\text{]}$

  be the *cumuulative* [sic] *distribution function* (C.D.F.). Then

  $$a = F^{-1}\left(\frac{\alpha}{2}\Big|y\right) \quad \text{and} \quad b = F^{-1}\left(1 - \frac{\alpha}{2}\Big|y\right)$$

  give the $100(1-\alpha)\%$ credible interval $(a,b)$. Here $F^{-1}(\cdot|y)$ is the inverse C.D.F.

  > ⏳ VERIFY[c1]: for a continuous posterior, $a=F^{-1}(\alpha/2\mid y)$ and $b=F^{-1}(1-\alpha/2\mid y)$ satisfy $\int_a^b f(t\mid y)\,dt = 1-\alpha$.

- **Method II: Sampling based**

  Draw a sample of 1000 values from $f(\theta|y)$. Order the values

  $$\theta^{(1)} < \theta^{(2)} < ...\theta^{(1000)},$$

  then an estimate of the 95% credible interval is

  $$\left(\theta^{(25)}, \theta^{(975)}\right).$$

  > ⏳ VERIFY[c2]: with $n=1000$ ordered draws, the order statistics $\theta^{(25)}$ and $\theta^{(975)}$ are the empirical $\alpha/2 = 0.025$ and $1-\alpha/2 = 0.975$ quantiles, enclosing $950/1000 = 95\%$ of the sample.

  This method is usually used in complex problems.

## 8.1.2 Highest Posterior Density (H.P.D) Interval

Not only should we be concerned with the probability content of the interval, but we wish to use the interval with the highest posterior density.

===CLAIMS===

CLAIM c1: Equal-tail inversion of the C.D.F. gives probability content exactly $1-\alpha$: for a posterior with c.d.f. $F$, $\int_{F^{-1}(\alpha/2)}^{F^{-1}(1-\alpha/2)} f(t)\,dt = 1-\alpha$.
```python
import math
from scipy import stats
from scipy.integrate import quad
from sympy import symbols, simplify

# Concrete continuous posterior: Beta(2, 3) on (0,1), 90% credible interval.
# (Numeric inversion: symbolic solve of the quartic cdf equation times out.)
alpha = 0.10
dist = stats.beta(2, 3)

a = dist.ppf(alpha / 2)
b = dist.ppf(1 - alpha / 2)
assert 0 < a < b < 1

# round-trip: F(a) = alpha/2, F(b) = 1 - alpha/2
assert abs(dist.cdf(a) - alpha / 2) < 1e-12
assert abs(dist.cdf(b) - (1 - alpha / 2)) < 1e-12

content, err = quad(dist.pdf, a, b)
assert err < 1e-10
assert abs(content - (1 - alpha)) < 1e-9

# General statement: for any continuous strictly increasing cdf F,
# content = F(b) - F(a) = (1 - alpha/2) - (alpha/2) = 1 - alpha.
al = symbols('alpha', positive=True)
assert simplify((1 - al/2) - (al/2) - (1 - al)) == 0
```

CLAIM c2: With $n=1000$ ordered draws, indices 25 and 975 are the $0.025$ and $0.975$ empirical quantiles and span 95% of the sample.
```python
from sympy import Rational
import numpy as np

n = 1000
lo_idx, hi_idx = 25, 975

# The booklet's indices are exactly the alpha/2 and 1-alpha/2 sample quantiles for alpha = 0.05.
alpha = Rational(5, 100)
assert Rational(lo_idx, n) == alpha / 2          # 25/1000 == 0.025
assert Rational(hi_idx, n) == 1 - alpha / 2      # 975/1000 == 0.975

# Number of ordered values lying in [theta^(25), theta^(975)] inclusive-of-endpoints span:
assert hi_idx - lo_idx == 950
assert Rational(hi_idx - lo_idx, n) == Rational(95, 100)   # exactly 95%

# Sanity check against an actual ordered sample: the interval really does
# leave 24 points strictly below and 25 points strictly above (2.5% tails).
rng = np.random.default_rng(0)
th = np.sort(rng.normal(size=n))
a, b = th[lo_idx - 1], th[hi_idx - 1]   # 1-indexed order statistics
assert a < b
assert np.sum(th < a) == lo_idx - 1     # 24 below
assert np.sum(th > b) == n - hi_idx     # 25 above
assert np.sum((th >= a) & (th <= b)) == 951   # endpoints included
```
