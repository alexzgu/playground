**8.2.2 Marginal Likelihoods**

**Definition.**

$$\theta \in \Theta$$

$$\pi(\theta) \;\text{prior (must be proper)}$$

$$l(\theta\mid x) \propto f(x\mid\theta) \;\text{is the likelihood function.}$$

*[margin note: a large hand-drawn brace groups the three lines above; the word written beside it is illegible (possibly "one-one"(?))]*

The *Marginal likelihood* is

$$f(x) = \int_\Theta f(x\mid\theta)\pi(\theta)\,d\theta.$$

Note: $f(x\mid\theta)$ not $l(\theta\mid x)$ in the integral.

Remark (1): if $\theta$ is a point mass at $\theta_0$, then $f(x) = f(x\mid\theta_0)$.

> ⏳ VERIFY[c1]: if the prior is a point mass at $\theta_0$, the marginal likelihood $\int f(x\mid\theta)\pi(\theta)d\theta$ equals $f(x\mid\theta_0)$.

Remark (2): Generalization to multi-parameter problem is conceptually simple,

$$f(x) = \int_\Theta f(x\mid\theta)\pi(\theta)\,d\theta.$$

*(The booklet writes θ with under-tildes here to indicate vectors.)*

Remark (3): Computation of marginal likelihoods is a research activity.

**Theorem 8.2.1.** The Bayes Factor is the ratio of the marginal likelihoods of $M_1$ to $M_2$.

*Proof.* From Bayes' Theorem we have

$$P(M_1\mid x) = f(x\mid M_1)P(M_1)/f(x)$$
$$P(M_2\mid x) = f(x\mid M_2)P(M_2)/f(x)$$

for $f(x) \neq 0$ and

$$\frac{P(M_1\mid x)}{P(M_2\mid x)} = \frac{f(x\mid M_1)P(M_1)}{f(x\mid M_2)P(M_2)}$$

> ⏳ VERIFY[c2]: the posterior odds equal $f(x\mid M_1)P(M_1)/\big(f(x\mid M_2)P(M_2)\big)$.

then

$$\frac{f(x\mid M_1)}{f(x\mid M_2)} = \frac{P(M_1\mid x)/P(M_1)}{P(M_2\mid x)/P(M_2)} = \frac{\text{Posterior odds}}{\text{Prior odds}} = \text{Bayes Factor}$$

CLAIM c1: A point-mass prior at $\theta_0$ gives marginal likelihood $f(x) = f(x\mid\theta_0)$.
```python
from sympy import *

x, th = symbols('x theta', real=True)
th0 = Rational(1, 2)

# concrete likelihood f(x|theta) = N(x; theta, 1)
f_cond = exp(-(x - th)**2 / 2) / sqrt(2*pi)

# prior = point mass at th0
marginal = integrate(f_cond * DiracDelta(th - th0), (th, -oo, oo))

assert simplify(marginal - f_cond.subs(th, th0)) == 0
```

CLAIM c2: Posterior odds equal $f(x\mid M_1)P(M_1)\,/\,\big(f(x\mid M_2)P(M_2)\big)$.
```python
from sympy import *

fx1, fx2, pm1, pm2, fx = symbols('fx1 fx2 pm1 pm2 fx', positive=True)

# Bayes' theorem, as printed on the page
post1 = fx1 * pm1 / fx
post2 = fx2 * pm2 / fx

assert simplify(post1/post2 - (fx1*pm1)/(fx2*pm2)) == 0
```

CLAIM c3: The Bayes factor identity: $f(x|M_1)/f(x|M_2) = \frac{P(M_1|x)/P(M_1)}{P(M_2|x)/P(M_2)} = \text{posterior odds}/\text{prior odds}$.
```python
from sympy import *

fx1, fx2, pm1, pm2, fx = symbols('fx1 fx2 pm1 pm2 fx', positive=True)

post1 = fx1 * pm1 / fx
post2 = fx2 * pm2 / fx

# the corrected (handwritten-slash) middle expression
middle = (post1/pm1) / (post2/pm2)
assert simplify(fx1/fx2 - middle) == 0

# = posterior odds / prior odds
posterior_odds = post1 / post2
prior_odds = pm1 / pm2
assert simplify(middle - posterior_odds/prior_odds) == 0

# and the Bayes factor is the ratio of marginal likelihoods (Theorem 8.2.1)
assert simplify(posterior_odds/prior_odds - fx1/fx2) == 0

# the printed (uncorrected) product form is NOT equal -> the handwritten fix is needed
printed = (post1*pm1) / (post2*pm2)
assert simplify(fx1/fx2 - printed) != 0
```

CLAIM c4: From $f(\theta|x) = f(x|\theta)\pi(\theta)/f(x)$ it follows that $f(x) = f(x|\theta)\pi(\theta)/f(\theta|x)$.
```python
from sympy import *

fx, f_cond, prior, post = symbols('fx f_cond prior post', positive=True)

eq = Eq(post, f_cond * prior / fx)
sol = solve(eq, fx, dict=True)

assert len(sol) == 1
assert simplify(sol[0][fx] - f_cond * prior / post) == 0
```
