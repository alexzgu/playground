$$E = \int_{-\infty}^{\infty} h(x) f(x)\,dx$$

$$= \frac{\int_{-\infty}^{\infty} h(x) f(x)\,dx}{\int_{-\infty}^{\infty} f(x)\,dx}$$

$$= \frac{\int_{-\infty}^{\infty}\left[h(x)\dfrac{f(x)}{g(x)}\right]g(x)\,dx}{\int_{-\infty}^{\infty}\left[\dfrac{f(x)}{g(x)}\right]g(x)\,dx}$$

> ⏳ VERIFY[c1]: the importance-sampling identity $\int h f\,dx = \dfrac{\int [h\,f/g]\,g\,dx}{\int [f/g]\,g\,dx}$ holds when $f$ is a density.

Now for $X_1, ..., X_M \overset{iid}{\sim} g(x)$, we have a second Monte Carlo estimator:

$$\widetilde{E}_M^{(2)} = \frac{\sum_{i=1}^{M} h(x_i)\dfrac{f(x_i)}{g(x_i)}}{\sum_{i=1}^{M} \dfrac{f(x_i)}{g(x_i)}} = \sum_{i=1}^{M} w(x_i) h(x_i)$$

$$\text{Where } w(x_i) = \frac{\dfrac{f(x_i)}{g(x_i)}}{\sum_{i=1}^{M} \dfrac{f(x_i)}{g(x_i)}}, i = 1, ..., M$$

> ⏳ VERIFY[c2]: with $w(x_i)$ as defined, $\sum_i w(x_i)h(x_i)$ equals the ratio $\sum_i h(x_i)\frac{f(x_i)}{g(x_i)} \big/ \sum_i \frac{f(x_i)}{g(x_i)}$.

$$\text{And } Var(\widetilde{E}_M^{(2)}) \approx \frac{1}{M}\sum_{i=1}^{M} w(x_i)[h(x_i) - \widetilde{E}_M^{(2)}]^2$$

$\widetilde{E}_M^{(2)}$ may perform better than $\widetilde{E}_M^{(1)}$. See *Casella and Robert (1998, Bionietrika [sic])*. *[a small hand-drawn caret/hook stroke sits above "Robert"(?); no inserted text is legible]* For one thing, the denominator and numerator will be positively correlated, reducing the variance.

**5.11 How to pick the importance (proposal) density**

$$f(x) = \int f(x\mid\theta)\pi(\theta)d\theta = \int \pi^{*}(\theta\mid x)d\theta \qquad \left(\pi^{*}(\theta\mid x) = f(x\mid\theta)\pi(\theta)\right)$$

*(The booklet writes $x$ with under-tildes to indicate vectors.)*

Generally $\quad \pi(\theta\mid x), \quad g(\theta) = \log(\pi(\theta\mid x)), \quad g'(\hat{\theta}) = 0 \quad g''(\hat{\theta}) \leq 0 \quad \text{at } \hat{\theta}$

$$g(\theta) \approx g(\hat{\theta}) + (\theta - \hat{\theta})g'(\hat{\theta}) + \frac{1}{2}(\theta - \hat{\theta})^2 g''(\hat{\theta})$$

$$\pi(\theta\mid x) \overset{\text{approx}}{\propto} e^{\frac{1}{2}(\theta - \hat{\theta})^2 g''(\hat{\theta})} = e^{-\frac{1}{2}(\theta - \hat{\theta})^2 (-g''(\hat{\theta}))}$$

$$\theta\mid x \overset{\text{approx}}{\sim} N\left(\hat{\theta}, -g''(\hat{\theta})^{-1}\right), \quad \hat{\sigma}^2 = \left(-g''(\hat{\theta})\right)^{-1}$$

> ⏳ VERIFY[c3]: the Laplace expansion with $g'(\hat\theta)=0$ gives a kernel $e^{-\frac12(\theta-\hat\theta)^2(-g''(\hat\theta))}$ that normalizes to the $N(\hat\theta, (-g''(\hat\theta))^{-1})$ density.

CLAIM c1: For a density f and proposal g, ∫h f dx equals [∫ h(f/g) g dx] / [∫ (f/g) g dx].
```python
from sympy import *

x = symbols('x', real=True)
f = exp(-x**2/2)/sqrt(2*pi)          # target density
g = exp(-x**2/4)/sqrt(4*pi)          # proposal density
h = x**2

num = integrate(h*(f/g)*g, (x, -oo, oo))
den = integrate((f/g)*g, (x, -oo, oo))
lhs = integrate(h*f, (x, -oo, oo))

assert simplify(den - 1) == 0                 # denominator is ∫f = 1
assert simplify(lhs - num/den) == 0           # the stated identity
assert simplify(lhs - 1) == 0                 # sanity: Var of standard normal
```

CLAIM c2: The self-normalized weights w(x_i) = (f_i/g_i)/Σ_j(f_j/g_j) make Σ_i w_i h_i equal the ratio estimator.
```python
from sympy import *

M = 5
h = symbols('h1:6', real=True)
f = symbols('f1:6', positive=True)
g = symbols('g1:6', positive=True)

S = sum(f[i]/g[i] for i in range(M))
w = [(f[i]/g[i])/S for i in range(M)]

ratio = sum(h[i]*f[i]/g[i] for i in range(M)) / S
weighted = sum(w[i]*h[i] for i in range(M))

assert simplify(weighted - ratio) == 0
assert simplify(sum(w) - 1) == 0              # weights sum to one
```

CLAIM c3: With g'(θ̂)=0 and g''(θ̂)<0, exp(½(θ-θ̂)²g''(θ̂)) normalizes to N(θ̂, (-g''(θ̂))^{-1}).
```python
from sympy import *

th, that = symbols('theta thetahat', real=True)
s = symbols('s', positive=True)
g2 = -1/s**2                                   # g''(thetahat) < 0, so sigma^2 = -1/g'' = s^2

kernel = exp(Rational(1,2)*(th-that)**2*g2)
assert simplify(kernel - exp(-Rational(1,2)*(th-that)**2*(-g2))) == 0   # the printed rewriting

Z = integrate(kernel, (th, -oo, oo))
normal = exp(-(th-that)**2/(2*s**2))/(s*sqrt(2*pi))
assert simplify(kernel/Z - normal) == 0        # normalizes to N(thetahat, s^2)

sigma2 = (-g2)**-1
assert simplify(sigma2 - s**2) == 0            # sigmahat^2 = (-g''(thetahat))^{-1}
```
