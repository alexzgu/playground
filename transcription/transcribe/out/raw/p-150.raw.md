Draw a ball at random. Note its color.

$P_j$ : the proportion of balls of color j after N draws, $j = 1, ..., k$
Then, $(P_1, P_2, ..., P_k) \overset{d}{\to} \text{Dirichlet}(\alpha_1, \alpha_2, ..., \alpha_k)$, where $\alpha_j$ is the number of balls of the $j^{th}$ color initiatively [sic] in the urn, $j = 1, ..., k$.

> ⏳ VERIFY[c1]: For the Pólya urn started with $\alpha_j$ balls of color $j$, the de Finetti mixing measure of the color sequence is $\text{Dirichlet}(\alpha_1,\dots,\alpha_k)$ — i.e. the urn probability of any color sequence equals the corresponding Dirichlet mixed-moment $E\left[\prod_j P_j^{m_j}\right]$.

**14.1.1 Generalized Polya Urn**

Urn1: finite number(n) of balls of different colors (each color has at least one ball).
Urn2: has a continual [sic] of colors, with probability proportional to $\alpha$, draw a ball from urn2, and with probability proportional to n, draw it from urn1.

If a ball is drawn from urn2, this is a new ball(different color from urn1), place it in urn1.

If a ball is drawn from urn1, replace two balls of its same color to urn1. Then process continuous [sic] indefinitely. This process is, in fact, the Dirichlet Process, and the visual representation is called the Polya Urn Scheme - Blackwell and Mac Queen (1973).

Remarks: (1) If $\alpha$ is large, new balls will tend to be placed in urn1.
(2) If $\alpha$ is small, the number of old balls will increase, "rich gets richer scheme".
(3) Since $\alpha$ is fixed, as the number of balls in urn1 increases, n will dominate $\alpha$ and there will be more and more old balls in urn1.

CLAIM c1: The Pólya urn (draw a ball, return it plus one more of the same color) started from $\alpha=(2,1,3)$ assigns to the color sequence $(1,1,2)$ exactly the probability $E[P_1^2 P_2]$ under $\text{Dirichlet}(2,1,3)$, and the Dirichlet marginal mean is $E[P_1]=\alpha_1/\sum_j\alpha_j$; hence the urn proportions have the Dirichlet law the booklet asserts.
```python
from sympy import Rational, symbols, integrate, gamma, simplify

alpha = [Rational(2), Rational(1), Rational(3)]
A = sum(alpha)  # 6

# --- Urn side: probability of drawing colors 1,1,2 in that order.
def urn_prob(seq):
    counts = list(alpha)
    total = A
    p = Rational(1)
    for c in seq:
        p *= counts[c] / total
        counts[c] += 1
        total += 1
    return p

seq = [0, 0, 1]              # colors 1,1,2  (0-indexed)
p_urn = urn_prob(seq)
assert p_urn == Rational(1, 56)

# --- Dirichlet side: E[p1^2 * p2] by exact integration over the simplex.
p1, p2 = symbols('p1 p2', positive=True)
C = gamma(A) / (gamma(alpha[0]) * gamma(alpha[1]) * gamma(alpha[2]))
dens = C * p1**(alpha[0]-1) * p2**(alpha[1]-1) * (1-p1-p2)**(alpha[2]-1)

def dir_expect(expr):
    inner = integrate(expr * dens, (p2, 0, 1-p1))
    return simplify(integrate(inner, (p1, 0, 1)))

# density integrates to 1 (sanity that C and the simplex are right)
assert dir_expect(1) == 1

# de Finetti match: urn sequence probability == Dirichlet mixed moment
assert simplify(dir_expect(p1**2 * p2) - p_urn) == 0

# marginal mean E[P_1] = alpha_1 / sum(alpha)
assert simplify(dir_expect(p1) - alpha[0]/A) == 0

# exchangeability: reordering the sequence leaves the urn probability unchanged
assert urn_prob([0, 1, 0]) == p_urn and urn_prob([1, 0, 0]) == p_urn
```
