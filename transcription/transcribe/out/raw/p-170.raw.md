**15.2.4 Rao-Blackwellization**

Rao-Blackwellization ensures efficiency, much needed in MCMC computations. Let $\theta$ and $\phi$ be two parameters with $\theta$ the parameter of interest. Then, note that

$$\pi(\theta|\boldsymbol{y}) = \int \pi(\theta|\phi,\boldsymbol{y})\pi(\phi|\boldsymbol{y})d\phi$$

*(The booklet writes $y$ with an under-tilde to indicate a vector.)*

and

$$E(\theta|\boldsymbol{y}) = \int E(\theta|\phi,\boldsymbol{y})\pi(\phi|\boldsymbol{y})d\phi$$

> ⏳ VERIFY[c1]: the two displayed identities are the law of total probability / total expectation, marginalizing over $\phi$.

where we assume that $\pi(\theta|\phi,\boldsymbol{y})$ and $E(\theta|\phi,\boldsymbol{y})$ are in closed forms.

Suppose we have a random sample $\phi_1,\cdots,\phi_M \sim \pi(\phi|\boldsymbol{y})$. Then,

$$\hat{\pi}(\theta|\boldsymbol{y}) = \frac{1}{M}\sum_{h=1}^{M} \pi(\theta|\phi^{(h)},\boldsymbol{y})$$

is a Rao-Blackwellized density estimator, and

$$E(\hat{\theta}|\boldsymbol{y}) = \frac{1}{M}\sum_{h=1}^{M} E(\theta|\phi^{(h)},\boldsymbol{y})$$

is a Rao-Blackwellized estimator of $E(\theta|\boldsymbol{y})$.

Suppose we also have a random sample $\theta^{(1)},\cdots,\theta^{(M)}$. Estimators based on $\theta^{(1)},\cdots,\theta^{(M)}$ are not as good as Rao-Blackwellized estimators which have smaller mean squared errors (Gelfand and Smith, JASA 1990). For example, $\frac{1}{M}\sum_{h=1}^{M} E(\theta|\phi^{(h)},\boldsymbol{y})$ has smaller MSE than $\frac{1}{M}\sum_{h=1}^{M}\theta^{(h)}$.

## 15.3 Examples

**15.3.1 Generalized Gamma**

$$f(y|\alpha,\beta,\gamma) = \frac{\gamma\beta(\beta y)^{\alpha\gamma-1}e^{-(\beta y)^{\gamma}}}{\Gamma(\alpha)},\;\; y>0, \alpha,\beta,\gamma>0$$

> ⏳ VERIFY[c2]: the generalized gamma density integrates to 1 over $y \in (0,\infty)$.

1. It is very flexible and it contains many distributions. When $\gamma=1$, $Y \sim Gam(\alpha,\beta)$. It also contains Weibull, half normal, Rayleigh, lognormal, etc.

> ⏳ VERIFY[c3]: setting $\gamma=1$ reduces the density to the $Gam(\alpha,\beta)$ (rate $\beta$) density.

2. $E(Y^s) = \frac{1}{\beta^s}\frac{\Gamma(\frac{s}{\gamma}+\alpha)}{\Gamma(\alpha)}$, $\;s=1,2,\cdots$

> ⏳ VERIFY[c4]: $E(Y^s) = \beta^{-s}\,\Gamma(s/\gamma+\alpha)/\Gamma(\alpha)$ under the generalized gamma density.

3. Closed under power transformation (i.e., $Y^s$ has a generalized gamma distribution)

> ⏳ VERIFY[c5]: if $Y \sim GG(\alpha,\beta,\gamma)$ then $Y^s \sim GG(\alpha,\beta^s,\gamma/s)$.

4. $Y^{\gamma} \sim Gam(\alpha,\beta^{\gamma})$ ; it is easy to sample.

> ⏳ VERIFY[c6]: the density of $W=Y^{\gamma}$ is the $Gam(\alpha,\beta^{\gamma})$ (rate $\beta^\gamma$) density.

5. $Y_1,\cdots,Y_n \overset{iid}{\sim} GG(\alpha,\beta,\gamma)$

CLAIM c1: Marginalization identities π(θ|y)=∫π(θ|φ,y)π(φ|y)dφ and E(θ|y)=∫E(θ|φ,y)π(φ|y)dφ, checked on a concrete conjugate normal example.
```python
from sympy import *

th, ph = symbols('theta phi', real=True)

# theta | phi ~ N(phi, 1);  phi ~ N(0, 1)   (conditioning on y suppressed)
pi_th_given_ph = exp(-(th - ph)**2 / 2) / sqrt(2*pi)
pi_ph         = exp(-ph**2 / 2) / sqrt(2*pi)

# Law of total probability: marginal of theta is N(0, 2)
marg = integrate(pi_th_given_ph * pi_ph, (ph, -oo, oo))
target = exp(-th**2 / 4) / sqrt(4*pi)
assert simplify(marg - target) == 0

# Law of total expectation: E(theta) = int E(theta|phi) pi(phi) dphi
E_th_given_ph = ph                      # mean of N(phi, 1)
rhs = integrate(E_th_given_ph * pi_ph, (ph, -oo, oo))
lhs = integrate(th * target, (th, -oo, oo))
assert simplify(lhs - rhs) == 0
assert lhs == 0
```

CLAIM c2: The generalized gamma density f(y|α,β,γ) = γβ(βy)^{αγ-1} e^{-(βy)^γ} / Γ(α) integrates to 1.

from sympy import *

y = symbols('y', positive=True)

def gg(y, a, b, g):
    return g*b*(b*y)**(a*g - 1)*exp(-(b*y)**g) / gamma(a)

# Symbolic in alpha, with exact rational beta, gamma
a = symbols('alpha', positive=True)
tot = integrate(gg(y, a, Rational(3, 2), Rational(5, 2)), (y, 0, oo))

# Fully exact numeric substitutions as an independent check

    assert simplify(integrate(gg(y, av, bv, gv), (y, 0, oo)) - 1) == 0

CLAIM c4: E(Y^s) = β^{-s} Γ(s/γ + α) / Γ(α) for Y ~ GG(α,β,γ).
```python
from sympy import *

y = symbols('y', positive=True)

def gg(y, a, b, g):
    return g*b*(b*y)**(a*g - 1)*exp(-(b*y)**g) / gamma(a)

def stated(s, a, b, g):
    return gamma(Rational(s, 1)/g + a) / (b**s * gamma(a))

# Symbolic in alpha (exact rational beta, gamma), for s = 1, 2, 3
a = symbols('alpha', positive=True)
bv, gv = Rational(3, 2), Rational(5, 2)
for s in [1, 2, 3]:
    lhs = integrate(y**s * gg(y, a, bv, gv), (y, 0, oo))
    assert simplify(lhs - stated(s, a, bv, gv)) == 0

# Exact numeric parameter substitutions
for av, bv, gv in [(Rational(2), Rational(1), Rational(1)),
                   (Rational(5, 2), Rational(7, 3), Rational(1, 2))]:
    for s in [1, 2]:
        lhs = integrate(y**s * gg(y, av, bv, gv), (y, 0, oo))
        assert simplify(lhs - stated(s, av, bv, gv)) == 0
```

CLAIM c5: Closure under power transformation: if Y ~ GG(α,β,γ) then Z = Y^s ~ GG(α, β^s, γ/s).
```python
from sympy import *

z = symbols('z', positive=True)
a, b, g, s = symbols('alpha beta gamma s', positive=True)

def gg(y, a, b, g):
    return g*b*(b*y)**(a*g - 1)*exp(-(b*y)**g) / gamma(a)

# Change of variables z = y**s  =>  y = z**(1/s),  dy = (1/s) z**(1/s - 1) dz
y_of_z = z**(1/s)
jac = diff(y_of_z, z)
induced = gg(y_of_z, a, b, g) * jac

# Claimed law: GG(alpha, beta**s, gamma/s)
claimed = gg(z, a, b**s, g/s)

assert simplify(powsimp(expand_power_exp(induced - claimed), force=True)) == 0
```

CLAIM c6: W = Y^γ ~ Gam(α, β^γ), i.e. W has density (β^γ)^α w^{α-1} e^{-β^γ w} / Γ(α).
```python
from sympy import *

w = symbols('w', positive=True)
a, b, g = symbols('alpha beta gamma', positive=True)

def gg(y, a, b, g):
    return g*b*(b*y)**(a*g - 1)*exp(-(b*y)**g) / gamma(a)

# w = y**gamma  =>  y = w**(1/gamma),  dy = (1/gamma) w**(1/gamma - 1) dw
y_of_w = w**(1/g)
induced = gg(y_of_w, a, b, g) * diff(y_of_w, w)

rate = b**g
gamma_pdf = rate**a * w**(a - 1) * exp(-rate*w) / gamma(a)

assert simplify(powsimp(expand_power_exp(induced - gamma_pdf), force=True)) == 0

# Consistency with c5 at s = gamma: GG(alpha, beta**gamma, 1) is Gam(alpha, beta**gamma)
assert simplify(gg(w, a, b**g, 1) - gamma_pdf) == 0
```
