**14.2 Two more formulations of the Dirichlet Process**

We have discussed extensively the Polya Urn Scheme (Chinese Restaurant Process). There are two more formulations. The three are equivalent.

(1) Mixture
Ishwaran and Zarepoin(2002) [sic: Zarepour]

$$G^L(x) = \sum_{j=1}^{L} \omega'_j S_{X_j}(x), \quad \sum_{j=1}^{L} \omega'_j = 1$$

$$X_j \sim G_0$$

$$(\omega_1,\ldots,\omega_L) \sim \mathrm{Dir}\left(\tfrac{\alpha}{L},\ldots \tfrac{\alpha}{L}\right)$$

*(The point mass is written $S_{X_j}$ here, but $\delta_{X_j}$ in equation (1) below.)*

For any integrable function $f(x)$ with respect to $G_0$:

$$\int f(x)\,dG^{(L)}(x) \overset{d}{\to} \int f(x)\,dG(x), \quad \text{as } L \to \infty$$

That is,

$$G^{(L)} \to G, \quad \text{as } L \to \infty$$

(2) Stick breaking
Setheraman(1994) [sic: Sethuraman]

$$G(x) = \sum_{j=1}^{\infty} \omega_j \delta_{X_j}(x), \quad \sum_{j=1}^{\infty} \omega_j = 1 \tag{1}$$

$$\omega_1 = \gamma_1, \quad \omega_2 = \gamma_2(1-\gamma_1), \quad \ldots, \quad \omega_j = \gamma_j \prod_{i=1}^{j-1}(1-\gamma_i) \tag{2}$$

where,

$$\gamma_j \overset{iid}{\sim} \mathrm{Beta}(1,\alpha)$$

$$W \sim \mathrm{GEM}(\alpha)$$

$$\text{(Griffiths, Eagen and McCloskey)} \quad \text{[sic: Engen]}$$

> ⏳ VERIFY[c1]: the stick-breaking partial sums telescope, $\sum_{j=1}^{J}\omega_j = 1 - \prod_{i=1}^{J}(1-\gamma_i)$, so $\sum_{j=1}^{\infty}\omega_j = 1$ whenever $\gamma_i \in (0,1)$.

> ⏳ VERIFY[c2]: with $\gamma_j \overset{iid}{\sim} \mathrm{Beta}(1,\alpha)$, $E[\omega_j] = \frac{1}{1+\alpha}\left(\frac{\alpha}{1+\alpha}\right)^{j-1}$ and $\sum_{j=1}^{\infty} E[\omega_j] = 1$.

$W$ is a random probability measure defined by (1) and (2).

CLAIM c1: Stick-breaking partial sums telescope to $1-\prod_{i=1}^{J}(1-\gamma_i)$, hence the weights sum to 1.
```python
from sympy import *

J = 8
gam = symbols('gamma1:%d' % (J + 1), positive=True)

# omega_j = gamma_j * prod_{i<j} (1 - gamma_i), per equation (2)
omega = [gam[j] * prod([1 - gam[i] for i in range(j)]) for j in range(J)]

partial = Add(*omega)
closed = 1 - prod([1 - gam[i] for i in range(J)])
assert simplify(expand(partial - closed)) == 0

# Tail vanishes: prod_{i=1}^{J}(1-g) -> 0 for any fixed g in (0,1),
# so the infinite sum in eq. (1) equals 1.
g, n = symbols('g n', positive=True)
assert limit((1 - Rational(1, 3)) ** n, n, oo) == 0
assert simplify(partial.subs({s: Rational(1, 3) for s in gam})
                - (1 - Rational(2, 3) ** J)) == 0
```

CLAIM c2: For $\gamma_j\overset{iid}{\sim}\mathrm{Beta}(1,\alpha)$, $E[\omega_j]=\frac{1}{1+\alpha}(\frac{\alpha}{1+\alpha})^{j-1}$ and these expectations sum to 1.
```python
from sympy import *

alpha, x = symbols('alpha x', positive=True)
j = symbols('j', integer=True, positive=True)

# Beta(1, alpha) density and its mean
dens = x ** (1 - 1) * (1 - x) ** (alpha - 1) / beta(1, alpha)
assert simplify(integrate(dens, (x, 0, 1)) - 1) == 0
m = simplify(integrate(x * dens, (x, 0, 1)))
assert simplify(m - 1 / (1 + alpha)) == 0

m1 = simplify(integrate((1 - x) * dens, (x, 0, 1)))   # E[1 - gamma]
assert simplify(m1 - alpha / (1 + alpha)) == 0

# Independence => E[omega_j] = E[gamma] * E[1-gamma]^(j-1)
Eom = m * m1 ** (j - 1)
assert simplify(Eom - (1 / (1 + alpha)) * (alpha / (1 + alpha)) ** (j - 1)) == 0

total = simplify(summation(Eom, (j, 1, oo)))
assert simplify(total - 1) == 0
```
