CLAIM c1: The chi-square density with $n-1$ degrees of freedom equals the Gamma density with shape $(n-1)/2$ and rate $1/2$.
```python
from sympy import *

x = symbols('x', positive=True)
for nu in [1, 2, 3, 5, 7]:
    chi2 = x**(Rational(nu,2)-1) * exp(-x/2) / (2**Rational(nu,2) * gamma(Rational(nu,2)))
    # Gam(shape=nu/2, rate=1/2)
    a, b = Rational(nu,2), Rational(1,2)
    gam = b**a / gamma(a) * x**(a-1) * exp(-b*x)
    assert simplify(chi2 - gam) == 0
    assert simplify(integrate(gam, (x, 0, oo)) - 1) == 0
```

CLAIM c2: Sum-of-squares decomposition $\sum (x_i-\mu)^2 = (n-1)s^2 + n(\bar{x}-\mu)^2$, and the resulting likelihood form.
```python
from sympy import *

mu, sigma2 = symbols('mu sigma2', real=True, positive=True)
mu = symbols('mu', real=True)

for n in [2, 3, 4, 5]:
    xs = symbols('x1:%d' % (n+1), real=True)
    xbar = sum(xs)/n
    s2 = sum((xi - xbar)**2 for xi in xs)/(n-1)
    lhs = sum((xi - mu)**2 for xi in xs)
    rhs = (n-1)*s2 + n*(xbar - mu)**2
    assert simplify(expand(lhs - rhs)) == 0

# likelihood: product of normal densities equals stated expression
n = 4
sig2 = symbols('sigma2', positive=True)
xs = symbols('x1:5', real=True)
xbar = sum(xs)/n
s2 = sum((xi - xbar)**2 for xi in xs)/(n-1)
prod = 1
for xi in xs:
    prod *= 1/sqrt(2*pi*sig2) * exp(-(xi-mu)**2/(2*sig2))
stated = (1/(2*pi*sig2))**Rational(n,2) * exp(-((n-1)*s2 + n*(xbar-mu)**2)/(2*sig2))
assert simplify(prod/stated - 1) == 0
```

CLAIM c3: prior $1/\sigma^2$ times the likelihood is proportional to $(1/\sigma^2)^{n/2+1}\exp\{-\frac{1}{2\sigma^2}[(n-1)s^2+n(\bar x-\mu)^2]\}$, with proportionality constant free of $\mu,\sigma^2$.
```python
from sympy import *

mu, xbar = symbols('mu xbar', real=True)
sig2, s2 = symbols('sigma2 s2', positive=True)
n = symbols('n', positive=True, integer=True)

Q = (n-1)*s2 + n*(xbar-mu)**2
prior = 1/sig2
L = (1/(2*pi*sig2))**(n/2) * exp(-Q/(2*sig2))
stated = (1/sig2)**(n/2 + 1) * exp(-Q/(2*sig2))

ratio = simplify(prior*L/stated)
assert simplify(ratio - (2*pi)**(-n/2)) == 0
assert simplify(diff(ratio, mu)) == 0
assert simplify(diff(ratio, sig2)) == 0
```

CLAIM c4: $\exp\{-\frac{n}{2\sigma^2}(\bar x-\mu)^2\}$ is the kernel of a $N(\bar x,\sigma^2/n)$ density in $\mu$.
```python
from sympy import *

mu, xbar = symbols('mu xbar', real=True)
sig2 = symbols('sigma2', positive=True)

for nval in [1, 2, 3, 7]:
    kernel = exp(-Rational(nval,2)/sig2 * (xbar-mu)**2)
    Z = integrate(kernel, (mu, -oo, oo))
    var = sig2/nval
    dens = 1/sqrt(2*pi*var) * exp(-(mu-xbar)**2/(2*var))
    assert simplify(kernel/Z - dens) == 0
```
