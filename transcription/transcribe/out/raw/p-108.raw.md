**10.2.3 Monte Carlo error**

The Monte Carlo error, also called the numerical standard error (NSE), gives the variation that can be expected in the estimate if the computation were to be done again.

Batch means method

$$\underbrace{\theta^{(1)},...,\theta^{(\ell)}}_{\bar{\theta}_1=\ell^{-1}\sum_{k=1}^{\ell}\theta^{(k)}}\;,\;\underbrace{\theta^{(\ell+1)},...,\theta^{(2\ell)}}_{\bar{\theta}_2}\;,\;\underbrace{\theta^{(2\ell+1)},...,\theta^{(3\ell)}}_{\bar{\theta}_3}\;,\;...,\;\underbrace{\theta^{((n-1)\ell)},...,\theta^{(n\ell)}}_{\bar{\theta}_n=\ell^{-1}\sum_{k=(n-1)\ell}^{n\ell}\theta^{(k)}}$$

$\text{[sic: the last batch is printed as }\theta^{((n-1)\ell)},...,\theta^{(n\ell)}\text{ with the sum running from }k=(n-1)\ell\text{; these limits enclose }\ell+1\text{ terms, so for batches of equal length }\ell\text{ the lower limit should be }(n-1)\ell+1\text{]}$

> ⏳ VERIFY[c1]: the printed last-batch index range $k=(n-1)\ell,\dots,n\ell$ contains $\ell+1$ terms, whereas equal batches of length $\ell$ require $k=(n-1)\ell+1,\dots,n\ell$.

$n$ batches of equal length $\ell$.

$$NSE = \sqrt{\frac{\sum_{k=1}^{n}(\bar{\theta}_k - \bar{\theta})^2}{n(n-1)}}, \text{ where } \bar{\theta} = n^{-1}\sum_{k=1}^{n}\bar{\theta}_k$$

> ⏳ VERIFY[c2]: $NSE$ as printed equals $s/\sqrt{n}$, where $s^2 = (n-1)^{-1}\sum_{k=1}^{n}(\bar{\theta}_k-\bar{\theta})^2$ is the sample variance of the batch means — i.e. it is the standard error of the mean of the $n$ batch means.

`Remark` : The $NSE$ gives an idea of the accuracy of the computations. Note that the $NSE$ is not the standard error of one of the parameters in the model!

**10.2.4 Geweke test (Geweke 1992)**

Gives a p-value for a test of stationarity. Checks whether the early part of the chain differs from the later part.

Let $\bar{X}_1, S_1^2$ denote the sample mean and variance of the early part for $n_1$ samples.

Let $\bar{X}_2, S_2^2$ denote the sample mean and variance of the later part for $n_2$ samples.

Compute

$$T = \frac{\bar{X}_1 - \bar{X}_2}{\sqrt{\frac{S_1^2}{n_1} + \frac{S_2^2}{n_2}}}$$

For large $n_1$ and $n_2$, $T \sim N(0,1)$

> ⏳ VERIFY[c3]: if $\bar{X}_1$ and $\bar{X}_2$ are independent with common mean $\mu$ and variances $\sigma_1^2/n_1$ and $\sigma_2^2/n_2$, then $(\bar{X}_1-\bar{X}_2)/\sqrt{\sigma_1^2/n_1+\sigma_2^2/n_2}$ has mean $0$ and variance $1$, so $T \sim N(0,1)$ once $S_i^2 \to \sigma_i^2$.

Compute p-value (want large p-value).

`Note(1)`: $S_1^2$ and $S_2^2$ need to be adjusted because the two samples are not independent. Geweke's test uses spectral densities and gamma regression to estimate the sample variances.

`Note(2)`: Geweke's test can be used to determine the burn-in period (the smallest early part of the chain that passes the test).

CLAIM c1: The printed last-batch summation limits $k=(n-1)\ell$ to $n\ell$ enclose $\ell+1$ terms, not $\ell$; the corrected lower limit $(n-1)\ell+1$ gives exactly $\ell$.
```python
from sympy import symbols, simplify

n, l = symbols('n ell', positive=True, integer=True)

# number of integers in [a, b] inclusive is b - a + 1
printed_count   = (n*l) - ((n-1)*l) + 1          # lower limit as printed: (n-1)*l
corrected_count = (n*l) - ((n-1)*l + 1) + 1      # lower limit corrected: (n-1)*l + 1

assert simplify(printed_count - (l + 1)) == 0    # printed limits give l+1 terms
assert simplify(printed_count - l) != 0          # ... hence NOT a batch of length l
assert simplify(corrected_count - l) == 0        # corrected limits give exactly l terms

# sanity: the first batch, k = 1..l, as printed, does have length l
assert simplify((l - 1 + 1) - l) == 0
```

CLAIM c3: Geweke's T is standardized: with independent Xbar_1, Xbar_2 of common mean mu and variances sigma_1^2/n_1, sigma_2^2/n_2, the statistic has mean 0 and variance 1.
```python
from sympy import symbols, sqrt, simplify, exp, I

mu = symbols('mu', real=True)
s1, s2 = symbols('sigma_1 sigma_2', positive=True)
n1, n2 = symbols('n_1 n_2', positive=True)
t = symbols('t', real=True)

v1, v2 = s1**2 / n1, s2**2 / n2      # Var(Xbar_1), Var(Xbar_2)
c = sqrt(v1 + v2)                    # denominator of T

# mean and variance of T = (X1 - X2)/c by linearity and independence
mean_T = (mu - mu) / c
var_T = (v1 + v2) / c**2
assert simplify(mean_T) == 0
assert simplify(var_T - 1) == 0

# characteristic function of N(m, v): exp(i t m - v t^2 / 2)
def cf_normal(t, m, v):
    return exp(I*t*m - v*t**2/2)

# X1 - X2 independent normals => cf product; T scales the argument by 1/c
cf_T = cf_normal(t/c, mu, v1) * cf_normal(-t/c, mu, v2)
assert simplify(cf_T - exp(-t**2/2)) == 0   # exactly the N(0,1) characteristic function
```
