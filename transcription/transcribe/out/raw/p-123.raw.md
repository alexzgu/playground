**11.2 Convergence**

Convergence to the target density $\pi(x)$ occurs under mild regularity condition [sic]: The Markov chain must be ergodic.

Ergodic *[the heading "Ergodic" is hand-underlined with a wavy line]*

(1) Irreducible: Let $x_1$ and $x_2$ be any two points in the domain of $\pi(x)$. The chain must go from $x_1 \to x_2$ and $x_2 \to x_1$ with positive probability in a finite number of steps. *[the phrase "finite number of steps" is hand-underlined]*

(2) Aperiodic: The number of moves required to go from $x_1 \to x_2$ is not a multiple of some integer(i.e.,period=1). *[the word "integer" is hand-underlined]*

Generally, the regularity condition is met if $q(x,y)$ has a positive density on the same support as $\pi(x)$.

Special cases *[margin mark: a short unreadable dash (?) in the left margin here]*

CLAIM c1: For a symmetric candidate density, q(x,y)=q(y,x), the MH acceptance ratio π(y)q(y,x)/(π(x)q(x,y)) equals π(y)/π(x).
```python
from sympy import *

x, y = symbols('x y', real=True)
pi_ = Function('pi', positive=True)
q = Function('q', positive=True)

# Symmetric proposal: impose q(x,y) == q(y,x) by using a single symmetric expression.
qs = Function('qs', positive=True)          # qs(a,b) stands for the symmetric kernel
# Model symmetry concretely: q(a,b) = s(a)*s(b) is symmetric in (a,b) but non-trivial.
s = Function('s', positive=True)
q_xy = s(x)*s(y)
q_yx = s(y)*s(x)
assert simplify(q_xy - q_yx) == 0            # the proposal really is symmetric

ratio = pi_(y)*q_yx / (pi_(x)*q_xy)
assert simplify(ratio - pi_(y)/pi_(x)) == 0

# Hence the acceptance probability collapses to min{1, pi(y)/pi(x)}.
alpha_general = Min(1, ratio)
alpha_reduced = Min(1, pi_(y)/pi_(x))
assert simplify(alpha_general - alpha_reduced) == 0

# Decisive numeric spot-check with exact rationals.
for pv_y, pv_x, sv_x, sv_y in [(Rational(1,3), Rational(2,5), Rational(7,2), Rational(5,9)),
                               (Rational(9,4), Rational(1,8), Rational(1,6), Rational(11,3))]:
    r = pv_y*(sv_y*sv_x) / (pv_x*(sv_x*sv_y))
    assert r == pv_y/pv_x
```

from sympy import *

x, y = symbols('x y', real=True)
phi = Function('phi', positive=True)
h = Function('h', positive=True)

pi_x = phi(x)*h(x)
pi_y = phi(y)*h(y)
q_xy = h(y)        # proposal from x to y does not depend on x
q_yx = h(x)        # proposal from y to x

ratio = (pi_y*q_yx) / (pi_x*q_xy)
assert simplify(ratio - phi(y)/phi(x)) == 0

alpha_general = Min(1, ratio)
alpha_booklet = Min(phi(y)/phi(x), 1)
assert simplify(alpha_general - alpha_booklet) == 0
