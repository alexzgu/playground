Note that

$$Var(\widetilde{E}_M^{(1)}) = \frac{1}{M}\left\{ E_g[\frac{h(x)f(x)}{g(x)}]^2 - [E_g(\frac{h(x)f(x)}{g(x)})]^2 \right\}$$

> ⏳ VERIFY[c1]: the variance of the importance-sampling estimator equals $\frac{1}{M}\{E_g[W^2] - (E_g[W])^2\}$ with $W = hf/g$.

So for finite variance

$$
\begin{aligned}
E_g[\frac{h(x)f(x)}{g(x)}]^2 &= \int_{-\infty}^{\infty} [\frac{h(x)f(x)}{g(x)}]^2 g(x)dx \\
&= \int_{-\infty}^{\infty} \frac{h^2(x)f^2(x)}{g(x)}dx \\
&= \int_{-\infty}^{\infty} h^2(x)[\frac{f(x)}{g(x)}]f(x)dx \\
&= E\left\{ h^2(x)[\frac{f(x)}{g(x)}] \right\} < \infty
\end{aligned}
$$

> ⏳ VERIFY[c2]: the four-step chain $\int [hf/g]^2 g = \int h^2 f^2/g = \int h^2 (f/g) f = E_f\{h^2 (f/g)\}$ holds.

If the ratio $\frac{f(x)}{g(x)}$ is unbounded, $h^2(x)\frac{f(x)}{g(x)}$ will vary too much, making the estimator $\widetilde{E}_M^{(1)}$ inefficient.

We can obtain a slightly better estimator than $\widetilde{E}_M^{(1)}$ by observing the following:

===CLAIMS===

CLAIM c1: Var of the M-sample importance-sampling mean equals (1/M){E_g[W^2] − (E_g[W])^2}, W = h·f/g; verified exactly at M=2 with f=Exp(1), g=Exp(1/2), h(x)=x.
```python
from sympy import *

x1, x2 = symbols('x1 x2', positive=True)

f = lambda t: exp(-t)                # Exp(1) density
g = lambda t: Rational(1, 2) * exp(-t / 2)   # Exp(1/2) density (heavier tails)
h = lambda t: t
W = lambda t: h(t) * f(t) / g(t)

# Moments of W under g
EW  = integrate(W(x1) * g(x1), (x1, 0, oo))
EW2 = integrate(W(x1)**2 * g(x1), (x1, 0, oo))

# Direct Var of the M=2 estimator (W(x1)+W(x2))/2 under independent g-sampling
est = (W(x1) + W(x2)) / 2
joint = g(x1) * g(x2)
E_est  = integrate(integrate(est * joint, (x1, 0, oo)), (x2, 0, oo))
E_est2 = integrate(integrate(est**2 * joint, (x1, 0, oo)), (x2, 0, oo))
var_direct = simplify(E_est2 - E_est**2)

var_formula = simplify(Rational(1, 2) * (EW2 - EW**2))

assert simplify(var_direct - var_formula) == 0
# and the estimator is unbiased for E_f[h]
assert simplify(E_est - integrate(h(x1) * f(x1), (x1, 0, oo))) == 0
```

CLAIM c2: the identity chain ∫[hf/g]^2 g dx = ∫ h^2 f^2/g dx = ∫ h^2 (f/g) f dx = E_f{h^2 (f/g)}, finite when f/g is bounded.
```python
from sympy import *

x = symbols('x', positive=True)

f = exp(-x)                     # Exp(1)
g = Rational(1, 2) * exp(-x / 2)  # Exp(1/2)
h = x

# The four expressions in the booklet's chain
e1 = integrate((h * f / g)**2 * g, (x, 0, oo))   # E_g[(hf/g)^2]
e2 = integrate(h**2 * f**2 / g, (x, 0, oo))
e3 = integrate(h**2 * (f / g) * f, (x, 0, oo))
e4 = integrate(h**2 * (f / g) * f, (x, 0, oo))   # E_f{h^2 (f/g)}

assert simplify(e1 - e2) == 0
assert simplify(e2 - e3) == 0
assert simplify(e3 - e4) == 0
assert e1.is_finite and e1 > 0

# and the finiteness hinges on sup_x f/g < infinity, as the page asserts
ratio = simplify(f / g)
assert simplify(limit(ratio, x, oo)) == 0        # ratio decays: g has heavier tails
assert simplify(maximum(ratio, x, Interval(0, oo))) == 2   # bounded, B = 2 < oo
```
