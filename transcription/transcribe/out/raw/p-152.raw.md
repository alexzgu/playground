CLAIM c1: For $X_i \mid G \overset{iid}{\sim} G$, $G \sim DP(\alpha, G_0)$, the correlation $\text{Cor}(X_i, X_j) = 1/(1+\alpha)$ for $i \neq j$.
```python
from sympy import *

# Discretize G_0 on k=3 atoms x1,x2,x3 with base weights p1,p2,p3.
# Then (P1,P2,P3) ~ Dirichlet(alpha*p1, alpha*p2, alpha*p3), and G = sum P_i delta_{x_i}.
alpha = symbols('alpha', positive=True)
p1, p2 = symbols('p1 p2', positive=True)
p3 = 1 - p1 - p2
x1, x2, x3 = symbols('x1 x2 x3', real=True)

p = [p1, p2, p3]
x = [x1, x2, x3]

# Dirichlet covariance with concentration a0 = alpha and means p_i:
#   Cov(P_i,P_j) = (delta_ij*p_i - p_i*p_j)/(alpha+1)
def cov_P(i, j):
    return ((p[i] if i == j else 0) - p[i]*p[j]) / (alpha + 1)

# m(G) = E[X | G] = sum P_i x_i
# Cov(X_i, X_j) = Var(m(G))  for i != j   (since X_i, X_j indep given G)
var_m = sum(x[i]*x[j]*cov_P(i, j) for i in range(3) for j in range(3))

mu0 = sum(p[i]*x[i] for i in range(3))                 # E[X_i]
sigma0_sq = sum(p[i]*x[i]**2 for i in range(3)) - mu0**2  # Var(X_i) = Var under G_0

# Step 1: Var(m(G)) = sigma0^2 / (1 + alpha)
assert simplify(var_m - sigma0_sq/(1 + alpha)) == 0

# Step 2: Var(X_i) = E[Var(X|G)] + Var(m(G)) = sigma0^2  (total variance)
E_var_given_G = sum(p[i]*x[i]**2 for i in range(3)) - sum(
    x[i]*x[j]*(cov_P(i, j) + p[i]*p[j]) for i in range(3) for j in range(3))
var_X = simplify(E_var_given_G + var_m)
assert simplify(var_X - sigma0_sq) == 0

# Step 3: Cor(X_i, X_j) = Cov/Var = 1/(1+alpha)
cor = simplify(var_m / var_X)
assert simplify(cor - 1/(1 + alpha)) == 0
```

CLAIM c2: The Pólya-urn predictive mixture weights for $X_2 \mid x_1$ and $X_3 \mid x_1,x_2$ each sum to 1.
```python
from sympy import *

alpha = symbols('alpha', positive=True)

# X_2 | x_1 : weight 1/(alpha+1) on delta_{x_1}, alpha/(alpha+1) on G_0
w2 = Rational(1)/(alpha + 1) + alpha/(alpha + 1)
assert simplify(w2 - 1) == 0

w3 = 2*(Rational(1)/(alpha + 2)) + alpha/(alpha + 2)
```

CLAIM c3: The DP posterior base measure $\frac{1}{\alpha+n}\sum_{i=1}^n \delta_{x_i} + \frac{\alpha}{\alpha+n}G_0$ is a probability measure, and its precision is $\alpha+n$.

```python
alpha = symbols('alpha', positive=True)
n = symbols('n', positive=True, integer=True)

# Total mass of the posterior base measure over the whole space:
# each delta_{x_i} has mass 1 (n of them), G_0 has mass 1.
total_mass = n * (1/(alpha + n)) + alpha/(alpha + n) * 1
assert simplify(total_mass - 1) == 0

# The unnormalized posterior measure is alpha*G_0 + sum_i delta_{x_i},
# whose total mass -- the posterior precision -- is alpha + n.

assert simplify(precision - (alpha + n)) == 0

# Consistency with c2: the n=1 posterior base measure reproduces the
```
