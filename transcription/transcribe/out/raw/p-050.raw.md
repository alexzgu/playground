## 5.4 Big idea

Have $\pi(\theta\mid x)$, interest in $\pi(\theta_1\mid x)$.

$$\pi(\theta_1\mid x) = \int \ldots \int \pi(\theta_1, \ldots, \theta_{10000}\mid x)\, d\theta_2 \ldots d\theta_{10000}$$

Have a sample from the joint posterior density of $\theta_1, \ldots, \theta_{10000}$. A sample from the posterior density of $\theta_1$ is $\theta_{11}, \ldots, \theta_{1M}$ and so on.

CLAIM c1: The binomial–uniform marginal integral equals the binomial coefficient times the Beta function.
```python
from sympy import *

p = symbols('p', positive=True)
for n in range(0, 7):
    for x in range(0, n + 1):
        lhs = integrate(binomial(n, x) * p**x * (1 - p)**(n - x), (p, 0, 1))
        rhs = binomial(n, x) * beta(x + 1, n - x + 1)
        # rewrite beta via gamma so it evaluates at integer arguments
        assert simplify((lhs - rhs).rewrite(gamma)) == 0, (n, x)
```

CLAIM c2: The factorial cancellation gives 1/(n+1) for every x = 0,...,n.
```python
from sympy import *

n, x = symbols('n x', integer=True, nonnegative=True)
expr = factorial(n) / (factorial(x) * factorial(n - x)) * (factorial(x) * factorial(n - x) / factorial(n + 1))
assert simplify(expr - 1 / (n + 1)) == 0

# and concretely, independent of x
for nn in range(0, 8):
    for xx in range(0, nn + 1):
        val = Rational(factorial(nn), factorial(xx) * factorial(nn - xx)) * Rational(
            factorial(xx) * factorial(nn - xx), factorial(nn + 1))
        assert val == Rational(1, nn + 1), (nn, xx)
```

CLAIM c3: Completing the square in mu with lambda = n*delta^2/(sigma^2 + n*delta^2).
```python
from sympy import *

mu, xbar, theta = symbols('mu xbar theta', real=True)
n, s2, d2 = symbols('n sigma2 delta2', positive=True)

lam = n * d2 / (s2 + n * d2)

lhs = n / (2 * s2) * (mu - xbar)**2 + 1 / (2 * d2) * (mu - theta)**2
rhs = (1 / (2 * (1 - lam) * d2) * (mu - (lam * xbar + (1 - lam) * theta))**2
       + lam / (2 * d2) * (xbar - theta)**2)

assert simplify(expand(lhs - rhs)) == 0
```

from sympy import *

mu = symbols('mu', real=True)

# exact rational parameter values; the symbolic mu-integral closes for these
n, s2, d2, s_sq, xbar, theta = 3, Rational(2), Rational(5), Rational(7, 4), Rational(1), Rational(0)
lam = n * d2 / (s2 + n * d2)

integrand = ((1 / (2 * pi * s2))**Rational(n, 2)
             * exp(-(n - 1) * s_sq / (2 * s2))
             * exp(-n * (mu - xbar)**2 / (2 * s2))
             * 1 / sqrt(2 * pi * d2)
             * exp(-(mu - theta)**2 / (2 * d2)))

lhs = integrate(integrand, (mu, -oo, oo))

rhs = ((1 / (2 * pi * s2))**Rational(n, 2)
       * 1 / sqrt(2 * pi * d2)
       * exp(-(n - 1) * s_sq / (2 * s2))
