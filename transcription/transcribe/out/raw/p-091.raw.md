## 9.2 Small Area Estimation

The pooling of data across clusters takes places [sic] *(a handwritten stroke crosses the trailing "s", apparently correcting to "takes place")* when $\theta$ is being estimated (i.e., $\theta$ is integrated out). This induces a correlation across means a posteriori. This phenomenon is called a borrowing of strength *[hand-underlined, with a hook drawn into the left margin]*. In fact, this is the key idea in small area estimation. For a review see Pfeffermann (SS, 2013).

Typically only a small sample size is available for many areas, the small areas. The estimators for the small area effects, the random effects in the current example, become very unstable. Thus, there is a need to borrow strength from the ensemble *[hand-underlined with a wavy line]*. This also creates a need for good prior information. Where is that information in this example?

### 9.2.1 Form of estimation

Squared error Loss:

$$E\left\{\left(\mu_i - t(\boldsymbol{y})\right)^2 |\boldsymbol{y}\right\}$$

This is minimized when $t(\boldsymbol{y}) = E(\mu_i|\boldsymbol{y})$ (i.e., , [sic: doubled comma] the posterior mean of $\mu_i|\boldsymbol{y}$ ). This is the reason that $E(\mu_i|\boldsymbol{y})$ is called the Bayes estimator of $\mu_i$

> ⏳ VERIFY[c4]: $E\{(\mu_i-t)^2\mid\boldsymbol y\}$ is uniquely minimized over constants $t$ at $t=E(\mu_i\mid\boldsymbol y)$.

$$E\left(\mu_i|\boldsymbol{y}\right) = \lambda_i \bar{y}_i + (1-\lambda_i)\hat{\theta}$$

$$\lambda_i = \frac{\delta^2}{\delta^2 + \sigma_i^2/n_i}$$

$$\hat{\theta} = \frac{\sum_{i=1}^{l} \lambda_i \bar{y}_i}{\sum_{i=1}^{1} \lambda_i}$$

*[sic: the upper limit of the denominator's sum is printed as "1"; it should be $l$, as in the numerator.]*

> ⏳ VERIFY[c5]: $\hat\theta=\sum_i\lambda_i\bar y_i/\sum_i\lambda_i$ is the BLUE of $\theta$, i.e. the minimum-variance linear unbiased combination of the $\bar y_i$, whose variances are $\delta^2+\sigma_i^2/n_i=\delta^2/\lambda_i$.

$$\bar{y}_i = \frac{\sum_{j=1}^{n_i} y_{ij}}{n_i}$$

CLAIM c1: Normal–normal conjugate update: p(ȳ|μ)p(μ|θ) = p(ȳ|θ) · N(μ; λȳ+(1−λ)θ, (1−λ)δ²) with λ = δ²/(δ²+σ²/n).
```python
from sympy import *

mu, th, yb = symbols('mu theta ybar', real=True)
d, s, n = symbols('delta sigma n', positive=True)

tau = s**2/n                      # sampling variance of ybar
lam = d**2/(d**2 + tau)
M   = lam*yb + (1 - lam)*th       # claimed posterior mean
V   = (1 - lam)*d**2              # claimed posterior variance

# exponents of  p(ybar|mu) p(mu|theta)   vs   p(ybar|theta) p(mu|ybar,theta)
lhs_exp = -(yb - mu)**2/(2*tau) - (mu - th)**2/(2*d**2)
rhs_exp = -(yb - th)**2/(2*(d**2 + tau)) - (mu - M)**2/(2*V)
assert simplify(lhs_exp - rhs_exp) == 0

# normalizing constants must match too
lhs_norm = sqrt(2*pi*tau)*sqrt(2*pi*d**2)
rhs_norm = sqrt(2*pi*(d**2 + tau))*sqrt(2*pi*V)
assert simplify(lhs_norm - rhs_norm) == 0
```

CLAIM c2: Marginalizing θ|y ~ N(θ̂, ν²) yields μ_i|y ~ N(λȳ+(1−λ)θ̂, (1−λ)²ν² + (1−λ)δ²).

```python
mu, th = symbols('mu theta', real=True)

def N(x, m, v):
    return exp(-(x - m)**2/(2*v))/sqrt(2*pi*v)

# exact rational parameters
lam, d2, nu2, yb, that = Rational(1,3), Integer(1), Rational(1,4), Integer(2), Integer(1)
V = (1 - lam)*d2                                  # = 2/3

inner = N(mu, lam*yb + (1 - lam)*th, V) * N(th, that, nu2)
got   = simplify(integrate(inner, (th, -oo, oo)))

M_claim = lam*yb + (1 - lam)*that                 # = 4/3
```

CLAIM c3: Cov(μ_i, μ_i'|y) = (1−λ_i)(1−λ_i')ν² for i ≠ i'.

```python
from sympy import *
from sympy.stats import Normal, E, covariance, variance

nu, d = symbols('nu delta', positive=True)
li, lj = symbols('lambda_i lambda_j', positive=True)
ybi, ybj, that = symbols('ybar_i ybar_j thetahat', real=True)

Z  = Normal('Z',  0, 1)   # theta|y  = thetahat + nu*Z
Ei = Normal('Ei', 0, 1)   # independent residuals of mu_i | theta, y
Ej = Normal('Ej', 0, 1)

theta = that + nu*Z
mu_i = li*ybi + (1 - li)*theta + sqrt((1 - li)*d**2)*Ei
mu_j = lj*ybj + (1 - lj)*theta + sqrt((1 - lj)*d**2)*Ej

assert simplify(covariance(mu_i, mu_j) - (1 - li)*(1 - lj)*nu**2) == 0
# consistency: the marginal variance matches the Main result of c2
assert simplify(variance(mu_i) - ((1 - li)**2*nu**2 + (1 - li)*d**2)) == 0
```

CLAIM c4: E{(μ − t)² | y} is uniquely minimized over constants t at t = E(μ|y).
