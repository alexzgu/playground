CLAIM c1: Mean = Sum/N to 4 decimals (1804.8950 / 2075 rounds to 0.8698)
```python
from fractions import Fraction
Sum = Fraction("1804.8950"); N = Fraction(2075)
mean = Sum / N
assert round(float(mean), 4) == 0.8698
```

CLAIM c2: Variance = CSS/(N-1) = 0.0249 and Std Dev = sqrt(Variance) = 0.1579
```python
from fractions import Fraction
import math
CSS = Fraction("51.7266"); N = 2075
var = CSS / (N - 1)
assert round(float(var), 4) == 0.0249
assert round(math.sqrt(float(var)), 4) == 0.1579
```

CLAIM c3: USS - Sum^2/N equals CSS (uncorrected minus correction = corrected sum of squares)
```python
from fractions import Fraction
USS = Fraction("1621.6765"); Sum = Fraction("1804.8950")
N = Fraction(2075); CSS = Fraction("51.7266")
assert abs(float(USS - Sum**2 / N) - float(CSS)) < 0.001
```

CLAIM c4: CV = 100*sd/mean = 18.1559 and Std Mean = sd/sqrt(N) = 0.0035
```python
from fractions import Fraction
import math
CSS = Fraction("51.7266"); Sum = Fraction("1804.8950"); N = 2075
sd = math.sqrt(float(CSS) / (N - 1))
mean = float(Sum) / N
assert round(100.0 * sd / mean, 4) == 18.1559
assert round(sd / math.sqrt(N), 4) == 0.0035
```
