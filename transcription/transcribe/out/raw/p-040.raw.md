CLAIM c1: The two displayed expressions for $L(\mu,\sigma^2\mid x)$ differ by the constant $(2\pi)^{(1-n)/2}/\sqrt{n}$, free of $\mu$ and $\sigma^2$.
```python
from sympy import *

mu, xbar = symbols('mu xbar', real=True)
sigma2, s2, n = symbols('sigma2 s2 n', positive=True)

LHS = (1/(2*pi*sigma2))**(n/2) * exp(-(1/(2*sigma2))*((n-1)*s2 + n*(xbar-mu)**2))
RHS = (1/sqrt(2*pi*sigma2/n)) * exp(-n/(2*sigma2)*(mu-xbar)**2) \
      * (1/sigma2)**((n-1)/2) * exp(-(n-1)*s2/(2*sigma2))

ratio = simplify(LHS/RHS)

# free of mu and sigma^2
assert simplify(diff(ratio, mu)) == 0
assert simplify(diff(ratio, sigma2)) == 0
# and equals the explicit constant
assert simplify(ratio - (2*pi)**((1-n)/2)/sqrt(n)) == 0
```

CLAIM c2: Completing the square: $k_o(\mu-\mu_o)^2+n(\mu-\bar x)^2=(k_o+n)(\mu-\mu_n)^2+\frac{k_o n}{k_o+n}(\bar x-\mu_o)^2$ for $\mu_n=\frac{k_o\mu_o+n\bar x}{k_o+n}$; the printed $k_o+n_o$ denominator does not give an identity.
```python
from sympy import *

mu, mu_o, xbar = symbols('mu mu_o xbar', real=True)
k_o, n, n_o = symbols('k_o n n_o', positive=True)

mu_n = (k_o*mu_o + n*xbar)/(k_o + n)

LHS = k_o*(mu - mu_o)**2 + n*(mu - xbar)**2
RHS = (k_o + n)*(mu - mu_n)**2 + (k_o*n/(k_o + n))*(xbar - mu_o)**2
assert simplify(LHS - RHS) == 0

# printed version: mu_n = k_o*mu_o/(k_o+n_o) + n*xbar/(k_o+n)
mu_n_printed = k_o*mu_o/(k_o + n_o) + n*xbar/(k_o + n)
RHS_printed = (k_o + n)*(mu - mu_n_printed)**2 + (k_o*n/(k_o + n))*(xbar - mu_o)**2
# not an identity: exhibit a counterexample (n_o != n)
sub = {k_o: Rational(1), n: Rational(2), n_o: Rational(5),
       mu: Rational(0), mu_o: Rational(1), xbar: Rational(0)}
assert simplify((LHS - RHS_printed).subs(sub)) != 0
```

CLAIM c3: Prior kernel $\times$ likelihood kernel equals the Nor-Inv-$\chi^2(\mu_n,\sigma_n^2/k_n,\nu_n,\nu_n\sigma_n^2)$ kernel exactly, with $k_n=k_o+n$, $\nu_n=\nu_o+n$, $\nu_n\sigma_n^2=\nu_o\sigma_o^2+(n-1)s^2+\frac{k_o n}{k_o+n}(\bar x-\mu_o)^2$.
```python
from sympy import *

mu, mu_o, xbar = symbols('mu mu_o xbar', real=True)
sigma2, s2, n, k_o, nu_o, sigma_o2 = symbols('sigma2 s2 n k_o nu_o sigma_o2', positive=True)

# posterior parameters as stated (mu_n with the corrected k_o + n denominator)
k_n = k_o + n
nu_n = nu_o + n
mu_n = (k_o*mu_o + n*xbar)/k_n
nu_n_sigma_n2 = nu_o*sigma_o2 + (n-1)*s2 + (k_o*n/(k_o+n))*(xbar - mu_o)**2

# Compare kernels on the log scale (SymPy cannot cancel the symbolic quadratics
# inside exp() directly): the power of (1/sigma2) and the numerator Q of the
# exp(-Q/(2 sigma2)) exponent must match exactly.

# prior kernel: sigma2^{-1/2} (1/sigma2)^{nu_o/2+1} exp(-Q_prior/(2 sigma2))
pow_prior = Rational(1,2) + nu_o/2 + 1
Q_prior   = k_o*(mu-mu_o)**2 + nu_o*sigma_o2

# likelihood kernel: sigma2^{-1/2} (1/sigma2)^{(n-1)/2} exp(-Q_lik/(2 sigma2))
pow_lik = Rational(1,2) + (n-1)/2
Q_lik   = n*(mu-xbar)**2 + (n-1)*s2

# posterior kernel: sigma2^{-1/2} (1/sigma2)^{nu_n/2+1} exp(-Q_post/(2 sigma2))
pow_post = Rational(1,2) + nu_n/2 + 1
Q_post   = k_n*(mu-mu_n)**2 + nu_n_sigma_n2

# powers of 1/sigma2 add up exactly
assert simplify(pow_prior + pow_lik - pow_post) == 0
# exponents agree identically (completing the square)
assert simplify(expand(Q_prior + Q_lik - Q_post)) == 0
```
