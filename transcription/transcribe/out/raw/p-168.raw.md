and

$$E(\mu\mid y) = \lambda y + (1-\lambda)\theta$$

Taking $\lambda \sim U(0,1)$, $\pi(\delta^2) = \frac{\sigma^2}{(\sigma^2+\delta^2)^2}$ and one may use $\pi(\delta^2) = \frac{a}{(a+\delta^2)^2}$, $\delta^2 > 0$ as the shrinkage prior for $\delta^2$. Generally $a = 1$ is used (e.g., Albert, Bayesian Statistics 3, 1988), Leonard and Novick (JASA, 1986) and Christensen and Morris (JASA, 1997). Shrinkage prior and Cauchy priors are generally recommended deep down in a hierarchical model. Priors like $\pi(\sigma^2) \propto \frac{1}{\sigma^2}$ create difficulties deep down in a hierarchical model (Gelman, BA 2006). Integration, blocking and multiplication rule are used when weakly identified parameters are suspect in a model. Rao-Blackwellization should be used to improve efficiency. Reparametrization is also recommended (Nandram and Chen, JSCS 1996).

CLAIM c1: Normal–normal conjugacy: the posterior of $\mu$ is $N(\lambda y+(1-\lambda)\theta,\ (1-\lambda)\delta^2)$ with $\lambda=\delta^2/(\delta^2+\sigma^2)$.
```python
from sympy import *

mu, y, theta = symbols('mu y theta', real=True)
sigma2, delta2 = symbols('sigma2 delta2', positive=True)

lam = delta2 / (delta2 + sigma2)
m = lam * y + (1 - lam) * theta      # claimed posterior mean
v = (1 - lam) * delta2               # claimed posterior variance

# log of likelihood x prior, and log of the claimed posterior kernel
log_post = -(y - mu)**2 / (2 * sigma2) - (mu - theta)**2 / (2 * delta2)
log_claim = -(mu - m)**2 / (2 * v)

# they must differ by a constant in mu  =>  d/dmu of the difference vanishes
assert simplify(diff(log_post - log_claim, mu)) == 0

# and the claimed variance equals the standard sigma2*delta2/(sigma2+delta2)
v_std = sigma2 * delta2 / (sigma2 + delta2)   # manifestly positive form
assert simplify(v - v_std) == 0
# equivalently v == lam * sigma2
assert simplify(v - lam * sigma2) == 0

# the claimed posterior kernel is a normalizable density in mu
# (integrate with the manifestly positive variance so no polar_lift artifacts appear)
kernel = exp(-(mu - m)**2 / (2 * v_std))
assert simplify(integrate(kernel, (mu, -oo, oo)) - sqrt(2 * pi * v_std)) == 0
```

CLAIM c2: $\lambda=\delta^2/(\delta^2+\sigma^2)$, $\lambda\sim U(0,1)$, induces $\pi(\delta^2)=\sigma^2/(\sigma^2+\delta^2)^2$ on $\delta^2>0$.
```python
from sympy import *

d, s = symbols('d s', positive=True)   # d = delta^2, s = sigma^2

lam = d / (d + s)

# lambda maps (0, oo) onto (0, 1) monotonically
assert limit(lam, d, 0, '+') == 0
assert limit(lam, d, oo) == 1
assert simplify(diff(lam, d) - s / (s + d)**2) == 0   # derivative > 0, so increasing

# change of variables: pi(d) = f_lambda(lam(d)) * |d lam / d d| = 1 * |d lam / d d|
pi_d = Abs(diff(lam, d))
assert simplify(pi_d - s / (s + d)**2) == 0

# and it is a proper density on (0, oo)
assert simplify(integrate(s / (s + d)**2, (d, 0, oo))) == 1
```

CLAIM c3: $\pi(\delta^2)=a/(a+\delta^2)^2$ integrates to 1 over $\delta^2>0$ for any $a>0$ (in particular $a=1$).
```python
from sympy import *

d = symbols('d', positive=True)
a = symbols('a', positive=True)

assert simplify(integrate(a / (a + d)**2, (d, 0, oo))) == 1

# check the a = 1 special case exactly as well
assert integrate(Rational(1) / (1 + d)**2, (d, 0, oo)) == 1
```
