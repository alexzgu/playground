CLAIM c1: Multinomial × Dirichlet($\mu\tau$) gives Dirichlet($n_i + \mu\tau$): the kernel $\prod_j \theta_j^{n_j+\tau\mu_j-1}$ integrates over the simplex to $D(n+\mu\tau)=\prod_j\Gamma(n_j+\tau\mu_j)/\Gamma(\sum_j(n_j+\tau\mu_j))$.
```python
from sympy import *
from mpmath import mp, quad, gamma as mgamma

t1, t2 = symbols('t1 t2', positive=True)

# k = 3, exact rational data and hyperparameters
n = [Integer(2), Integer(3), Integer(5)]
a = [Rational(1, 2), Integer(1), Rational(3, 2)]   # a_j = tau * mu_j

th = [t1, t2, 1 - t1 - t2]

# posterior kernel = multinomial kernel * Dirichlet prior kernel
mult_kernel = prod([th[j]**n[j] for j in range(3)])
dir_kernel = prod([th[j]**(a[j] - 1) for j in range(3)])
kernel = simplify(mult_kernel * dir_kernel)

post_kernel = prod([th[j]**(n[j] + a[j] - 1) for j in range(3)])
assert simplify(kernel - post_kernel) == 0

# integrate the kernel over the 2-simplex numerically
# (the symbolic simplex integral with half-integer exponents times out)
mp.dps = 25
e = [float(n[j] + a[j] - 1) for j in range(3)]
Z = quad(lambda y: quad(lambda x: x**e[0] * y**e[1] * (1 - x - y)**e[2],
                        [0, 1 - y]), [0, 1])

# Dirichlet normalizing constant D(n + a)
alpha = [float(n[j] + a[j]) for j in range(3)]
D = mgamma(alpha[0]) * mgamma(alpha[1]) * mgamma(alpha[2]) / mgamma(sum(alpha))

assert abs(Z - D) < mp.mpf('1e-15') * D
```
