CLAIM c1: Taking logs of $BF = f(x\mid M_2)/f(x\mid M_1)$ gives $\log BF = \log f(x\mid M_2) - \log f(x\mid M_1)$.
```python
from sympy import symbols, log, simplify, Rational

f1, f2 = symbols('f1 f2', positive=True)

BF = f2 / f1
lhs = log(BF)
rhs = log(f2) - log(f1)

# symbolic identity for positive marginal likelihoods
assert simplify(lhs - rhs) == 0

# decisive numeric spot-check at exact rationals
for a, b in [(Rational(1, 3), Rational(5, 7)), (Rational(9, 2), Rational(1, 11))]:
    assert simplify((lhs - rhs).subs({f1: a, f2: b})) == 0
```
