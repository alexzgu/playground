**9.1.4 Posterior Distributions**

Consider a simplified model

$$Y_{i1},\ldots,Y_{in_i}\mid \mu_i,\sigma_i^2 \overset{iid}{\sim} N\left(\mu_i,\sigma_i^2\right)$$
$$\mu_1,\ldots,\mu_l\mid \theta,\delta^2 \overset{iid}{\sim} N\left(\theta,\delta^2\right)$$
$$p(\theta) = 1$$

(i.e., assume $\sigma_i^2$ and $\delta^2$ are fixed and known.)

Let

$$\bar{y}_i = \frac{\sum_{j=1}^{n_i} y_{ij}}{n_i},$$
$$\lambda_i = \frac{\delta^2}{\delta^2 + \sigma_i^2/n_i}, \quad i = 1,\ldots,l$$

By Bayes theorem

$$\mu_i\mid \boldsymbol{y}_i,\theta \overset{ind}{\sim} N\left(\lambda_i\bar{y}_i + (1-\lambda_i)\,\theta,\ (1-\lambda_i)\,\delta^2\right)$$

> ⏳ VERIFY[c1]: with $\bar{y}_i\mid\mu_i \sim N(\mu_i,\sigma_i^2/n_i)$ and $\mu_i\mid\theta \sim N(\theta,\delta^2)$, the posterior of $\mu_i$ is $N\!\left(\lambda_i\bar{y}_i + (1-\lambda_i)\theta,\ (1-\lambda_i)\delta^2\right)$ for $\lambda_i = \delta^2/(\delta^2+\sigma_i^2/n_i)$

Remarks
1. Inference is required about $\mu_i$, but $\theta$ is unknown and random.
2. The unconditional posterior density of $\mu_i\mid\boldsymbol{y}_i$ is obtained by first finding the poste rior [sic] density $\theta\mid\boldsymbol{y}$.

**9.1.5 More sophistication**

We now integrate out $\theta$. Let

$$\nu^2 = \frac{\delta^2}{\sum_{i=1}^{l}\lambda_i} \quad \text{and} \quad \hat{\theta} = \frac{\sum_{i=1}^{l}\lambda_i\bar{y}_i}{\sum_{i=1}^{l}\lambda_i}$$

===CLAIMS===

CLAIM c1: The normal–normal update gives posterior $\mu_i\mid\bar y_i,\theta \sim N(\lambda_i\bar y_i+(1-\lambda_i)\theta,\ (1-\lambda_i)\delta^2)$ with $\lambda_i=\delta^2/(\delta^2+\sigma_i^2/n_i)$.
```python
from sympy import *

mu, theta, ybar = symbols('mu theta ybar', real=True)
d2, s2, n = symbols('delta2 sigma2 n', positive=True)

tau2 = s2 / n                      # sampling variance of ybar given mu
lam = d2 / (d2 + tau2)             # booklet's lambda_i

def lognorm(x, m, v):
    return -log(sqrt(2 * pi * v)) - (x - m)**2 / (2 * v)

# log of  f(ybar | mu) * f(mu | theta), as a function of mu
log_joint = lognorm(ybar, mu, tau2) + lognorm(mu, theta, d2)

# booklet's claimed posterior density for mu
post_mean = lam * ybar + (1 - lam) * theta
post_var = (1 - lam) * d2
log_post = lognorm(mu, post_mean, post_var)

# proportional in mu  <=>  d/dmu of the difference vanishes identically
assert simplify(diff(log_joint - log_post, mu)) == 0

# the leftover constant is mu-free and equals the log marginal density of
# ybar, i.e. N(theta, d2 + tau2) evaluated at ybar (joint = posterior * marginal)
diff_const = simplify(log_joint - log_post)
assert diff_const.has(mu) is False
assert simplify(diff_const - lognorm(ybar, theta, d2 + tau2)) == 0

# and the stated variance really is the standard precision-sum result
assert simplify(post_var - 1 / (1 / d2 + 1 / tau2)) == 0

# and the stated mean really is the precision-weighted average
assert simplify(post_mean - (1 / d2 + 1 / tau2)**-1 * (theta / d2 + ybar / tau2)) == 0
```
