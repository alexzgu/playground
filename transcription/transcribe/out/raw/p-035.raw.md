CLAIM c1: For the compound-symmetry covariance $\Gamma_{ii}=\sigma^2+\delta^2$, $\Gamma_{ij}=\delta^2$, the off-diagonal correlation is $\delta^2/(\delta^2+\sigma^2)$.
```python
from sympy import *

sigma, delta = symbols('sigma delta', positive=True)
n = 4
G = Matrix(n, n, lambda i, j: sigma**2 + delta**2 if i == j else delta**2)

# intraclass correlation from the stated covariance matrix
rho = G[0, 1] / sqrt(G[0, 0] * G[1, 1])
assert simplify(rho - delta**2 / (delta**2 + sigma**2)) == 0
```

CLAIM c2: With $\Gamma=\sigma^2 I+\delta^2 J$, the GLS estimator $(1'\Gamma^{-1}Y)/(1'\Gamma^{-1}1)$ equals $\bar{y}$.
```python
from sympy import *

sigma, delta = symbols('sigma delta', positive=True)

for n in (2, 3, 4, 5):
    G = Matrix(n, n, lambda i, j: sigma**2 + delta**2 if i == j else delta**2)
    y = Matrix(symbols(f'y1:{n+1}', real=True))
    one = ones(n, 1)
    Ginv = G.inv()
    theta_hat = (one.T * Ginv * y)[0] / (one.T * Ginv * one)[0]
    ybar = sum(y) / n
    assert simplify(theta_hat - ybar) == 0
```

CLAIM c3: The Cauchy scale-location density $\frac{1}{\sigma}f(\frac{x-\mu}{\sigma})$, $f(y)=\frac{1}{\pi(1+y^2)}$, equals the booklet's $f(x\mid\mu,\sigma)$ and integrates to 1.
```python
from sympy import *

x, mu = symbols('x mu', real=True)
sigma = symbols('sigma', positive=True)
y = symbols('y', real=True)

f = 1 / (pi * (1 + y**2))
transformed = (f / sigma).subs(y, (x - mu) / sigma)
booklet = 1 / (sigma * pi * (1 + ((x - mu) / sigma)**2))

assert simplify(transformed - booklet) == 0
assert simplify(integrate(booklet, (x, -oo, oo)) - 1) == 0
```

CLAIM c4: The normal scale-location density $\frac{1}{\sigma}f(\frac{x-\mu}{\sigma})$, $f(y)=\frac{1}{\sqrt{2\pi}}e^{-y^2/2}$, equals the booklet's $N(\mu,\sigma^2)$ density and integrates to 1.

```python
from sympy import *

x, mu = symbols('x mu', real=True)
sigma = symbols('sigma', positive=True)

```
