CLAIM c1: The Dirichlet–multinomial posterior kernel $\prod_i \theta_i^{n_i+\alpha_i-1}$ normalizes to $D(\alpha+n)$, i.e. $\theta\mid n \sim Dir(\alpha+n)$.
```python
from sympy import *

t1, t2 = symbols('t1 t2', positive=True)
al = [Integer(1), Integer(2), Integer(3)]
n  = [Integer(2), Integer(0), Integer(1)]
post = [a + b for a, b in zip(al, n)]

kern = t1**(post[0]-1) * t2**(post[1]-1) * (1 - t1 - t2)**(post[2]-1)
I = integrate(integrate(kern, (t2, 0, 1 - t1)), (t1, 0, 1))
D = prod([gamma(x) for x in post]) / gamma(sum(post))

assert simplify(I - D) == 0
```

CLAIM c2: For $\theta \sim Dir(\alpha)$, $cov(\theta_i,\theta_j) = -\alpha_i\alpha_j/(\alpha_0^2(\alpha_0+1))$.
```python
from sympy import *

t1, t2 = symbols('t1 t2', positive=True)
al = [Integer(1), Integer(2), Integer(3)]
a0 = sum(al)

def mom(e1, e2, e3):
    k = t1**(al[0]-1+e1) * t2**(al[1]-1+e2) * (1 - t1 - t2)**(al[2]-1+e3)
    return integrate(integrate(k, (t2, 0, 1 - t1)), (t1, 0, 1))

Z = mom(0, 0, 0)
cov = simplify(mom(1, 1, 0)/Z - (mom(1, 0, 0)/Z) * (mom(0, 1, 0)/Z))
claimed = -al[0]*al[1] / (a0**2 * (a0 + 1))

assert simplify(cov - claimed) == 0
```

CLAIM c5: The printed variance is a typo — the true $Beta(a,b)$ variance is $ab/((a+b)^2(a+b+1))$, which differs from the printed $a/((a+b)^2(a+b+1))$.
```python
from sympy import *

x = symbols('x', positive=True)
aa, bb = Rational(3, 2), Rational(5, 2)

B = beta(aa, bb).rewrite(gamma)   # force evaluation of the Beta function
pdf = x**(aa-1) * (1-x)**(bb-1) / B
Ex  = integrate(x * pdf, (x, 0, 1))
Ex2 = integrate(x**2 * pdf, (x, 0, 1))
V = simplify(Ex2 - Ex**2)

true_form    = aa*bb / ((aa+bb)**2 * (aa+bb+1))
printed_form = aa    / ((aa+bb)**2 * (aa+bb+1))

assert simplify(V - true_form) == 0       # correct formula holds
assert simplify(V - printed_form) != 0    # printed formula does not
```

CLAIM c6: Independent $Gamma(\alpha_j,1)$ variates normalized by their sum are $Dir(\alpha)$ (checked at $k=2$, where $Dir$ reduces to $Beta$).
```python
from sympy import *

s = symbols('s', positive=True)
r = symbols('r', positive=True)
g1, g2 = Integer(2), Integer(3)

# density of R = X1/(X1+X2) via the (R, S=X1+X2) change of variables, Jacobian s

target = r**(g1-1) * (1-r)**(g2-1) / beta(g1, g2)

assert simplify(fR - target) == 0
```
