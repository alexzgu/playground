===CLAIMS===

CLAIM c1: As $M \to \infty$, $psr = \sqrt{\big[(1-\tfrac{1}{M})\,wss + \tfrac{1}{M}\,bss\big]/wss} \to 1$ for fixed $bss \ge 0$ and $wss > 0$.
```python
from sympy import symbols, sqrt, limit, simplify, oo, Rational

M = symbols('M', positive=True)
bss, wss = symbols('bss wss', positive=True)

psr = sqrt(((1 - 1/M) * wss + bss / M) / wss)

# the limit as the chain length M -> oo is exactly 1
assert simplify(limit(psr, M, oo) - 1) == 0

# and psr == 1 exactly whenever bss == wss, for any M
assert simplify(psr.subs(bss, wss) - 1) == 0

# sanity: with bss > wss the factor exceeds 1 (finite M)
val = psr.subs({M: 10, wss: Rational(1), bss: Rational(4)})
assert simplify(val**2 - Rational(13, 10)) == 0
assert val > 1
```
