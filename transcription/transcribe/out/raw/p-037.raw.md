Aside:

$X \sim Gamma(\alpha,\beta)$

$f(x) = \frac{\beta^{\alpha}}{\Gamma(\alpha)}x^{\alpha-1}e^{-\beta x}, x \geq 0$

Let $y = \frac{1}{x}$, by transformation,

$f(y) = \frac{\beta^{\alpha}}{\Gamma(\alpha)}\left(\frac{1}{y}\right)^{\alpha+1}e^{-\frac{\beta}{y}}, y > 0$

> ⏳ VERIFY[c4]: if $X\sim Gamma(\alpha,\beta)$ and $Y=1/X$ then $f(y)=\frac{\beta^{\alpha}}{\Gamma(\alpha)}(1/y)^{\alpha+1}e^{-\beta/y}$.

$\sigma^2|y \sim \text{InvGam}\left(\frac{n-1}{2},\frac{(n-1)s^2}{2}\right)$

CLAIM c1: The integral $\int_{-\infty}^{\infty} e^{-\frac{n}{2\sigma^2}(\mu-\bar{x})^2}d\mu$ equals $\sqrt{2\pi\sigma^2/n}$, i.e. is proportional to $\sqrt{\sigma^2}$.
```python
from sympy import *

mu, xbar = symbols('mu xbar', real=True)
v = symbols('v', positive=True)   # v = sigma^2
n = symbols('n', positive=True)

I = integrate(exp(-n*(mu - xbar)**2/(2*v)), (mu, -oo, oo))
assert simplify(I - sqrt(2*pi*v/n)) == 0

# proportional to sqrt(sigma^2): ratio is free of v
assert simplify(diff(I/sqrt(v), v)) == 0
```

CLAIM c2: $(1/\sigma^2)^{n/2+1}\sqrt{\sigma^2} = (1/\sigma^2)^{(n+1)/2}$.
```python
from sympy import *

v = symbols('v', positive=True)   # v = sigma^2
n = symbols('n', positive=True)

lhs = (1/v)**(n/2 + 1) * sqrt(v)
rhs = (1/v)**((n + 1)/2)
assert simplify(lhs - rhs) == 0
```

CLAIM c3: $(1/\sigma^2)^{\frac{n-1}{2}+1}e^{-\frac{(n-1)s^2}{2\sigma^2}}$ is the $IG\!\left(\frac{n-1}{2},\frac{(n-1)s^2}{2}\right)$ kernel, and $\frac{n-1}{2}+1=\frac{n+1}{2}$.
```python
from sympy import *

v = symbols('v', positive=True)    # v = sigma^2
s2 = symbols('s2', positive=True)
n = symbols('n', positive=True)

assert simplify((n - 1)/2 + 1 - (n + 1)/2) == 0

a = (n - 1)/2
b = (n - 1)*s2/2

kernel = (1/v)**((n - 1)/2 + 1) * exp(-(n - 1)*s2/(2*v))
ig_pdf = b**a / gamma(a) * v**(-a - 1) * exp(-b/v)

# pdf / kernel must be constant in v (sigma^2)
ratio = simplify(ig_pdf / kernel)
assert simplify(diff(ratio, v)) == 0
# equal to the IG normalizing constant (force base splitting: all symbols positive)
assert simplify(powsimp(expand_power_base(ratio - b**a/gamma(a), force=True), force=True)) == 0

# and the IG density integrates to 1 (exact rational parameters)
vals = {n: Integer(7), s2: Rational(4, 3)}
assert simplify(integrate(ig_pdf.subs(vals), (v, 0, oo)) - 1) == 0
```

CLAIM c4: If $X\sim Gamma(\alpha,\beta)$ with $f(x)=\frac{\beta^\alpha}{\Gamma(\alpha)}x^{\alpha-1}e^{-\beta x}$ and $Y=1/X$, then $f(y)=\frac{\beta^\alpha}{\Gamma(\alpha)}(1/y)^{\alpha+1}e^{-\beta/y}$.
```python
from sympy import *

x, y = symbols('x y', positive=True)
al, be = symbols('alpha beta', positive=True)

f_x = be**al / gamma(al) * x**(al - 1) * exp(-be*x)

# change of variables y = 1/x  =>  x = 1/y, |dx/dy| = 1/y**2
f_y = simplify(f_x.subs(x, 1/y) * Abs(diff(1/y, y)))

stated = be**al / gamma(al) * (1/y)**(al + 1) * exp(-be/y)
assert simplify(f_y - stated) == 0

# sanity: the transformed density integrates to 1 for exact parameters
vals = {al: Rational(5, 2), be: Integer(3)}
assert simplify(integrate(stated.subs(vals), (y, 0, oo)) - 1) == 0
```
