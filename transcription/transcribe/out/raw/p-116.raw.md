## Example III. Nonresponse binary data

Do you belong to Massachusetts?

103 students, 80 responded, 60 said yes.

**Pattern mixture Model**

See Figure 5.

Each student responded independently.

$$p(y,r,z\mid\pi,p,q) = \frac{n!(\pi p)^{y}(\pi(1-p))^{r-y}((1-\pi)q)^{z}((1-\pi)(1-q))^{n-r-z}}{y!(r-y)!z!(n-r-z)!}$$

> ⏳ VERIFY[c3]: the four cell probabilities $\pi p,\ \pi(1-p),\ (1-\pi)q,\ (1-\pi)(1-q)$ sum to 1, so $p(y,r,z\mid\pi,p,q)$ is a multinomial mass function

Ignorable nonresponse model: $p = q$. (no problem)

Nonignorable nonresponse model: $p \neq q$. (difficult)

Take $q = ap$, so that $0 < p < \dfrac{1}{a}, a \geq 1$. Take $\pi, p \;\underset{\sim}{ind}\; Uniform(0,1)$

> ⏳ VERIFY[c4]: for $a>0$, the constraint $p<1/a$ is exactly the condition $q=ap<1$

Then

$$\pi(\pi,p,z\mid r,y,a) \propto \frac{(\pi p)^{y}(\pi(1-p))^{r-y}((1-\pi)ap)^{z}((1-\pi)(1-ap))^{n-r-z}}{z!(n-r-z)!}$$

> ⏳ VERIFY[c5]: substituting $q=ap$ into $p(y,r,z\mid\pi,p,q)$ gives the displayed kernel times $n!/(y!(r-y)!)$, a factor free of $\pi$, $p$ and $z$

CLAIM c4: For a > 0, the constraint p < 1/a is exactly q = ap < 1.
```python
from sympy import symbols, simplify

a, p = symbols('a p', positive=True)
# 1 - a*p and a*(1/a - p) are the same quantity, so they share a sign:
# p < 1/a  <=>  1/a - p > 0  <=>  1 - a*p > 0  <=>  q = a*p < 1.
assert simplify((1 - a * p) - a * (1 / a - p)) == 0

# spot-check the equivalence numerically on exact rationals
from sympy import Rational
for av in [Rational(1), Rational(2), Rational(5, 2)]:
    for pv in [Rational(1, 10), Rational(1, 3), Rational(9, 10)]:
        assert (pv < 1 / av) == (av * pv < 1)
```

CLAIM c5: Substituting q = ap into p(y,r,z|pi,p,q) reproduces the displayed kernel times n!/(y!(r-y)!).
```python
from sympy import symbols, simplify, factorial, Symbol

pi, p, a = symbols('pi p a', positive=True)

def full(n, r, y, z):
    q = a * p
    num = (factorial(n) * (pi * p) ** y * (pi * (1 - p)) ** (r - y)
           * ((1 - pi) * q) ** z * ((1 - pi) * (1 - q)) ** (n - r - z))
    den = factorial(y) * factorial(r - y) * factorial(z) * factorial(n - r - z)
    return num / den

def kernel(n, r, y, z):
    num = ((pi * p) ** y * (pi * (1 - p)) ** (r - y)
           * ((1 - pi) * a * p) ** z * ((1 - pi) * (1 - a * p)) ** (n - r - z))
    den = factorial(z) * factorial(n - r - z)
    return num / den

# the booklet's own data shape (n=103, r=80, y=60) plus small cases
cases = [(103, 80, 60, 5), (103, 80, 60, 17), (5, 3, 2, 1), (7, 4, 1, 2), (6, 2, 0, 3)]
for n, r, y, z in cases:
    ratio = simplify(full(n, r, y, z) / kernel(n, r, y, z))
    expected = factorial(n) / (factorial(y) * factorial(r - y))
    assert simplify(ratio - expected) == 0
    # the dropped factor is free of pi, p and z
    assert ratio.free_symbols == set()
```
