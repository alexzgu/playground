**9.1.3 Hierarchical Model**

The one-way random effects model can be rewritten in the hierarchy:
Scott and Smith (JASA 1969)

*[margin note, left of the two displays: "$Y_{ij} = \theta + v_i + e_{ij}$"]*

$$Y_{i1}, \ldots, Y_{im_i} \mid \mu_i, \sigma_i^2 \overset{iid}{\sim} N\left(\mu_i, \sigma_i^2\right) \tag{1}$$

$$\mu_1, \ldots, \mu_l \mid \theta, \delta^2 \overset{iid}{\sim} N\left(\theta, \delta^2\right) \tag{2}$$

Remark: If $\theta, \sigma_i^2$ and $\delta^2$ are fixed but unknown, this model is exactly the one-way random effects model.

&nbsp;&nbsp;&nbsp;&nbsp;This is a two stage hierarchical model. The first statement (1) describes the sampling process, and the second statement (2) describes a prior position.

CLAIM c1: For $Y_{ij}\mid\mu_i \sim N(\mu_i,\sigma_i^2)$ with $\mu_i \sim N(\theta,\delta^2)$, the marginal moments are $E(Y_{ij})=\theta$, $\mathrm{Var}(Y_{ij})=\sigma_i^2+\delta^2$, and $\mathrm{Cov}(Y_{ij},Y_{ij'})=\delta^2$ for $j\neq j'$.
```python
from sympy import *

y, m = symbols('y m', real=True)

# exact rational parameters: theta = 2, sigma^2 = 1/2, delta^2 = 1/3
th, s2, d2 = Rational(2), Rational(1, 2), Rational(1, 3)

f_y = lambda yv: exp(-(yv - m)**2 / (2*s2)) / sqrt(2*pi*s2)   # Y_ij | mu_i
f_m = exp(-(m - th)**2 / (2*d2)) / sqrt(2*pi*d2)              # mu_i

# conditional moments of Y_ij given mu_i
m1 = simplify(integrate(y * f_y(y), (y, -oo, oo)))            # E[Y | mu]  = mu
m2 = simplify(integrate(y**2 * f_y(y), (y, -oo, oo)))         # E[Y^2 | mu]

# marginal moments (integrate out mu)
EY   = simplify(integrate(m1 * f_m, (m, -oo, oo)))
EY2  = simplify(integrate(m2 * f_m, (m, -oo, oo)))
# for j != j', Y_ij and Y_ij' are conditionally independent given mu_i
EYY  = simplify(integrate(m1**2 * f_m, (m, -oo, oo)))

assert EY == th                       # E(Y_ij) = theta
assert simplify(EY2 - EY**2 - (s2 + d2)) == 0   # Var(Y_ij) = sigma^2 + delta^2
assert simplify(EYY - EY**2 - d2) == 0          # Cov(Y_ij, Y_ij') = delta^2
```
