> ⏳ VERIFY[c1]: the law of total expectation $E(U)=E_{V,W}\{E(U\mid V,W)\}$ holds.

and an estimation of $E(U)$ is $\widehat{E(U)}$, where

$$\widehat{E(U)} = M^{-1}\sum_{i=1}^{M} E(U\mid V^{(i)}, W^{(i)})$$

This process is called Rao-Blackwellization, and estimators like $\widehat{[U]}$ (?) and $\widehat{E(U)}$ are called Rao-Blackwellized estimators. It is true that $M^{-1}\sum_{i=1}^{M} E(U\mid V^{(i)}, W^{(i)})$ has smaller mean square error than $M^{-1}\sum_{i=1}^{M} U^{(i)}$, as an estimator of $E(U)$.

> ⏳ VERIFY[c2]: $\operatorname{Var}\!\big(E(U\mid V,W)\big) \le \operatorname{Var}(U)$, so the Rao-Blackwellized average has no larger mean square error than the raw average (both being unbiased for $E(U)$).

`Remark:` Use the Rao-Blackwellized estimators whenever it is possible. However, when the conditional does not have a "nice" form, use an "empirical" estimator (e.g., $\bar{U}$) or a nonparameteric [sic] density estimator (kernel).

CLAIM c1: The law of total expectation $E(U) = E_{V,W}\{E(U\mid V,W)\}$ holds for a discrete joint distribution.
```python
from sympy import Rational, simplify, symbols

# Joint pmf over (U, V, W) with U taking values in u_vals.
# States: (v, w) -> (prob of (v,w), conditional pmf of U over u_vals)
u_vals = [Rational(-1), Rational(2), Rational(5)]
states = {
    (0, 0): (Rational(1, 6), [Rational(1, 2), Rational(1, 4), Rational(1, 4)]),
    (0, 1): (Rational(1, 3), [Rational(1, 5), Rational(3, 5), Rational(1, 5)]),
    (1, 0): (Rational(1, 4), [Rational(1, 8), Rational(1, 8), Rational(3, 4)]),
    (1, 1): (Rational(1, 4), [Rational(1, 3), Rational(1, 3), Rational(1, 3)]),
}
assert sum(p for p, _ in states.values()) == 1
for _, c in states.values():
    assert sum(c) == 1

# E(U) directly from the marginal pmf of U
marg = [sum(p * c[k] for p, c in states.values()) for k in range(len(u_vals))]
assert sum(marg) == 1
EU = sum(m * u for m, u in zip(marg, u_vals))

# E_{V,W}{E(U|V,W)}
EU_iter = sum(p * sum(c[k] * u_vals[k] for k in range(len(u_vals))) for p, c in states.values())

assert simplify(EU - EU_iter) == 0
```

CLAIM c2: Rao-Blackwellization does not increase variance: Var(E(U|V,W)) <= Var(U), with equality iff U is (a.s.) a function of (V,W); hence M^{-1} sum E(U|V,W) has no larger MSE than M^{-1} sum U, both being unbiased for E(U).
```python
from sympy import Rational, simplify

u_vals = [Rational(-1), Rational(2), Rational(5)]
states = {
    (0, 0): (Rational(1, 6), [Rational(1, 2), Rational(1, 4), Rational(1, 4)]),
    (0, 1): (Rational(1, 3), [Rational(1, 5), Rational(3, 5), Rational(1, 5)]),
    (1, 0): (Rational(1, 4), [Rational(1, 8), Rational(1, 8), Rational(3, 4)]),
    (1, 1): (Rational(1, 4), [Rational(1, 3), Rational(1, 3), Rational(1, 3)]),
}

def cond_mean(c):
    return sum(c[k] * u_vals[k] for k in range(len(u_vals)))

def cond_second(c):
    return sum(c[k] * u_vals[k] ** 2 for k in range(len(u_vals)))

EU = sum(p * cond_mean(c) for p, c in states.values())
EU2 = sum(p * cond_second(c) for p, c in states.values())
Var_U = EU2 - EU ** 2

# g(V,W) = E(U|V,W)
E_g = sum(p * cond_mean(c) for p, c in states.values())
E_g2 = sum(p * cond_mean(c) ** 2 for p, c in states.values())
Var_g = E_g2 - E_g ** 2

# unbiasedness of the Rao-Blackwellized statistic
assert simplify(E_g - EU) == 0

# variance decomposition Var(U) = E Var(U|V,W) + Var E(U|V,W)
E_condvar = sum(p * (cond_second(c) - cond_mean(c) ** 2) for p, c in states.values())
assert simplify(Var_U - (E_condvar + Var_g)) == 0

# strict reduction here (U is not a function of (V,W))
assert Var_g < Var_U
# and the mean-square errors of the two Monte Carlo averages scale with these variances
M = 7
assert Var_g / M < Var_U / M
```

CLAIM c3: For mu_i | sigma^2, delta^2, y ~ N(m, v) with m = lam*ybar + (1-lam)*thetahat and v = (1-lam)^2*nu^2 + (1-lam)*delta^2, the marginal posterior variance equals E{v | y} + Var{m | y}.
```python
from sympy import Rational, simplify, symbols, sqrt, exp, pi, integrate, oo

ybar = Rational(3)

# Two posterior draws of (sigma^2, delta^2) -> (lam, nu^2, delta^2, thetahat), each w.p. given below.
# Exact rationals so the check is decisive.
comps = [
    # (prob, lam, nu2, delta2, thetahat)
    (Rational(2, 5), Rational(1, 3), Rational(1, 2), Rational(2), Rational(1)),
    (Rational(3, 5), Rational(3, 4), Rational(1, 5), Rational(1, 3), Rational(-2)),
]
assert sum(p for p, *_ in comps) == 1

def m_of(lam, thetahat):
    return lam * ybar + (1 - lam) * thetahat

def v_of(lam, nu2, delta2):
    return (1 - lam) ** 2 * nu2 + (1 - lam) * delta2

# Exact moments of the finite normal mixture (marginal over the posterior draws)
E_mu = sum(p * m_of(lam, th) for p, lam, nu2, d2, th in comps)
E_mu2 = sum(p * (v_of(lam, nu2, d2) + m_of(lam, th) ** 2) for p, lam, nu2, d2, th in comps)
Var_mu = E_mu2 - E_mu ** 2

# Right-hand side of the booklet's decomposition
E_condvar = sum(p * v_of(lam, nu2, d2) for p, lam, nu2, d2, th in comps)
E_m = sum(p * m_of(lam, th) for p, lam, nu2, d2, th in comps)
E_m2 = sum(p * m_of(lam, th) ** 2 for p, lam, nu2, d2, th in comps)
Var_condmean = E_m2 - E_m ** 2

assert simplify(Var_mu - (E_condvar + Var_condmean)) == 0

# Cross-check Var_mu against direct integration of the mixture density
x = symbols('x', real=True)
dens = sum(p * exp(-(x - m_of(lam, th)) ** 2 / (2 * v_of(lam, nu2, d2)))
           / sqrt(2 * pi * v_of(lam, nu2, d2)) for p, lam, nu2, d2, th in comps)
assert simplify(integrate(dens, (x, -oo, oo)) - 1) == 0
m1 = integrate(x * dens, (x, -oo, oo))
m2 = integrate(x ** 2 * dens, (x, -oo, oo))
assert simplify(m1 - E_mu) == 0
assert simplify((m2 - m1 ** 2) - Var_mu) == 0
```

CLAIM c4: The printed Monte Carlo estimator is the sample analogue of c3: its second term is exactly the sample variance of t_i^{(h)} about tbar_i, and the whole estimator converges to Var(mu_i | y) as the draws' empirical distribution matches the posterior.
```python
from sympy import Rational, simplify

ybar = Rational(3)

# A finite "posterior sample" of size M, drawn so the empirical distribution equals
# the two-point posterior of c3 exactly (2 of 5 draws in comp A, 3 of 5 in comp B).
draws = [
    # (lam^(h), nu2^(h), delta2^(h), thetahat^(h))
    (Rational(1, 3), Rational(1, 2), Rational(2), Rational(1)),
    (Rational(1, 3), Rational(1, 2), Rational(2), Rational(1)),
    (Rational(3, 4), Rational(1, 5), Rational(1, 3), Rational(-2)),
    (Rational(3, 4), Rational(1, 5), Rational(1, 3), Rational(-2)),
    (Rational(3, 4), Rational(1, 5), Rational(1, 3), Rational(-2)),
]
M = len(draws)

t = [lam * ybar + (1 - lam) * th for lam, nu2, d2, th in draws]
tbar = sum(t) / M

# Printed estimator
first = sum((1 - lam) ** 2 * nu2 + (1 - lam) * d2 for lam, nu2, d2, th in draws) / M
second = sum((ti - tbar) ** 2 for ti in t) / M
Var_hat = first + second

# Second term is the (1/M-normalized) sample variance of the t^(h)
sample_var = sum(ti ** 2 for ti in t) / M - tbar ** 2
assert simplify(second - sample_var) == 0

# Population target from c3 under the matching two-point posterior
comps = [
    (Rational(2, 5), Rational(1, 3), Rational(1, 2), Rational(2), Rational(1)),
    (Rational(3, 5), Rational(3, 4), Rational(1, 5), Rational(1, 3), Rational(-2)),
]
E_condvar = sum(p * ((1 - lam) ** 2 * nu2 + (1 - lam) * d2) for p, lam, nu2, d2, th in comps)
E_m = sum(p * (lam * ybar + (1 - lam) * th) for p, lam, nu2, d2, th in comps)
E_m2 = sum(p * (lam * ybar + (1 - lam) * th) ** 2 for p, lam, nu2, d2, th in comps)
target = E_condvar + (E_m2 - E_m ** 2)

# With the empirical distribution equal to the posterior, the estimator hits the target exactly
assert simplify(Var_hat - target) == 0

# And Ehat(mu_i|y) is the sample mean of t^(h)
assert simplify(tbar - E_m) == 0
```
