A similar problem occurs in the Dirichlet process mixture model,

$$Y_i \mid \mu_i, \sigma^2 \sim N(\mu_i, \sigma^2), \; i = 1, \cdots, l$$

$$\mu_i \mid G \sim G$$

$$G \sim DP\{\alpha, N(\theta, \delta^2)\}$$

with priors $\pi(\alpha)$ and $\pi(\theta, \; \delta^2)$. Escobar and West (JASA 1995) gave a general algorithm in which the infinite dimensional distribution $G$ is integrated out. But it gives poor mixing (long-range dependence). Ever since researches [sic] have been searching for better algorithms. The most recent algorithm keeps $G$ and express [sic] $G$ using the stick breaking representation $G(\mu) = \sum_{s=1}^{\infty} w_s \delta_{\mu_s}(\mu)$ (Sethuraman, 1994). The algorithm of Kalli, Griffin, Walker (SC 2011) is now recommended. This algorithm will be discussed later.

When two or more parameters are weakly identified, we need to integrate some of them out and draw the others simultaneously. When it is desirable to draw two or more parameters at the same time, we use the multiplication rule of probability. Suppose, after integration or no integration, we have three parameters $(\theta_1, \theta_2, \theta_3)$. Then we can write

$$\pi(\theta_1, \theta_2, \theta_3 \mid \boldsymbol{y}) = \pi_1(\theta_1 \mid \boldsymbol{y}) \pi_2(\theta_2 \mid \theta_1, \boldsymbol{y}) \pi(\theta_3 \mid \theta_1, \theta_2, \boldsymbol{y})$$

> ⏳ VERIFY[c2]: the chain-rule factorization together with the three formulas below is an identity — the product of the marginal, the conditional, and the un-normalized last factor reconstructs the joint exactly.

where

$$\pi(\theta_1 \mid \boldsymbol{y}) = \int \int \pi(\theta_1, \theta_2, \theta_3 \mid \boldsymbol{y}) d\theta_2 d\theta_3$$

$$\pi(\theta_2 \mid \theta_1, \boldsymbol{y}) = \frac{\int \pi(\theta_1, \theta_2, \theta_3 \mid \boldsymbol{y}) d\theta_3}{\int \int \pi(\theta_1, \theta_2, \theta_3 \mid \boldsymbol{y}) d\theta_2 d\theta_3}$$

$$\propto \int \pi(\theta_1, \theta_2, \theta_3 \mid \boldsymbol{y}) d\theta_3$$

> ⏳ VERIFY[c3]: the denominator $\int\int \pi(\theta_1,\theta_2,\theta_3\mid\boldsymbol{y}) d\theta_2 d\theta_3$ is free of $\theta_2$, which is what licenses the stated $\propto$.

$$\pi(\theta_3 \mid \theta_1, \theta_2, \boldsymbol{y}) \propto \pi(\theta_1, \theta_2, \theta_3 \mid \boldsymbol{y}), \; \textit{no integration.}$$

It is sensible to consider the importance of the parameters in this decomposition. Here, $\theta_3$ is the most important parameter, followed by $\theta_2$ and then $\theta_1$. Also $\theta_1$ or $\theta_2$ may be nuisance

CLAIM c1: $\mu = \theta + \delta Z$ with $Z \sim N(0,1)$ has density equal to the $N(\theta, \delta^2)$ density.
```python
from sympy import *

m, theta = symbols('m theta', real=True)
delta = symbols('delta', positive=True)
z = symbols('z', real=True)

phi = exp(-z**2/2)/sqrt(2*pi)

# density of mu = theta + delta*Z by the change-of-variables rule
g_inv = (m - theta)/delta                      # z as a function of m
jac = Abs(diff(g_inv, m))                      # = 1/delta
induced = simplify(phi.subs(z, g_inv) * jac)

normal_pdf = exp(-(m - theta)**2/(2*delta**2))/(delta*sqrt(2*pi))

assert simplify(induced - normal_pdf) == 0

# and it really is a density (integrates to 1)
assert simplify(integrate(normal_pdf, (m, -oo, oo))) == 1

# mean theta, variance delta**2
assert simplify(integrate(m*normal_pdf, (m, -oo, oo)) - theta) == 0
assert simplify(integrate((m - theta)**2 * normal_pdf, (m, -oo, oo)) - delta**2) == 0
```

CLAIM c2: the stated decomposition $\pi(\theta_1,\theta_2,\theta_3\mid y)=\pi_1(\theta_1\mid y)\,\pi_2(\theta_2\mid\theta_1,y)\,\pi(\theta_3\mid\theta_1,\theta_2,y)$ holds with the marginal/conditional defined by the printed integrals.
```python
from sympy import *

t1, t2, t3 = symbols('theta1 theta2 theta3', real=True)

# a concrete, non-factorizing joint posterior on [0,1]^3 (so the test is not vacuous)
K = Rational(5, 2)                     # integral of (t1+t2+t3+1) over the unit cube
joint = (t1 + t2 + t3 + 1) / K
assert simplify(integrate(joint, (t1, 0, 1), (t2, 0, 1), (t3, 0, 1)) - 1) == 0

# printed definitions, verbatim
pi1 = integrate(joint, (t2, 0, 1), (t3, 0, 1))                    # pi(t1 | y)
num = integrate(joint, (t3, 0, 1))                                # int pi dtheta3
den = integrate(joint, (t2, 0, 1), (t3, 0, 1))                    # int int pi dtheta2 dtheta3
pi2 = num / den                                                   # pi(t2 | t1, y)
pi3 = joint / num                                                 # pi(t3 | t1, t2, y), prop to joint

# each is a proper conditional density
assert simplify(integrate(pi1, (t1, 0, 1)) - 1) == 0
assert simplify(integrate(pi2, (t2, 0, 1)) - 1) == 0
assert simplify(integrate(pi3, (t3, 0, 1)) - 1) == 0

# the multiplication rule reconstructs the joint exactly
assert simplify(pi1 * pi2 * pi3 - joint) == 0

# pi3 is proportional to the joint in theta3 (ratio free of theta3)
assert simplify(diff(joint / pi3, t3)) == 0
```

CLAIM c3: the normalizing denominator of $\pi(\theta_2\mid\theta_1,y)$ does not depend on $\theta_2$, so $\pi(\theta_2\mid\theta_1,y) \propto \int \pi(\theta_1,\theta_2,\theta_3\mid y)\,d\theta_3$.
```python
from sympy import *

t1, t2, t3 = symbols('theta1 theta2 theta3', real=True)

K = Rational(5, 2)
joint = (t1 + t2 + t3 + 1) / K

den = integrate(joint, (t2, 0, 1), (t3, 0, 1))
assert simplify(diff(den, t2)) == 0        # denominator is free of theta2
assert t2 not in den.free_symbols

num = integrate(joint, (t3, 0, 1))
pi2 = num / den
# hence pi2 and num agree up to a theta2-free factor
assert simplify(diff(pi2 / num, t2)) == 0
```
