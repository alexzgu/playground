**9.5 Appendix**

<u>Binomial Model Example</u>

$$y_i|\nu_i \overset{iid}{\sim} \text{Bin}\left(n_i, \frac{e^{\nu_i}}{1+e^{\nu_i}}\right), i = 1,\cdots, l$$

$$\nu_i|\theta,\delta^2 \overset{iid}{\sim} N\left(\theta,\delta^2\right), \pi\left(\theta,\delta^2\right)\ \text{ hyperprior}$$

Then

$$\pi\left(\boldsymbol{\nu},\theta,\delta^2|\boldsymbol{y}\right) \propto \prod_{i=1}^{l}\left\{\frac{e^{y_i\nu_i}}{\left(1+e^{\nu_i}\right)^{n_i}}\frac{1}{\sqrt{2\pi\delta^2}}e^{-\frac{1}{2\delta^2}\left(\nu_i-\theta\right)^2}\right\}\pi\left(\theta,\delta^2\right)$$

*(The booklet writes $\nu$ and $y$ with under-tildes to indicate vectors.)*

Look at

$$\pi\left(\nu_i|\boldsymbol{y}\right) \propto \frac{e^{y_i\nu_i}}{\left(1+e^{\nu_i}\right)^{n_i}}, i = 1,\cdots, l\ (\text{i.e., }\ \text{likelihood })$$

<u>Aside</u>

Set $\delta\left(\nu_i\right) = y_i\nu_i - n_i\log\left(1+e^{\nu_i}\right)$. Then *[sic: the function is named $\delta(\nu_i)$ here but is written $\Delta$ in the derivatives that follow; also $\delta$ clashes with the hyperparameter $\delta^2$ above]*

$$\frac{\partial \Delta}{\partial \nu_i} = y_i - \frac{n_i e^{\nu_i}}{1+e^{\nu_i}}$$

> ⏳ VERIFY[c1]: $\partial\Delta/\partial\nu_i = y_i - n_i e^{\nu_i}/(1+e^{\nu_i})$ for $\Delta(\nu_i)=y_i\nu_i-n_i\log(1+e^{\nu_i})$.

$$\frac{\partial^2 \Delta}{\partial \nu_i^2} = -n_i\left\{\frac{e^{\nu_i}\left(1+e^{\nu_i}\right)-e^{2\nu_i}}{\left(1+e^{\nu_i}\right)^2}\right\} = -n_i\frac{e^{\nu_i}}{\left(1+e^{\nu_i}\right)^2}$$

> ⏳ VERIFY[c2]: the second derivative equals both stated expressions.

Setting $\frac{\partial\Delta}{\partial\nu_i} = 0$, get $\frac{e^{\hat{\nu}_i}}{1+e^{\hat{\nu}_i}} = \frac{y_i}{n_i}$. That is

$$\hat{\nu}_i = \log\left(\frac{y_i}{n_i-y_i}\right), i = 1,\cdots, l$$

> ⏳ VERIFY[c3]: the stationary point of $\Delta$ satisfies $e^{\hat\nu_i}/(1+e^{\hat\nu_i}) = y_i/n_i$, i.e. $\hat\nu_i = \log\!\big(y_i/(n_i-y_i)\big)$.

Adjusting by using empirical logistic transform is needed.

$$\tilde{\nu}_i = \log\left(\frac{y_i+\frac{1}{2}}{n_i-y_i+\frac{1}{2}}\right)$$

$\frac{1}{2}$ to 0s and 1s (like Jeffreys prior). This is needed when $y_i = 0$ or $n_i$( MLE does not exist).

However,

$$\text{Var}\left(\hat{\nu}_i\right) = \left(\frac{n_i e^{\hat{\nu}_i}}{\left(1+e^{\hat{\nu}_i}\right)^2}\right)^{-1} = \hat{\sigma}_i^2$$

> ⏳ VERIFY[c4]: $\text{Var}(\hat\nu_i)$ is the inverse observed information $\big(-\partial^2\Delta/\partial\nu_i^2\big)^{-1}$ evaluated at $\hat\nu_i$, and equals $n_i/\big(y_i(n_i-y_i)\big)$.

is always defined for the empirical logistic transform (Cox, 1970). We have approximately

$$\hat{\nu}_i \sim N\left(\nu_i, \hat{\sigma}_i^2\right)$$

CLAIM c1: First derivative of the binomial log-likelihood kernel $\Delta(\nu)=y\nu-n\log(1+e^{\nu})$ is $y - n e^{\nu}/(1+e^{\nu})$.
```python
from sympy import *

nu = symbols('nu', real=True)
y, n = symbols('y n', positive=True)

Delta = y*nu - n*log(1 + exp(nu))
lhs = diff(Delta, nu)
rhs = y - n*exp(nu)/(1 + exp(nu))

assert simplify(lhs - rhs) == 0
```

CLAIM c2: Second derivative equals $-n_i\{(e^{\nu}(1+e^{\nu})-e^{2\nu})/(1+e^{\nu})^2\}$ and simplifies to $-n_i e^{\nu}/(1+e^{\nu})^2$.
```python
from sympy import *

nu = symbols('nu', real=True)
y, n = symbols('y n', positive=True)

Delta = y*nu - n*log(1 + exp(nu))
d2 = diff(Delta, nu, 2)

mid = -n*((exp(nu)*(1 + exp(nu)) - exp(2*nu)) / (1 + exp(nu))**2)
final = -n*exp(nu)/(1 + exp(nu))**2

assert simplify(d2 - mid) == 0
assert simplify(mid - final) == 0
```

CLAIM c3: The MLE solves $e^{\hat\nu}/(1+e^{\hat\nu})=y/n$, giving $\hat\nu=\log(y/(n-y))$; and it is a maximum.
```python
from sympy import *

nu = symbols('nu', real=True)
y, n = symbols('y n', positive=True)

Delta = y*nu - n*log(1 + exp(nu))
score = diff(Delta, nu)

nuhat = log(y/(n - y))

# nuhat is a stationary point (valid for 0 < y < n)
assert simplify(score.subs(nu, nuhat).rewrite(Pow)) == 0

# and the stated stationarity condition holds there
assert simplify(exp(nuhat)/(1 + exp(nuhat)) - y/n) == 0

# second derivative is strictly negative => maximum
d2 = diff(Delta, nu, 2)
assert simplify(d2 + n*exp(nu)/(1 + exp(nu))**2) == 0
assert (d2.subs([(nu, Rational(1, 3)), (n, 7)]) < 0) == True

# exact numeric spot-check of the root
for (yv, nv) in [(Rational(1), 4), (Rational(3), 10), (Rational(9), 11)]:
    r = score.subs([(y, yv), (n, nv)]).subs(nu, log(yv/(nv - yv)))
    assert simplify(r) == 0
```

CLAIM c4: $\hat\sigma_i^2 = \big((1+e^{\hat\nu})^2/(n e^{\hat\nu})\big)$ is the inverse observed information and equals $n/(y(n-y))$.
```python
from sympy import *

nu = symbols('nu', real=True)
y, n = symbols('y n', positive=True)

Delta = y*nu - n*log(1 + exp(nu))
obs_info = -diff(Delta, nu, 2)          # n e^nu / (1+e^nu)^2

booklet = (n*exp(nu)/(1 + exp(nu))**2)**(-1)
assert simplify(booklet - 1/obs_info) == 0

nuhat = log(y/(n - y))
var_at_hat = simplify(booklet.subs(nu, nuhat))
assert simplify(var_at_hat - n/(y*(n - y))) == 0

# exact numeric spot-checks
for (yv, nv) in [(Rational(1), 4), (Rational(3), 10), (Rational(9), 11)]:
    got = simplify(booklet.subs(nu, log(yv/(nv - yv))).subs([(y, yv), (n, nv)]))
    assert got == Rational(nv, yv*(nv - yv))
```
