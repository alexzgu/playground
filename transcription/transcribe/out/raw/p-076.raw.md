**8.2 Hypothesis-Testing**

Bayesian think of hypotheses as models e.g.

- Non-Bayesian

$$X_1, X_2, \dots X_n \overset{iid.}{\sim} \mathcal{N}(\mu, \sigma^2)$$

  assume $\sigma^2$ is known; inference about $\mu$

$$H_0 : \mu = \mu_0 \quad \text{vs.} \quad H_1 : \mu = \mu_1.$$

- Bayesian

$$X_1, X_2, \dots X_n \mid \mu \overset{iid.}{\sim} \mathcal{N}(\mu, \sigma^2)$$

  assume $\sigma^2$ is known; inference about $\mu$

$$
\begin{aligned}
M_1 : \quad & X_1, X_2, \dots X_n \overset{iid.}{\sim} \mathcal{N}(\mu_0, \sigma^2)\\
M_2 : \quad & X_1, X_2, \dots X_n \overset{iid.}{\sim} \mathcal{N}(\mu_1, \sigma^2)
\end{aligned}
$$

`Remark (1):` $M_1$ and $M_2$ are symmetric.
`Remark (2):` Bayesians put a "lump" of probability on point hypotheses. *[the word "lump" is underlined by hand with a wavy stroke]*

Example:
&nbsp;&nbsp;&nbsp;&nbsp;Body mass index of young white males, Middlesex, MA.

$$n = 60,\; \bar{x} = 20.38,\; s = 5.22,\; \mu_0 = 22,\; \sigma^2 = s^2$$

*(correction: the printed symbol "$\mu$" in "$\mu = 60$" is struck through and replaced by handwritten "$n$")*

$$H_0 : \mu = \mu_0 \quad \text{vs.} \quad H_1 : \mu = \mu_1 \;(\mu_1 \ge 22).$$

Ratio of likelihoods is

$$e^{-\frac{n}{\sigma^2}\left(\bar{x} - \frac{\mu_0 + \mu_1}{2}\right)(\mu_1 - \mu_0)},$$

*(correction: the printed numerator of the exponent's fraction reads "$\mu$"; a handwritten "$n$" is written above it)*

> ⏳ VERIFY[c2]: for $X_1,\dots,X_n\overset{iid}{\sim}\mathcal{N}(\mu,\sigma^2)$ with $\sigma^2$ known, $L(\mu_0)/L(\mu_1)=\exp\!\big[-\tfrac{n}{\sigma^2}\big(\bar x-\tfrac{\mu_0+\mu_1}{2}\big)(\mu_1-\mu_0)\big]$

Neyman-Pearson Lemma:
Reject $H_0$ if the ratio of likelihood is small $=$ Reject $H_0$ if $\bar{x}$ is large. *[the clause "the ratio of likelihood is small" is hand-underlined, with a stroke through the word "of"]*

$$

\end{aligned}
$$

CLAIM c1: For a d-variate normal density, the level set {f ≥ k} is the ellipsoid {(θ−θ̂)′Σ⁻¹(θ−θ̂) ≤ c} with c = −2·log(k·(2π)^{d/2}·|Σ|^{1/2}).
```python
from sympy import *

d = 2
Sig = Matrix([[2, 1], [1, 3]])
t1, t2 = symbols('t1 t2', real=True)
th = Matrix([t1, t2])
mu = Matrix([0, 0])

Q = ((th - mu).T * Sig.inv() * (th - mu))[0]
detS = Sig.det()

f = (2*pi)**Rational(-d, 2) * detS**Rational(-1, 2) * exp(-Q/2)

k = symbols('k', positive=True)
A = log((2*pi)**Rational(d, 2) * sqrt(detS))
c = -2 * (log(k) + A)

# log f = -Q/2 - A  exactly, so f >= k  <=>  -Q/2 - A >= log k  <=>  Q <= c
assert simplify(expand_log(log(f), force=True) - (-Q/2 - A)) == 0

# boundary: f = k  <=>  Q = c (solve for the quadratic form via a fresh symbol q)
q = symbols('q', real=True)
sol = solve(Eq(-q/2 - A, log(k)), q)
assert len(sol) == 1 and simplify(sol[0] - c) == 0

# and c indeed equals -2*log(k*(2*pi)**(d/2)*|Sigma|**(1/2))
assert simplify(c - (-2)*log(k * (2*pi)**Rational(d, 2) * sqrt(detS))) == 0

# Q is a positive-definite quadratic form => {Q <= c} is an ellipsoid (bounded, convex)
assert all(ev > 0 for ev in Sig.inv().eigenvals())
```

CLAIM c2: Ratio of normal likelihoods L(mu0)/L(mu1) equals exp[-(n/sigma^2)(xbar-(mu0+mu1)/2)(mu1-mu0)].
```python
from sympy import *

n = 5
sigma = symbols('sigma', positive=True)
mu0, mu1 = symbols('mu0 mu1', real=True)
xs = symbols('x1:6', real=True)
xbar = Add(*xs) / n

def loglik(mu):
    return Add(*[-(x - mu)**2 / (2 * sigma**2) for x in xs])

ratio = exp(loglik(mu0) - loglik(mu1))
booklet = exp(-n / sigma**2 * (xbar - (mu0 + mu1) / 2) * (mu1 - mu0))

assert simplify(log(ratio) - log(booklet)) == 0
```

CLAIM c3: Under H0, Pr(Xbar > xbar) = 1 - Phi((xbar-mu0)/(sigma/sqrt(n))).
```python
from sympy import *

t = symbols('t', real=True)
xbar, mu0 = symbols('xbar mu0', real=True)
sigma = Rational(3, 2)
n = 4
se = sigma / sqrt(n)

pdf = exp(-(t - mu0)**2 / (2 * se**2)) / (se * sqrt(2 * pi))
lhs = integrate(pdf, (t, xbar, oo))

Phi = lambda z: (1 + erf(z / sqrt(2))) / 2
rhs = 1 - Phi((xbar - mu0) / se)

assert simplify(lhs - rhs) == 0
```
