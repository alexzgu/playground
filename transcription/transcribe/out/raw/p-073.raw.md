CLAIM c1: Equation (8.4) — $F(b\mid y)-F(a\mid y)=\int_a^b f(\theta\mid y)\,d\theta$ holds as an identity for a posterior density with CDF $F$, so the displayed chain equating both to $1-\alpha$ is consistent.
```python
from sympy import *

th, a, b, x = symbols('theta a b x', real=True)

# A concrete unimodal posterior density with interior mode: Beta(2,2) on (0,1).
f = 6*th*(1 - th)

# Total mass is 1, so it is a genuine density.
assert simplify(integrate(f, (th, 0, 1)) - 1) == 0

# CDF of the posterior.
F = integrate(f.subs(th, x), (x, 0, th))

# (8.4), left equality: F(b) - F(a) == \int_a^b f(theta) d theta, identically in a, b.
lhs = F.subs(th, b) - F.subs(th, a)
rhs = integrate(f, (th, a, b))
assert simplify(lhs - rhs) == 0

# (8.4), right equality: setting the common value to 1 - alpha determines the credibility.
# Take an explicit HPD pair for this symmetric density: a = 1/4, b = 3/4.
a0, b0 = Rational(1, 4), Rational(3, 4)
coverage = integrate(f, (th, a0, b0))
assert coverage == Rational(11, 16)              # = 1 - alpha, exactly
assert simplify((F.subs(th, b0) - F.subs(th, a0)) - coverage) == 0
```

CLAIM c2: The displayed "Solve for a" equation — with $b=F^{-1}\{F(a)+(1-\alpha)\}$ the coverage constraint (8.4) holds identically, so (8.3)+(8.4) reduce to $f(a\mid y)=f[F^{-1}\{F(a)+(1-\alpha)\}]$ in the single unknown $a$.
```python
from sympy import *

th = symbols('theta', real=True)
# Logistic posterior: unimodal, mode 0 in the interior, closed-form F and F^{-1}.
F_expr = 1/(1 + exp(-th))
f = simplify(diff(F_expr, th))                    # exp(-th)/(1+exp(-th))**2

F    = lambda t: 1/(1 + exp(-t))
Finv = lambda u: log(u/(1 - u))
fden = lambda t: f.subs(th, t)

# Fix a credibility 1 - alpha = 1/2, and let a be any point left of the mode,
# so that F(a) < 1/2 and the argument of F^{-1} stays inside (0,1).
one_minus_alpha = Rational(1, 2)
a = Symbol('a', negative=True)

b = Finv(F(a) + one_minus_alpha)                  # the booklet's inversion of (8.4)

# (8.4) then holds identically in a: the interval (a, b) always carries mass 1 - alpha.
assert simplify(F(b) - F(a) - one_minus_alpha) == 0
# same statement via the integral, checked to high precision at exact points a < 0
for aval in [Rational(-2), Rational(-1), Rational(-1, 2)]:
    bval = b.subs(a, aval)
    cov = integrate(f, (th, aval, bval))
    assert abs(float(N(cov - one_minus_alpha, 30))) < 1e-20

# Hence the HPD system (8.3) & (8.4) is exactly the displayed single equation in a:
#     f(a) = f[F^{-1}{F(a) + (1 - alpha)}]
residual = simplify(fden(a) - fden(Finv(F(a) + one_minus_alpha)))

# Its root reproduces the true equal-ordinate HPD endpoints, symmetric about the mode 0.
a_star = Rational(-1, 1)*log(3)                   # F(a) = 1/4, so b = log(3) = -a
assert simplify(residual.subs(a, a_star)) == 0                      # (8.3) satisfied
# (8.4) at the root: F(log 3) - F(-log 3) = 3/4 - 1/4 = 1/2, exactly
assert simplify(F(-a_star) - F(a_star) - one_minus_alpha) == 0
assert abs(float(N(integrate(f, (th, a_star, -a_star)) - one_minus_alpha, 30))) < 1e-20
```

CLAIM c3: The Remark — for a density symmetric about its mode $m$, $(m-a,m+a)$ satisfies the equal-ordinate condition (8.3), and it is the $100(1-\alpha)\%$ HPD interval exactly when $\int_m^{m+a} f(\theta\mid y)\,d\theta=(1-\alpha)/2$.
```python
from sympy import *

th = symbols('theta', real=True)
a  = Symbol('a', positive=True)

# Beta(2,2): symmetric about its mode m = 1/2.
f = 6*th*(1 - th)
m = Rational(1, 2)
assert simplify(integrate(f, (th, 0, 1)) - 1) == 0
assert simplify(diff(f, th).subs(th, m)) == 0        # m is the mode
assert simplify(f.subs(th, m - a) - f.subs(th, m + a)) == 0   # symmetry about m

# (8.3) holds automatically at the endpoints m - a and m + a: equal ordinates.
# Coverage of (m - a, m + a) is exactly twice the half-interval mass.
half  = integrate(f, (th, m, m + a))
whole = integrate(f, (th, m - a, m + a))
assert simplify(whole - 2*half) == 0

# Therefore coverage == 1 - alpha  <=>  half == (1 - alpha)/2, which is the Remark.
alpha = Symbol('alpha', positive=True)
assert simplify((whole - (1 - alpha)) - 2*(half - (1 - alpha)/2)) == 0

# Concrete instance: a = 1/4 gives half = 11/32, whole = 11/16 = 1 - alpha.
assert half.subs(a, Rational(1, 4)) == Rational(11, 32)
assert whole.subs(a, Rational(1, 4)) == Rational(11, 16)
assert simplify(half.subs(a, Rational(1, 4)) - (Rational(11, 16))/2) == 0
```
