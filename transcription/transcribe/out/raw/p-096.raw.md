Then we consider Bayesian benchmarking. Consider the Scott-Smith model

$$y_{ij}\mid\mu_i,\sigma^2 \overset{ind}{\sim} N\left(\mu_i,\sigma^2\right), \qquad j = 1,\cdots,N_i, i = 1,\cdots,l$$

$$\mu_i\mid\theta,\sigma^2,\rho \overset{iid}{\sim} N\left(\theta,\frac{\rho}{1-\rho}\sigma^2\right)$$

$$\pi\left(\theta,\sigma^2,\rho\right) \propto \frac{1}{\sigma^2}$$

One benchmarking constraint is

$$\frac{\sum_{i=1}^{l} N_i\mu_i}{\sum_{i=1}^{l} N_i} = \sum_{i=1}^{l} w_i\mu_i = \psi,\; w_i = \frac{N_i}{\sum_{i=1}^{l} N_i} \qquad i = 1,\cdots,l$$

> ⏳ VERIFY[c1]: with $w_i = N_i/\sum_{i=1}^{l}N_i$, the weights sum to one and $\sum_i w_i\mu_i = \left(\sum_i N_i\mu_i\right)/\sum_i N_i$.

One can now put a prior on $\psi$. For example, $\psi \sim N\left(\theta_0,\delta_0\right)$, where $\theta_0$ and $\delta_0$ may be obtained from another survey or census. CLAIM c1: With $w_i = N_i/\sum_{i=1}^{l} N_i$, the weights sum to 1 and $\sum_i w_i\mu_i$ equals the $N_i$-weighted mean $\left(\sum_i N_i\mu_i\right)/\sum_i N_i$.
```python
from sympy import symbols, simplify, Rational, Integer

# symbolic check for general l via a concrete-but-arbitrary small l, plus exact rationals
for l in (1, 2, 3, 4, 5):
    N = symbols(f'N1:{l+1}', positive=True)
    mu = symbols(f'mu1:{l+1}', real=True)
    total = sum(N)
    w = [Ni / total for Ni in N]

    # weights sum to one
    assert simplify(sum(w) - 1) == 0

    # weighted mean identity
    lhs = sum(wi * mi for wi, mi in zip(w, mu))
    rhs = sum(Ni * mi for Ni, mi in zip(N, mu)) / total
    assert simplify(lhs - rhs) == 0

# exact numeric spot-check with Rationals
Nv = [Integer(3), Integer(7), Integer(10)]
muv = [Rational(1, 2), Rational(-4, 3), Rational(5, 6)]
tot = sum(Nv)
lhs = sum(Rational(n, tot) * m for n, m in zip(Nv, muv))
rhs = sum(n * m for n, m in zip(Nv, muv)) / tot
assert lhs == rhs
assert sum(Rational(n, tot) for n in Nv) == 1
```

CLAIM c2: With $w_i = N_i/\sum_i N_i$, $\bar{Y}_i = \frac{1}{N_i}\sum_{j=1}^{N_i} y_{ij}$, and $\bar{Y}$ the grand mean $\frac{\sum_i\sum_j y_{ij}}{\sum_i N_i}$, the benchmarking constraint $\sum_{i=1}^{l} w_i\bar{Y}_i = \bar{Y}$ holds identically.
```python
from sympy import symbols, simplify, Rational

# unequal group sizes N_i, symbolic observations y_ij
Nsizes = [2, 3, 1, 4]
l = len(Nsizes)

y = [symbols(f'y{i}_1:{Nsizes[i]+1}', real=True) for i in range(l)]

grand_total = sum(sum(yi) for yi in y)
n_total = sum(Nsizes)

Ybar = grand_total / n_total                       # grand mean
Ybar_i = [sum(y[i]) / Nsizes[i] for i in range(l)] # group means
w = [Rational(Nsizes[i], n_total) for i in range(l)]

lhs = sum(w[i] * Ybar_i[i] for i in range(l))
assert simplify(lhs - Ybar) == 0

# the identity fails for weights not proportional to N_i (constraint is not vacuous)
w_bad = [Rational(1, l)] * l
lhs_bad = sum(w_bad[i] * Ybar_i[i] for i in range(l))
```
