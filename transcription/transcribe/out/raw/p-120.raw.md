Consider a function $h(.)$ of $y_r$. Then, we need $E(h(y_r)|y_{(r)})$. Now,

$$
\begin{aligned}
E(h(y_r)\mid y_{(r)}) &= \int \ldots \int E(h(y_r)\mid y_{(r)},\theta)\,\pi(\theta\mid y_{(r)})\,d\theta \\[2ex]
&= \int \ldots \int E(h(y_r)\mid y_{(r)},\theta)\,\frac{\pi(\theta\mid y_{(r)})}{\pi(\theta\mid y)}\,\pi(\theta\mid y)\,d\theta \\[3ex]
&= \frac{\displaystyle \int \ldots \int E(h(y_r)\mid y_{(r)},\theta)\,\frac{\pi(\theta\mid y_{(r)})}{\pi(\theta\mid y)}\,\pi(\theta\mid y)\,d\theta}{\displaystyle \int \ldots \int \frac{\pi(\theta\mid y_{(r)})}{\pi(\theta\mid y)}\,\pi(\theta\mid y)\,d\theta}
\end{aligned}
$$

> ⏳ VERIFY[c1]: dividing by $\int \frac{\pi(\theta\mid y_{(r)})}{\pi(\theta\mid y)}\pi(\theta\mid y)\,d\theta$ is legitimate because that integral equals 1, and the resulting ratio equals $\int E(h(y_r)\mid y_{(r)},\theta)\pi(\theta\mid y_{(r)})\,d\theta$.

Then

$$
E(h(y_r)\mid y_{(r)}) \approx \frac{\displaystyle M^{-1}\sum_{k=1}^{M} E(h(y_r)\mid y_{(r)},\theta^{(k)})\,\frac{\pi(\theta^{(k)}\mid y_{(r)})}{\pi(\theta^{(k)}\mid y)}}{\displaystyle M^{-1}\sum_{k=1}^{M} \frac{\pi(\theta^{(k)}\mid y_{(r)})}{\pi(\theta^{(k)}\mid y)}}
$$

$$
= \sum_{k=1}^{M} w_r^{(k)}\, E\left\{ h(y_r)\mid y_{(r)},\theta^{(k)} \right\}
$$

where

$$
w_r^{(k)} = \frac{\dfrac{\pi(\theta^{(k)}\mid y_{(r)})}{\pi(\theta^{(k)}\mid y)}}{\displaystyle \sum_{k=1}^{M} \frac{\pi(\theta^{(k)}\mid y_{(r)})}{\pi(\theta^{(k)}\mid y)}},\, r = 1,...,n.k = 1,...,M.
$$

*(sic: the booklet prints "$r = 1,\ldots,n.k = 1,\ldots,M.$" — the separator between the two index ranges is a period rather than a comma or semicolon. Also, the summation index $k$ in the denominator of $w_r^{(k)}$ clashes with the free index $k$ in the numerator.)*

> ⏳ VERIFY[c2]: the self-normalized weights $w_r^{(k)}$ sum to 1 over $k$, and the ratio-of-averages estimator equals $\sum_{k=1}^{M} w_r^{(k)} E\{h(y_r)\mid y_{(r)},\theta^{(k)}\}$.

With $\theta^{(1)},...,\theta^{(M)}$ a random sample from $\pi(\theta\mid y)$, the posterior density with all the data.

Finally, by Bayes' theorem

$$
\frac{\pi(\theta\mid y_{(r)})}{\pi(\theta\mid y)} = \frac{\dfrac{f(y_{(r)}\mid \theta)}{f(y\mid \theta)}}{\dfrac{f(y_{(r)})}{f(y)}}
$$

> ⏳ VERIFY[c3]: with $\pi(\theta\mid y) = f(y\mid\theta)\pi(\theta)/f(y)$ and $\pi(\theta\mid y_{(r)}) = f(y_{(r)}\mid\theta)\pi(\theta)/f(y_{(r)})$, the displayed ratio identity holds.

CLAIM c1: The importance-sampling denominator $\int \frac{\pi(\theta\mid y_{(r)})}{\pi(\theta\mid y)}\pi(\theta\mid y)\,d\theta$ equals 1, so the ratio form on line 3 equals the plain expectation on line 1.
```python
from sympy import *

# Take theta on a 3-point support (an integral over theta becomes a finite sum).
# pi_r[j] = pi(theta_j | y_(r)),  pi_y[j] = pi(theta_j | y):  both are probability
# vectors (nonneg, summing to 1).  g[j] = E(h(y_r) | y_(r), theta_j) is arbitrary.
a1, a2, b1, b2 = symbols('a1 a2 b1 b2', positive=True)
g1, g2, g3 = symbols('g1 g2 g3', real=True)

pi_r = [a1, a2, 1 - a1 - a2]      # sums to 1 by construction
pi_y = [b1, b2, 1 - b1 - b2]      # sums to 1 by construction
g    = [g1, g2, g3]

# denominator of the booklet's line 3: int (pi_r/pi_y) * pi_y dtheta
denom = simplify(sum((pi_r[j] / pi_y[j]) * pi_y[j] for j in range(3)))
assert simplify(denom - 1) == 0          # the denominator really is 1

# numerator of line 3 vs. the plain expectation of line 1
num  = sum(g[j] * (pi_r[j] / pi_y[j]) * pi_y[j] for j in range(3))
line1 = sum(g[j] * pi_r[j] for j in range(3))
assert simplify(num / denom - line1) == 0

# and line 2 (the reweighted form) equals line 1 as well
line2 = sum(g[j] * (pi_r[j] / pi_y[j]) * pi_y[j] for j in range(3))
assert simplify(line2 - line1) == 0
```

CLAIM c2: The self-normalized weights $w_r^{(k)}$ sum to 1, and $\dfrac{M^{-1}\sum_k g_k u_k}{M^{-1}\sum_k u_k} = \sum_k w_r^{(k)} g_k$ with $u_k = \pi(\theta^{(k)}\mid y_{(r)})/\pi(\theta^{(k)}\mid y)$.
```python
from sympy import *

M = 4
u = symbols('u1:5', positive=True)        # u_k = pi(theta^(k)|y_(r)) / pi(theta^(k)|y)
g = symbols('g1:5', real=True)            # g_k = E{h(y_r) | y_(r), theta^(k)}

w = [u[k] / Add(*u) for k in range(M)]    # the booklet's w_r^(k)

# weights sum to one
assert simplify(Add(*w) - 1) == 0

# ratio-of-averages estimator == weighted average
ratio_of_averages = (Rational(1, M) * Add(*[g[k] * u[k] for k in range(M)])) / \
                    (Rational(1, M) * Add(*[u[k] for k in range(M)]))
weighted_average  = Add(*[w[k] * g[k] for k in range(M)])
assert simplify(ratio_of_averages - weighted_average) == 0
```

CLAIM c3: Bayes' theorem gives $\dfrac{\pi(\theta\mid y_{(r)})}{\pi(\theta\mid y)} = \dfrac{f(y_{(r)}\mid\theta)/f(y\mid\theta)}{f(y_{(r)})/f(y)}$.
```python
from sympy import *

# generic positive quantities; prior pi(theta) cancels
prior   = symbols('pi_theta',  positive=True)   # pi(theta)
f_r_th  = symbols('f_r_theta', positive=True)   # f(y_(r) | theta)
f_y_th  = symbols('f_y_theta', positive=True)   # f(y     | theta)
f_r     = symbols('f_r',       positive=True)   # f(y_(r))
f_y     = symbols('f_y',       positive=True)   # f(y)

post_r = f_r_th * prior / f_r        # pi(theta | y_(r))
post_y = f_y_th * prior / f_y        # pi(theta | y)

lhs = post_r / post_y
rhs = (f_r_th / f_y_th) / (f_r / f_y)
assert simplify(lhs - rhs) == 0

# sanity: the booklet's stacked fraction is the same object read top-down
assert simplify(rhs - (f_r_th * f_y) / (f_y_th * f_r)) == 0
```
