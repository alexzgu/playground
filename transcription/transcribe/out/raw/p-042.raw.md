CLAIM c1: For $X_i\mid\mu\overset{iid}{\sim}N(\mu,\sigma^2)$ ($i=1..n$, $\sigma^2$ known) and prior $\mu\sim N(\theta,\delta^2)$, the posterior variance of $\mu$ equals $\left(\frac{1}{\delta^2}+\frac{n}{\sigma^2}\right)^{-1}$.
```python
from sympy import *

mu = symbols('mu', real=True)

# exact rational parameters
sigma2 = Rational(2)
delta2 = Rational(3)
theta  = Rational(1)
xs     = [Rational(1), Rational(2), Rational(4)]
n      = len(xs)

# unnormalized posterior: likelihood * prior
like  = prod([exp(-(x - mu)**2 / (2*sigma2)) for x in xs])
prior = exp(-(mu - theta)**2 / (2*delta2))
post  = like * prior

Z    = integrate(post, (mu, -oo, oo))
m1   = integrate(mu    * post, (mu, -oo, oo)) / Z
m2   = integrate(mu**2 * post, (mu, -oo, oo)) / Z
var  = simplify(m2 - m1**2)

claimed = 1 / (1/delta2 + Rational(n)/sigma2)
assert simplify(var - claimed) == 0
assert var == Rational(6, 11)   # 1/(1/3 + 3/2)
```

CLAIM c2: $ESS = \dfrac{Var_N}{Var_I}\,n = \dfrac{\sigma^2/n}{\left(\frac{1}{\delta^2}+\frac{n}{\sigma^2}\right)^{-1}}\,n$ simplifies to $\dfrac{\sigma^2}{\delta^2}+n$, and consequently $ESS\ge n$.
```python
from sympy import *

sigma2, delta2, n = symbols('sigma2 delta2 n', positive=True)

Var_I = 1 / (1/delta2 + n/sigma2)
Var_N = sigma2 / n

ESS = (Var_N / Var_I) * n
assert simplify(ESS - (sigma2/delta2 + n)) == 0

# ESS - n = sigma^2/delta^2 > 0, so ESS >= n
assert simplify(ESS - n) == simplify(sigma2/delta2)
assert ask(Q.positive(simplify(ESS - n))) is True
```

CLAIM c3: The prior is worth $ESS-n=\sigma^2/\delta^2$ observations, which tends to $0$ as $\delta^2\to\infty$.
```python
from sympy import *

sigma2, delta2, n = symbols('sigma2 delta2 n', positive=True)

ESS = sigma2/delta2 + n
worth = simplify(ESS - n)
assert simplify(worth - sigma2/delta2) == 0
assert limit(worth, delta2, oo) == 0
```
