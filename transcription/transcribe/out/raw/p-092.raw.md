===CLAIMS===

CLAIM c1: As $\delta^2\to0$, $\lambda_i=\delta^2/(\delta^2+\sigma_i^2/n_i)\to0$, the precision-weighted $\hat\theta\to\sum(n_i/\sigma_i^2)\bar y_i/\sum(n_i/\sigma_i^2)$, and $E(\mu_i|y)=\hat\theta+\lambda_i(\bar y_i-\hat\theta)$ tends to that synthetic estimator.
```python
from sympy import *

d2 = symbols('delta2', positive=True)
L = 3
n = symbols('n1:4', positive=True)
s = symbols('s1:4', positive=True)   # sigma_i^2
yb = symbols('yb1:4', real=True)

lam = [d2/(d2 + s[i]/n[i]) for i in range(L)]
w   = [1/(d2 + s[i]/n[i]) for i in range(L)]
theta = Add(*[w[i]*yb[i] for i in range(L)]) / Add(*[w[i] for i in range(L)])

# lambda_i -> 0
for i in range(L):
    assert limit(lam[i], d2, 0, '+') == 0

# theta_hat -> synthetic estimator
synthetic = Add(*[n[i]/s[i]*yb[i] for i in range(L)]) / Add(*[n[i]/s[i] for i in range(L)])
assert simplify(limit(theta, d2, 0, '+') - synthetic) == 0

# E(mu_i | y) -> synthetic estimator
for i in range(L):
    E = theta + lam[i]*(yb[i] - theta)
    assert simplify(limit(E, d2, 0, '+') - synthetic) == 0
```

CLAIM c2: As $\delta^2\to\infty$, $\lambda_i\to1$ and $E(\mu_i|y)=\hat\theta+\lambda_i(\bar y_i-\hat\theta)\to\bar y_i$.
```python
from sympy import *

d2 = symbols('delta2', positive=True)
L = 3
n = symbols('n1:4', positive=True)
s = symbols('s1:4', positive=True)
yb = symbols('yb1:4', real=True)

lam = [d2/(d2 + s[i]/n[i]) for i in range(L)]
w   = [1/(d2 + s[i]/n[i]) for i in range(L)]
theta = Add(*[w[i]*yb[i] for i in range(L)]) / Add(*[w[i] for i in range(L)])

for i in range(L):
    assert limit(lam[i], d2, oo) == 1
    E = theta + lam[i]*(yb[i] - theta)
    assert simplify(limit(E, d2, oo) - yb[i]) == 0
```

CLAIM c3: With $\sigma_i^2=\sigma^2$ the synthetic estimator $\sum(n_i/\sigma_i^2)\bar y_i/\sum(n_i/\sigma_i^2)$ equals the pooled mean $\sum n_i\bar y_i/\sum n_i$ (complete pooling).
```python
from sympy import *

L = 3
sig = symbols('sigma2', positive=True)
n = symbols('n1:4', positive=True)
yb = symbols('yb1:4', real=True)

synthetic = Add(*[n[i]/sig*yb[i] for i in range(L)]) / Add(*[n[i]/sig for i in range(L)])
pooled = Add(*[n[i]*yb[i] for i in range(L)]) / Add(*[n[i] for i in range(L)])
assert simplify(synthetic - pooled) == 0
```
