**12.3 Slice sampling**

Slice sampler is a technique used to avoid rejection sampling. Slice sampler is actually a Gibbs sampler.
No normalization constant is needed.

*1. Damien et.al JRSSR [sic] 1999.  2. Neal (2003, AS)*

**example**

(1)

$$f(x) \propto e^{-\frac{1}{2}x^2}, \quad -\infty < x < +\infty$$

Introduce a hidden variable (latent variable),

$$f(x,u) \propto I\left(u < e^{-\frac{1}{2}x^2}\right), \quad u \sim U(0,1)$$

> ⏳ VERIFY[c1]: integrating the joint $f(x,u) \propto I(u < e^{-x^2/2})$ over $u \in (0,1)$ returns the marginal $f(x) \propto e^{-x^2/2}$.

$$u \mid x \sim U\left(0, e^{-\frac{1}{2}x^2}\right)$$

$$x \mid u \sim U\left(x : u < e^{-\frac{1}{2}x^2}\right)$$

$$\ln(u) < -\frac{1}{2}x^2, \quad -2\ln u > x^2, \quad |x| < \sqrt{-2\ln u}$$

> ⏳ VERIFY[c2]: for $0 < u < 1$, the set $\{x : u < e^{-x^2/2}\}$ equals the interval $|x| < \sqrt{-2\ln u}$.

Therefore,

$$x \mid u \sim U(-\sqrt{-2\ln u}, +\sqrt{-2\ln u}) \tag{2}$$

CLAIM c1: Integrating $I(u < e^{-x^2/2})$ over $u \in (0,1)$ yields $e^{-x^2/2}$, recovering the target marginal.
```python
from sympy import *

x, u = symbols('x u', real=True)

# Step 1: the slice threshold t = exp(-x^2/2) satisfies 0 < t <= 1 for all real x,
# because x^2 >= 0 (so -x^2/2 <= 0 and exp of it lies in (0, 1]).
assert ask(Q.nonnegative(x**2)) is True
assert ask(Q.positive(exp(-x**2/2))) is True

# Step 2: for a threshold t in (0, 1], the indicator integral over u in (0,1)
# splits as int_0^t 1 du + int_t^1 0 du = t.
# (integrating the Piecewise directly leaves an unresolved Min(1, exp(-x^2/2)))
t = symbols('t', positive=True)          # t <= 1 established in Step 1
piece = integrate(Integer(1), (u, 0, t)) + integrate(Integer(0), (u, t, 1))
assert simplify(piece - t) == 0

# hence the marginal is exp(-x^2/2), the target kernel
marginal = piece.subs(t, exp(-x**2/2))
assert simplify(marginal - exp(-x**2/2)) == 0

# Step 3: numeric confirmation of the raw indicator integral at several x
from mpmath import mp, quad, exp as mexp
mp.dps = 25
for xv in [0, 0.5, 1.3, -2.7, 4.0]:
    thresh = mexp(-xv**2/2)
    val = quad(lambda uu: 1 if uu < thresh else 0, [0, float(thresh), 1])
    assert abs(val - thresh) < 1e-15

# spot-check the auxiliary fact that the slice never exceeds 1
for val in [Rational(0), Rational(1, 2), Rational(3), Rational(-7, 2)]:
    assert exp(-val**2/2) <= 1
    assert exp(-val**2/2) > 0
```

CLAIM c2: For $0 < u < 1$, $\{x : u < e^{-x^2/2}\} = \{x : |x| < \sqrt{-2\ln u}\}$.
```python
from sympy import *

x = Symbol('x', real=True)

# Exact check at several rational-in-(0,1) values of u: the two sets have the
# same boundary, i.e. u = exp(-x^2/2) iff x^2 = -2*ln(u).
for u0 in [Rational(1, 2), Rational(1, 10), Rational(9, 10), Rational(1, 3)]:
    lhs = solveset(Eq(u0, exp(-x**2/2)), x, S.Reals)
    rhs = FiniteSet(-sqrt(-2*log(u0)), sqrt(-2*log(u0)))
    assert simplify(lhs) == simplify(rhs)

    # and the inequalities agree pointwise on a grid straddling the boundary
    b = sqrt(-2*log(u0))
    for t in [Rational(0), b/2, b*Rational(99, 100), b*Rational(101, 100), 2*b]:
        inside_orig = (u0 < exp(-t**2/2))
        inside_new = (Abs(t) < b)
        assert bool(inside_orig) == bool(inside_new)

# symbolic boundary identity
u = Symbol('u', positive=True)
assert simplify(exp(-(sqrt(-2*log(u)))**2/2) - u) == 0
```
