**8.3.2 How do non-Bayesians predict?**

$$\underbrace{y_1, \dots, y_n}_{\text{observed}}, y_{n+1} \overset{iid.}{\sim} \quad \mathcal{N}(\mu,\sigma^2)$$

===CLAIMS===

CLAIM c1: The Gaussian convolution $\int \mathcal{N}(y\mid\mu,\sigma^2)\mathcal{N}(\mu\mid\bar y,\sigma^2/n)\,d\mu$ equals the $\mathcal{N}(\bar y,(1+1/n)\sigma^2)$ density.
```python
from sympy import *

mu, y = symbols('mu y', real=True)
sigma, ybar = symbols('sigma ybar', positive=True)
n = symbols('n', positive=True)

integrand = (1/sqrt(2*pi*sigma**2)) * exp(-(y - mu)**2 / (2*sigma**2)) \
          * (1/sqrt(2*pi*sigma**2/n)) * exp(-n*(ybar - mu)**2 / (2*sigma**2))

# do the mu-integral for concrete exact parameters (symbolic n makes it conditional)
for nv in [Integer(2), Integer(3), Integer(5), Integer(60)]:
    for sv, yb in [(Integer(1), Integer(0)), (Integer(2), Rational(1, 2))]:
        lhs = integrate(integrand.subs({n: nv, sigma: sv, ybar: yb}), (mu, -oo, oo))
        var = (1 + Rational(1, nv)) * sv**2
        rhs = exp(-(y - yb)**2 / (2*var)) / sqrt(2*pi*var)
        assert simplify(lhs - rhs) == 0
```

CLAIM c2: With prior $p(\mu,\sigma^2)\propto 1/\sigma^2$, the posterior predictive density of $y_{n+1}$ is proportional to the $t_{n-1}$ kernel in $(y_{n+1}-\bar y)/(s\sqrt{1+1/n})$.
```python
from sympy import *

mu, y = symbols('mu y', real=True)
v = symbols('v', positive=True)          # v = 1/sigma^2

for nv in [3, 5, 8]:
    n = Integer(nv)
    s2 = Integer(1)                       # s^2 = 1, ybar = 0 (location/scale free)
    # N(y|mu,1/v) * posterior kernel (mu,sigma^2|y) with prior 1/sigma^2, in terms of v
    integrand = sqrt(v/(2*pi)) * exp(-(y - mu)**2 * v / 2) \
              * v**(Rational(n, 2) + 1) * exp(-((n - 1)*s2 + n*mu**2) * v / 2)
    Imu = simplify(integrate(integrand, (mu, -oo, oo)))
    # change of variable sigma^2 -> v contributes |d sigma^2/dv| = 1/v^2
    pred = integrate(simplify(Imu / v**2), (v, 0, oo))
    scale2 = s2 * (1 + Rational(1, n))
    t_kernel = (1 + (y**2 / scale2) / (n - 1))**(-Rational(n, 2))   # t_{n-1} kernel
    # predictive must be a constant multiple of the t_{n-1} kernel
    assert simplify(diff(log(pred) - log(t_kernel), y)) == 0
```
