## 8.3 Prediction

Prediction is remarkably easy to understand within the Bayesian paradigm.

$$\pi(\theta): \text{ proper or improper prior}$$
$$l(\theta|y) \propto f(y|\theta)$$

- Prior predictive distribution

$$f(y) = \int_{\Theta} f(y|\theta)\pi(\theta)d\theta, \quad \pi(\theta) \text{ must be proper}$$

*[margin note: "Law of total probability"]*

is marginal likelihood.

- Posterior predictive distribution Have data $y_s$ ; need to predict for nonsampled values $y_{ns}$ (or future values)

*(The booklet writes $y_s$ and $y_{ns}$ with under-tildes to indicate vectors.)*

*[margin note: "finite populations"]*

$$f(y_{ns}|y_s) = \int_{\Theta} f(y_{ns}|y_s,\theta)\pi(\theta|y_s)d\theta$$

$$f(y_{ns}|y_s) = \int_{\Theta} f(y_{ns}|\theta)\pi(\theta|y_s)d\theta$$

the posterior distribution $\pi(\theta|y_s)$ must be proper.

### 8.3.1 Examples

1. Normal

$$\underbrace{y_1,\ldots,y_n}_{y\,(\text{obs})}, y_{n+1}|\mu \quad \sim \quad \mathcal{N}(\mu,\sigma^2), \sigma^2 \text{ known}, p(\mu)=1$$

Define $\bar{y} = \sum_{i=1}^{n} \frac{y_i}{n}$, then

$$\mu|y \quad \sim \quad \mathcal{N}\left(\bar{y}, \frac{\sigma^2}{n}\right)$$

> ⏳ VERIFY[c4]: with flat prior $p(\mu)=1$ and $y_i \overset{iid}{\sim} \mathcal{N}(\mu,\sigma^2)$, the posterior is $\mathcal{N}(\bar y, \sigma^2/n)$.

===CLAIMS===

CLAIM c1: p̂ = y/n is the maximizer in p of the binomial(n,p) pmf.
```python
from sympy import *

p, y, n = symbols('p y n', positive=True)
loglik = y*log(p) + (n - y)*log(1 - p)
crit = solve(Eq(diff(loglik, p), 0), p)
assert crit == [y/n]
# second derivative negative at the critical point => maximum
d2 = diff(loglik, p, 2).subs(p, y/n)
assert simplify(d2.subs({y: Rational(3), n: Rational(10)})) < 0
```

CLAIM c2: binom(n,y) * (y/n)^y * (1-y/n)^(n-y) >= 1/(n+1) for all 0 <= y <= n.
```python
from sympy import Rational, binomial

for n in range(1, 41):
    for y in range(0, n + 1):
        ph = Rational(y, n)
        val = binomial(n, y) * ph**y * (1 - ph)**(n - y)
        assert val >= Rational(1, n + 1), (n, y, val)
```

CLAIM c3: the binomial pmf evaluated at a p0 away from the MLE can fall below 1/(n+1).
```python
from sympy import Rational, binomial

n, y, p0 = 10, 5, Rational(1, 10)
val = binomial(n, y) * p0**y * (1 - p0)**(n - y)
assert val < Rational(1, n + 1)
```

CLAIM c4: flat prior + iid normal likelihood gives posterior N(ybar, sigma^2/n).
```python
from sympy import *

mu, sigma = symbols('mu sigma', real=True), symbols('sigma', positive=True)
mu = symbols('mu', real=True)
sigma = symbols('sigma', positive=True)
ys = symbols('y1 y2 y3', real=True)
n = len(ys)
ybar = sum(ys) / n

lik = prod([exp(-(yi - mu)**2 / (2*sigma**2)) / sqrt(2*pi*sigma**2) for yi in ys])
Z = integrate(lik, (mu, -oo, oo))          # flat prior p(mu) = 1
post = simplify(lik / Z)

target = sqrt(n) / sqrt(2*pi*sigma**2) * exp(-n*(mu - ybar)**2 / (2*sigma**2))
assert simplify(post - target) == 0
```
