CLAIM c1: $ste_k=\{(N-k)/(N(N+2))\}^{1/2}$ squares to the exact white-noise variance $(N-k)/(N(N+2))$ of $r_k$, and $\sqrt{N}\,ste_k\to 1$ as $N\to\infty$ for fixed $k$.
```python
from sympy import symbols, sqrt, simplify, limit, oo, Rational

N, k = symbols('N k', positive=True)

ste_k = sqrt((N - k) / (N * (N + 2)))          # as printed on the page
var_rk = (N - k) / (N * (N + 2))               # exact variance of r_k for Gaussian white noise

# the printed standard error is the square root of that variance
assert simplify(ste_k**2 - var_rk) == 0

# ste_k is positive and well-defined whenever k < N
assert simplify(ste_k.subs({N: 100, k: 20}) - sqrt(Rational(80, 100 * 102))) == 0

# large-N behaviour: sqrt(N) * ste_k -> 1 for fixed lag k
assert limit(sqrt(N) * ste_k, N, oo) == 1

# monotone in the lag: variance decreases as k grows (toward N)
assert simplify(var_rk.subs(k, k + 1) - var_rk) == simplify(-1 / (N * (N + 2)))
```

CLAIM c2: the printed band $(-2ste_k, 2ste_k)$ reduces to the standard $\pm 2/\sqrt{N}$ significance band as $N\to\infty$ with $k$ fixed.
```python
from sympy import symbols, sqrt, limit, oo, simplify, Rational

N = symbols('N', positive=True)
k = symbols('k', positive=True)

ste_k = sqrt((N - k) / (N * (N + 2)))
upper = 2 * ste_k

# the band endpoint, rescaled by sqrt(N), tends to the classical value 2
assert limit(sqrt(N) * upper, N, oo) == 2

# and the band is symmetric about zero, so r_k is negligible iff |r_k| < 2*ste_k
lower = -2 * ste_k
assert simplify(upper + lower) == 0

# for fixed k >= 1 the exact band is strictly narrower than the 2/sqrt(N) approximation
n0, k0 = Rational(100), Rational(20)
exact = 2 * sqrt((n0 - k0) / (n0 * (n0 + 2)))
approx = 2 / sqrt(n0)
assert (exact**2 - approx**2) < 0
```
