**11.8 Reversible Jump Markov chain Monte Carlo Sampler**

Green,P.J.(1995), Reversible jumping [sic] Markov chain Monte Carlo computation and Bayesian model determination, Biometrika 82, 711-732.

Have many competing models with different parameter spaces(e.g., dimensions). There is uncertainty about the model itself, creating a parameter which indexes the model. All models are fit simultaneously, and the reversible jump sampler jumps over models.

There are four components

1. Indicidual [sic] models

$$y \mid \theta_s \sim f(y \mid \theta_s, M_s)$$

$$\pi(\theta_s \mid M_s)$$

$s = 1, ..., k$ for $k$ distinct models.

*(The booklet writes $y$, $\theta$ and $u$ with under-tildes to indicate vectors.)*

2. Prior probabilities

$$P(M_s) = w_s, s = 1, ..., k$$

All prior densities must be proper

$$P(M_s \mid y) = \frac{w_s \pi(\theta_s \mid M_s) f(y \mid \theta_s, M_s)}{\sum_{r=1}^{k} w_k \int \pi(\theta_s \mid M_s) f(y \mid \theta_s, M_s) d\theta_r}, s = 1, ..., k$$

$\text{[sic: the denominator's weight should be } w_r \text{ and its integrand } \pi(\theta_r \mid M_r) f(y \mid \theta_r, M_r)\text{, to match the differential } d\theta_r\text{]}$

> ⏳ VERIFY[c1]: with the intended weights and integrands, the posterior model probabilities $P(M_s\mid y) = w_s m_s(y)/\sum_{r=1}^k w_r m_r(y)$ sum to $1$ over $s$.

$$P(M_S \mid y) = \max P(M_s \mid y), s = 1, ..., k.$$

$M_S$ is the selected model(Model selection)

$$\pi(y^{(p)} \mid y) = \sum_{s=1}^{k} \pi(y^{(p)} \mid y, M_s) P(M_s \mid y); \; \text{predicted } y^{(p)}.$$

> ⏳ VERIFY[c2]: the posterior predictive is a proper density — a convex mixture of the model-specific predictives, since the weights $P(M_s\mid y)$ are nonnegative and sum to $1$.

3. Transition probabilities

$$P(M_s \mid M_r) = \pi_{rs}; \sum_{s=1}^{k} \pi_{rs} = 1; r = 1, ..., k.$$

These are not a part of the model specifications, but are chosen to provide good moves.

4. Dimension-matching condition

An elegant idea of Green(1995) is to match the dimension of the two model [sic] under consideration.

Suppose $dim(\theta_s) > dim(\theta_r)$. Introduce $(s-r)$ latent variables into $M_r$ to match the dimension of $M_s$, say $u = (u_1, ..., u_{s-r})$. We need a bijection:

$$\theta_s = A \begin{bmatrix} \theta_r \\ u \end{bmatrix}; \quad \begin{bmatrix} \theta_r \\ u \end{bmatrix} = A^{-1} \theta_s$$

> ⏳ VERIFY[c3]: the stated map is a bijection — $\theta_s = A[\theta_r; u]$ holds iff $[\theta_r; u] = A^{-1}\theta_s$, for any invertible $A$.

CLAIM c1: The posterior model probabilities $w_s m_s(y)/\sum_r w_r m_r(y)$ sum to 1 over $s=1,\dots,k$.
```python
from sympy import symbols, simplify, summation, Sum, Rational, IndexedBase, Idx

# concrete k=3 with symbolic weights and marginal likelihoods
w1, w2, w3 = symbols('w1 w2 w3', positive=True)
m1, m2, m3 = symbols('m1 m2 m3', positive=True)

w = [w1, w2, w3]
m = [m1, m2, m3]
denom = sum(wr * mr for wr, mr in zip(w, m))
post = [ws * ms / denom for ws, ms in zip(w, m)]

# probabilities sum to one
assert simplify(sum(post) - 1) == 0

# each is in [0,1]: p_s = w_s m_s / denom <= 1 since denom - w_s m_s = sum of positive terms
for s in range(3):
    rest = simplify(denom - w[s] * m[s])
    assert simplify(rest - sum(w[r] * m[r] for r in range(3) if r != s)) == 0
```

CLAIM c2: The posterior predictive $\pi(y^{(p)}\mid y)=\sum_s \pi(y^{(p)}\mid y,M_s)P(M_s\mid y)$ integrates to 1 when each component does.
```python
from sympy import symbols, simplify, integrate, exp, oo, Rational, sqrt, pi as spi
from sympy import erf

yp = symbols('yp', real=True)

# three proper component predictives (normal densities with distinct means/sds)
def normal(mu, sd):
    return exp(-(yp - mu)**2 / (2 * sd**2)) / (sd * sqrt(2 * spi))

comps = [normal(Rational(0), Rational(1)),
         normal(Rational(2), Rational(1, 2)),
         normal(Rational(-1), Rational(3, 2))]

# posterior model probabilities from weights and marginals (claim c1 construction)
w = [Rational(1, 2), Rational(1, 3), Rational(1, 6)]
m = [Rational(2), Rational(5), Rational(1)]
denom = sum(wr * mr for wr, mr in zip(w, m))
post = [ws * ms / denom for ws, ms in zip(w, m)]
assert simplify(sum(post) - 1) == 0

# each proper component integrates to 1 (rewrite erfc -> erf so SymPy can
# cancel the split-tail antiderivatives it produces)
for c in comps:
    assert simplify((integrate(c, (yp, -oo, oo)) - 1).rewrite(erf)) == 0

mixture = sum(p * c for p, c in zip(post, comps))
assert simplify((integrate(mixture, (yp, -oo, oo)) - 1).rewrite(erf)) == 0
```

CLAIM c3: $\theta_s = A[\theta_r; u]$ if and only if $[\theta_r; u] = A^{-1}\theta_s$, for invertible $A$ (dimension-matching bijection).
```python
from sympy import Matrix, symbols, simplify, eye

# theta_r has dim 2, u has dim 1, so theta_s has dim 3; A is 3x3 invertible
tr1, tr2, u1 = symbols('tr1 tr2 u1', real=True)

A = Matrix([[1, 2, 0],
            [0, 1, 3],
            [4, 0, 1]])
assert A.det() != 0

stacked = Matrix([tr1, tr2, u1])
theta_s = A * stacked

# forward then inverse recovers the stacked vector exactly
recovered = A.inv() * theta_s
assert simplify(recovered - stacked) == Matrix([0, 0, 0])

# inverse then forward recovers theta_s exactly (bijection both ways)
ts1, ts2, ts3 = symbols('ts1 ts2 ts3', real=True)
ts = Matrix([ts1, ts2, ts3])
assert simplify(A * (A.inv() * ts) - ts) == Matrix([0, 0, 0])
assert simplify(A * A.inv() - eye(3)) == Matrix.zeros(3, 3)
```
