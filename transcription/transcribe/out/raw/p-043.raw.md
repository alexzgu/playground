The Bayes' estimator of $\tau(\theta)$ is $d(x)$, so we search $d(x)$ minimizing

$$E[L(\tau(\theta),d(X))\mid x] = \int L(\tau(\theta),d(x))\pi(\theta\mid x)d\theta$$

Typical loss function of the form $L(\tau(\theta),d) = h(d-\tau(\theta))$

Three loss functions:

1) Squared error loss (SEL).

$$L(\theta,d) = [d-\tau(\theta)]^2$$
$$d(x) = E[\tau(\theta)\mid (x)] \quad \text{[sic: the stray parentheses around } x \text{ appear in the booklet]}$$

> ⏳ VERIFY[c1]: under SEL, the minimizer of $E[(d-\tau(\theta))^2\mid x]$ over $d$ is the posterior mean $E[\tau(\theta)\mid x]$.

## Example

$$X_1,...,X_n\mid\theta \;\sim\; Bernoulli(\theta)$$
$$\theta \;\sim\; Beta(\alpha,\beta)$$
$$\theta\mid x \;\sim\; Beta(s+\alpha, n-s+\beta)$$

> ⏳ VERIFY[c2]: with $X_i\mid\theta\overset{iid}{\sim}Bernoulli(\theta)$, $s=\sum x_i$, and $\theta\sim Beta(\alpha,\beta)$, the posterior is $Beta(s+\alpha,\,n-s+\beta)$.

SEL:

$$E(\theta\mid X) = \frac{s+\alpha}{n+\alpha+\beta}$$

> ⏳ VERIFY[c3]: the mean of $Beta(s+\alpha,\,n-s+\beta)$ equals $\dfrac{s+\alpha}{n+\alpha+\beta}$.

AEL:

$$\int_0^a \frac{\theta^{s+\alpha-1}(1-\theta)^{n-s+\beta-1}}{Beta(s+\alpha,n-s+\beta)}d\theta = \frac{1}{2}, \text{ use a root finder.}$$

> ⏳ VERIFY[c4]: the integrand is a normalized density on $(0,1)$ — i.e. the same integral taken over $[0,1]$ equals $1$ — so the $a$ solving the displayed equation is the posterior median.

===CLAIMS===

CLAIM c1: Under squared error loss, the minimizer over $d$ of the posterior expected loss $E[(d-T)^2\mid x]$ is the posterior mean $E[T\mid x]$.
```python
from sympy import *

d, t = symbols('d t', real=True)
# Represent the posterior of T = tau(theta) by an arbitrary Beta(a,b) posterior;
# the argmin must be the posterior mean regardless of the density.
a, b = Rational(7, 2), Rational(5, 3)
dens = t**(a - 1) * (1 - t)**(b - 1) / beta(a, b)

risk = integrate((d - t)**2 * dens, (t, 0, 1))
risk = simplify(risk)

crit = solve(Eq(diff(risk, d), 0), d)
assert len(crit) == 1
post_mean = simplify(integrate(t * dens, (t, 0, 1)))

assert simplify(crit[0] - post_mean) == 0
# and it is a minimum
assert simplify(diff(risk, d, 2)) > 0
# sanity: Beta mean
assert simplify(post_mean - a / (a + b)) == 0
```

CLAIM c2: Bernoulli likelihood with Beta(alpha,beta) prior yields a Beta(s+alpha, n-s+beta) posterior.
```python
from sympy import *

th = symbols('theta', positive=True)

# exact integer parameters (polynomial integrand -> fast exact integration)
alpha, beta_ = Integer(2), Integer(3)
n, s = 7, 3

prior = th**(alpha - 1) * (1 - th)**(beta_ - 1) / beta(alpha, beta_)
lik = th**s * (1 - th)**(n - s)

unnorm = prior * lik
norm = integrate(expand(unnorm), (th, 0, 1))
posterior = unnorm / norm

claimed_a = s + alpha
claimed_b = n - s + beta_
claimed = th**(claimed_a - 1) * (1 - th)**(claimed_b - 1) / beta(claimed_a, claimed_b)

# rewrite unevaluated beta(...) via gamma so the ratio closes exactly
assert simplify((posterior - claimed).rewrite(gamma)) == 0

# and a second point (asymmetric s) for good measure
n2, s2 = 5, 1
un2 = prior * th**s2 * (1 - th)**(n2 - s2)
post2 = un2 / integrate(expand(un2), (th, 0, 1))
cl2 = th**(s2 + alpha - 1) * (1 - th)**(n2 - s2 + beta_ - 1) / beta(s2 + alpha, n2 - s2 + beta_)
assert simplify((post2 - cl2).rewrite(gamma)) == 0
```

CLAIM c3: The posterior mean is (s+alpha)/(n+alpha+beta).
```python
from sympy import *

th = symbols('theta', positive=True)
alpha, beta_ = Integer(2), Integer(3)
n, s = 7, 3

A = s + alpha
B = n - s + beta_
dens = th**(A - 1) * (1 - th)**(B - 1) / beta(A, B)

mean = simplify(integrate(expand(th * dens), (th, 0, 1)).rewrite(gamma))
booklet = Rational(1, 1) * (s + alpha) / (n + alpha + beta_)

assert simplify(mean - booklet) == 0

# also check the general symbolic Beta-mean identity A/(A+B) matches the booklet form
assert simplify(A / (A + B) - booklet) == 0
```

CLAIM c4: The AEL integrand is a normalized density on (0,1), so the equation defines the posterior median.
```python
from sympy import *

th, a = symbols('theta a', positive=True)
alpha, beta_ = Integer(2), Integer(3)
n, s = 7, 3

A = s + alpha
B = n - s + beta_
dens = th**(A - 1) * (1 - th)**(B - 1) / beta(A, B)

# normalization: total mass is exactly 1
assert simplify(integrate(expand(dens), (th, 0, 1)).rewrite(gamma) - 1) == 0

# the root of the booklet's equation lies strictly inside (0,1) and is the median:
F = integrate(expand(dens), (th, 0, a))          # polynomial in a
F0 = simplify(F.subs(a, 0).rewrite(gamma))
F1 = simplify(F.subs(a, 1).rewrite(gamma))
assert F0 == 0
assert F1 == 1

# median is bracketed: F(0) - 1/2 < 0 < F(1) - 1/2
assert F0 - Rational(1, 2) < 0
assert F1 - Rational(1, 2) > 0

# and F is strictly increasing on (0,1) (F' = dens > 0), so the root is unique
assert simplify(diff(F, a) - dens.subs(th, a)) == 0
```
