Finally, I mention Dirichlet process mixture (DPM) model,

$$y_{ij}\mid \theta_i, \sigma^2 \overset{ind}{\sim} N(\theta_i, \sigma^2),\; j = 1, \cdots, n_i,\; i = 1, \cdots, l$$

$$\theta_i \mid G \overset{iid}{\sim} G$$

$$G \mid \eta, \delta^2 \sim DP(\alpha, N(\eta, \delta^2))$$

If $\alpha \to$, we get the normal-means model. There are two methods to fit model. *[sic: the limit is left blank after "$\alpha \to$"; also "to fit model" is missing an article]*

(1) Marginal method
Here, the infinite dimensional distribution, $G$, is integrated out – Escobar and West (JASA 1994),

$$\theta_1 \sim N(\eta, \delta^2)$$

$$\theta_2 \mid \theta_1 \sim \frac{1}{1+\alpha}\delta_{\theta_1}(\theta_2) + \frac{\alpha}{1+\alpha} N_{\theta_2}(\eta, \delta^2)$$

$$\theta_3 \mid \theta_1, \theta_2 \sim \frac{2}{2+\alpha}\,\frac{\delta_{\theta_1}(\theta_3) + \delta_{\theta_2}(\theta_3)}{2} + \frac{\alpha}{2+\alpha} N_{\theta_3}(\eta, \delta^2)$$

$$\vdots$$

from sympy import symbols, simplify, Rational, Symbol

alpha = symbols('alpha', positive=True)

# Line for theta_2 | theta_1 : weights 1/(1+a) on delta_{theta_1}, a/(1+a) on the base measure
w_atom_1 = 1/(1 + alpha)
w_base_1 = alpha/(1 + alpha)
assert simplify(w_atom_1 + w_base_1 - 1) == 0

# Line for theta_3 | theta_1, theta_2 : weights 2/(2+a) on the average of two atoms, a/(2+a) on base
w_atom_2 = 2/(2 + alpha)
w_base_2 = alpha/(2 + alpha)
assert simplify(w_atom_2 + w_base_2 - 1) == 0

# The averaged-atom form 2/(2+a) * (d1 + d2)/2 puts total mass 2/(2+a) on the atoms,
# i.e. mass 1/(2+a) on each atom.
d1, d2 = symbols('d1 d2', real=True)   # indicator placeholders for delta_{theta_1}, delta_{theta_2}
printed = w_atom_2 * (d1 + d2)/2
assert simplify(printed - (d1/(2 + alpha) + d2/(2 + alpha))) == 0

# General Polya urn: theta_{n+1} ~ n/(n+a) * (1/n) sum_i delta_i + a/(n+a) * N(eta, delta^2)
def urn(n, atoms):
    return (Rational(n, 1)/(n + alpha)) * sum(atoms)/n, alpha/(n + alpha)

# n = 1 must reproduce the theta_2 line
atom_part, base_part = urn(1, [d1])
assert simplify(atom_part - d1/(1 + alpha)) == 0
assert simplify(base_part - w_base_1) == 0

atom_part, base_part = urn(2, [d1, d2])
assert simplify(atom_part - printed) == 0
assert simplify(base_part - w_base_2) == 0
