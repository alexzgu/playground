**Figure 10.6: Selection model for nonignorable nonresponse**

$$p(y,z,r|\pi_0,\pi_1,p) = \frac{n!(\pi_1 p)^y((1-\pi_1)p)^z(\pi_0(1-p))^{r-y}((1-\pi_0)(1-p))^{n-r-z}}{y!z!(r-y)!(n-r-z)!}$$

> ÔÅ│ VERIFY[c1]: the four cell probabilities sum to 1 and the four counts sum to $n$, so the stated expression is a proper multinomial mass function.

$\pi_0,\pi_1$ cannot be estimated.

Take $\pi_0,\pi_1$ $\underset{\sim}{ind}$ $Beta(\alpha_0,\beta_0)$, $\alpha_0,\beta_0$ are to be specified. *(The booklet writes "ind" with an under-tilde, meaning independently distributed.)*

Independently, $p \sim Uniform(0,1)$

How to specify $\alpha_0,\beta_0$? Take $\pi_1 = \pi_0$ (ignorable nonresponse model)

$$\pi|y,z,r \;\sim\; Beta(r+1, n-r+1)$$

> ÔÅ│ VERIFY[c2]: with $\pi\sim Uniform(0,1)$ and $r\mid\pi\sim Binomial(n,\pi)$, the posterior of $\pi$ is $Beta(r+1,n-r+1)$.

So that $\alpha_0 = r+1, \beta_0 = n-r+1$ (data dependent prior).
Then, a posterior,

$$\pi(p,\pi_0,\pi_1,z|y,r) \propto \frac{p^{y+z}(1-p)^{n-(y+z)}\pi_0^{r-y+\alpha_0-1}(1-\pi_0)^{n-r-z+\beta_0-1}\pi_1^{y+\alpha_0-1}(1-\pi_1)^{z+\beta_0-1}}{z!(n-r-z)!}$$

> ÔÅ│ VERIFY[c3]: the likelihood times the $Beta(\alpha_0,\beta_0)$ priors on $\pi_0,\pi_1$ and the $Uniform(0,1)$ prior on $p$ equals the displayed kernel up to a factor free of $(p,\pi_0,\pi_1,z)$.

===CLAIMS===

CLAIM c1: The four selection-model cell probabilities sum to 1 and the four cell counts sum to $n$.
```python
from sympy import *

p, pi0, pi1 = symbols('p pi0 pi1', positive=True)
n, r, y, z = symbols('n r y z', positive=True)

probs = pi1*p + (1-pi1)*p + pi0*(1-p) + (1-pi0)*(1-p)
assert simplify(probs - 1) == 0

counts = y + z + (r - y) + (n - r - z)
assert simplify(counts - n) == 0
```

CLAIM c2: Uniform(0,1) prior with Binomial(n, pi) likelihood for r gives posterior Beta(r+1, n-r+1).
```python
from sympy import *

pi = symbols('pi', positive=True)

for nn, rr in [(5, 2), (7, 0), (7, 7), (9, 4)]:
    lik = binomial(nn, rr) * pi**rr * (1-pi)**(nn-rr)
    post = simplify(lik / integrate(lik, (pi, 0, 1)))
    a, b = rr + 1, nn - rr + 1
    beta_pdf = pi**(a-1) * (1-pi)**(b-1) / beta(a, b)
    assert simplify(post - beta_pdf) == 0
```

CLAIM c3: Likelihood x Beta(a0,b0) priors on pi0,pi1 x Uniform(0,1) on p equals the printed posterior kernel up to a factor free of (p, pi0, pi1, z).
```python
from sympy import *

p, pi0, pi1 = symbols('p pi0 pi1', positive=True)
a0, b0 = symbols('alpha0 beta0', positive=True)

n, r, y = 7, 4, 3          # fixed data
for z in [0, 1, 2, 3]:     # z is the unobserved count being sampled
    lik = (factorial(n)
           * (pi1*p)**y * ((1-pi1)*p)**z
           * (pi0*(1-p))**(r-y) * ((1-pi0)*(1-p))**(n-r-z)
           / (factorial(y)*factorial(z)*factorial(r-y)*factorial(n-r-z)))
    prior = (pi0**(a0-1)*(1-pi0)**(b0-1)/beta(a0, b0)
             * pi1**(a0-1)*(1-pi1)**(b0-1)/beta(a0, b0))
    joint = lik * prior

    kernel = (p**(y+z) * (1-p)**(n-(y+z))
              * pi0**(r-y+a0-1) * (1-pi0)**(n-r-z+b0-1)
              * pi1**(y+a0-1) * (1-pi1)**(z+b0-1)
              / (factorial(z)*factorial(n-r-z)))

    ratio = simplify(joint / kernel)
    # the proportionality constant must not depend on p, pi0, pi1, or z
    assert simplify(diff(ratio, p)) == 0
    assert simplify(diff(ratio, pi0)) == 0
    assert simplify(diff(ratio, pi1)) == 0
    expected = factorial(n)/(factorial(y)*factorial(r-y)*beta(a0, b0)**2)
    assert simplify(ratio - expected) == 0
```
