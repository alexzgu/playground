**12.1.1 Deviance information criteria ($DIC$)**

Deviance as: $D(y\mid\theta) = -2\log[P(y\mid\theta)]$. Let $\hat{\theta}$ be the posterior mean of $\theta$. Define

$$D_{\hat{\theta}}(y) = D(y\mid\hat{\theta})$$

$$D_{avg}(y) = E_{\theta\mid y}[D(y\mid\theta)\mid y]$$

$$\hat{D}_{avg} = \frac{1}{M}\sum_{n=1}^{M} D(y\mid\theta^{(n)}), \quad M \text{ samples.}$$

*(correction: a hat is added by hand over the printed $D_{avg}$, and a small trailing mark that may read "$(y)$" (?) — the Monte Carlo estimate of $D_{avg}$)*

We define the complexity of the model (effective number of parameters) as,

$$P_D = \hat{D}_{avg}(y) - D_{\hat{\theta}}(y)$$

$$DIC = 2\hat{D}_{avg}(y) - D_{\hat{\theta}}(y)$$

*(correction: in both displays a hat is added by hand over the printed $D_{avg}$)*

> ⏳ VERIFY[c1]: with $P_D = \hat{D}_{avg} - D_{\hat\theta}$, the stated $DIC = 2\hat{D}_{avg} - D_{\hat\theta}$ is identically $D_{\hat\theta} + 2P_D$.

Have two or more models, select the one with the smallest $DIC$.

CLAIM c1: With $P_D = \hat D_{avg} - D_{\hat\theta}$, the booklet's $DIC = 2\hat D_{avg} - D_{\hat\theta}$ equals $D_{\hat\theta} + 2P_D$.
```python
from sympy import symbols, simplify

D_avg, D_hat = symbols('D_avg D_hat', real=True)

P_D = D_avg - D_hat            # booklet: P_D = D_avg(y) - D_thetahat(y)
DIC = 2*D_avg - D_hat          # booklet: DIC = 2 D_avg(y) - D_thetahat(y)

# Standard equivalent form: DIC = D_thetahat + 2 P_D
assert simplify(DIC - (D_hat + 2*P_D)) == 0
# and equivalently DIC = D_avg + P_D
assert simplify(DIC - (D_avg + P_D)) == 0
```

CLAIM c2: In the SAT-schools DIC table, the minimum DIC is 61.5, attained at $\delta^2 = 0$ (the starred row).
```python
from fractions import Fraction

table = {
    'delta2=inf':     Fraction('79.3'),
    'delta2=0':       Fraction('61.5'),
    'delta2=unknown': Fraction('63.4'),
}
starred = 'delta2=0'

best = min(table, key=lambda k: table[k])
assert best == starred
assert table[best] == Fraction('61.5')
# strict minimum: no other model ties it
assert all(v > table[starred] for k, v in table.items() if k != starred)
```
