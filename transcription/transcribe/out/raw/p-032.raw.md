$$\mu \sim N(\theta, \delta^2), \text{ assume } \theta, \sigma^2, \delta^2 \text{ are known.}$$

So the typical form of the posterior mean is: $E(\mu|y) = \lambda y + (1-\lambda)\theta$, where $\lambda = \frac{\delta^2}{\delta^2 + \sigma^2}$.

> ⏳ VERIFY[c1]: for $y\mid\mu \sim N(\mu,\sigma^2)$, $\mu \sim N(\theta,\delta^2)$, the posterior mean is $\lambda y + (1-\lambda)\theta$ with $\lambda = \delta^2/(\delta^2+\sigma^2)$.

$\lambda$ is called the shrinkage parameter.

For a shrinkage prior, take $\lambda \sim U(0,1)$ *[margin note: "Strawderman (1980's)"]*

Assume $\delta^2$ is fixed, look for the prior for $\sigma^2$

Since $\lambda = \frac{\delta^2}{\delta^2 + \sigma^2}$, then we can get the shrinkage prior for $\sigma^2$:

$$\pi(\sigma^2) \propto \frac{\delta^2}{(\delta^2+\sigma^2)^2}, \sigma^2 > 0.$$

> ⏳ VERIFY[c2]: transforming $\lambda \sim U(0,1)$ through $\lambda = \delta^2/(\delta^2+\sigma^2)$ gives density $\propto \delta^2/(\delta^2+\sigma^2)^2$ on $\sigma^2>0$.

In some papers (*Albert (1980's), Christiansen and Morris (1997)*), $\pi(\sigma^2) \propto \frac{1}{(1+\sigma^2)^2}, \sigma^2 > 0$.

Example

$$X|p \sim Bernoulli(p)$$

$$p \sim Beta(\alpha, \beta)$$

If we let $\alpha = \mu\tau, \beta = (1-\mu)\tau, 0 < \mu < 1, \tau > 0$.

Then we can rewrite the model as:

$$X|p \sim Bernoulli(p)$$

$$p \sim Beta(\mu\tau, (1-\mu)\tau)$$

Construct a shrinkage prior for $\tau$.

The posterior mean of $p$ is $E(p|x) = \lambda x + (1-\lambda)\mu, \lambda = \frac{1}{\tau+1}$

> ⏳ VERIFY[c5]: for $X\mid p \sim \text{Bernoulli}(p)$, $p \sim \text{Beta}(\mu\tau,(1-\mu)\tau)$, $E(p\mid x) = \lambda x + (1-\lambda)\mu$ with $\lambda = 1/(\tau+1)$.

Take $\lambda \sim U(0,1)$, so $\frac{1}{\tau+1} \sim U(0,1)$, then by transformation,

$\pi(\tau) = \frac{1}{(1+\tau)^2}, \tau > 0$.

> ⏳ VERIFY[c6]: if $1/(\tau+1) \sim U(0,1)$ then $\pi(\tau) = 1/(1+\tau)^2$ on $\tau>0$.

===CLAIMS===

CLAIM c1: Normal–normal posterior mean equals $\lambda y+(1-\lambda)\theta$ with $\lambda=\delta^2/(\delta^2+\sigma^2)$.
```python
from sympy import *

mu = symbols('mu', real=True)
y, theta = symbols('y theta', real=True)
sig2, del2 = symbols('sigma2 delta2', positive=True)

# unnormalized posterior: likelihood N(mu, sig2) times prior N(theta, del2)
post = exp(-(y - mu)**2 / (2*sig2)) * exp(-(mu - theta)**2 / (2*del2))

Z = integrate(post, (mu, -oo, oo))
M = integrate(mu * post, (mu, -oo, oo))
post_mean = simplify(M / Z)

lam = del2 / (del2 + sig2)
claimed = lam*y + (1 - lam)*theta

assert simplify(post_mean - claimed) == 0
```

CLAIM c2: $\lambda=\delta^2/(\delta^2+\sigma^2)$ with $\lambda\sim U(0,1)$ induces density $\delta^2/(\delta^2+\sigma^2)^2$ on $\sigma^2>0$.
```python
from sympy import *

s, d2 = symbols('s delta2', positive=True)   # s plays the role of sigma^2

lam = d2 / (d2 + s)

# lambda is a decreasing 1-1 map from (0, oo) onto (0, 1)
assert simplify(limit(lam, s, 0, '+') - 1) == 0
assert limit(lam, s, oo) == 0
assert simplify(diff(lam, s) + d2/(d2 + s)**2) == 0   # strictly decreasing

# Jacobian rule: pi(s) = f_lambda(lam(s)) * |d lam / d s| = 1 * |d lam / d s|
induced = Abs(diff(lam, s))
claimed = d2 / (d2 + s)**2
assert simplify(induced - claimed) == 0

# and it is a proper density on (0, oo)
assert simplify(integrate(claimed, (s, 0, oo)) - 1) == 0
```

CLAIM c3: $1/(1+\sigma^2)^2$ is exactly the $F(2,2)$ density.

from sympy import *

CLAIM c5: Beta–Bernoulli posterior mean is $\lambda x+(1-\lambda)\mu$ with $\lambda=1/(\tau+1)$.
```python
from sympy import symbols, beta, simplify, gammasimp, Rational

m, t = symbols('mu tau', positive=True)

lam = 1 / (t + 1)

for x in (0, 1):
    # posterior is Beta(a, b) with a = mu*tau + x, b = (1-mu)*tau + 1 - x
    a = m*t + x
    b = (1 - m)*t + 1 - x
    # E[p | x] = B(a+1, b)/B(a, b) = a/(a+b); reduce the gamma-function ratio
    post_mean = gammasimp(beta(a + 1, b) / beta(a, b))
    claimed = lam*x + (1 - lam)*m
    assert simplify(post_mean - claimed) == 0

# numeric spot check of the full integral definition at exact rational values
from mpmath import mp, quad
mp.dps = 30
for x in (0, 1):
    for mv, tv in [(Rational(1, 3), Rational(5, 2)), (Rational(3, 4), Rational(7, 1))]:
        a, b = float(mv*tv), float((1 - mv)*tv)
        post = lambda p: p**(a + x - 1) * (1 - p)**(b + (1 - x) - 1)
        Z = quad(post, [0, 1])
        M = quad(lambda p: p*post(p), [0, 1])
        claimed = float((lam*x + (1 - lam)*m).subs({m: mv, t: tv}))
        assert abs(float(M/Z) - claimed) < 1e-12
```

CLAIM c6: $1/(\tau+1)\sim U(0,1)$ induces $\pi(\tau)=1/(1+\tau)^2$ on $\tau>0$.

from sympy import *

t = symbols('tau', positive=True)

lam = 1 / (t + 1)
assert simplify(limit(lam, t, 0, '+') - 1) == 0
assert limit(lam, t, oo) == 0

induced = Abs(diff(lam, t))          # |d lambda / d tau|, uniform density = 1

assert simplify(integrate(claimed, (t, 0, oo)) - 1) == 0
