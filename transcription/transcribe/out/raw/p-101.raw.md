CLAIM c1: With $\overline{\nu}=\sum\lambda_i\hat\nu_i/\sum\lambda_i$, the weighted sum of squares decomposes as $\sum\lambda_i(\theta-\hat\nu_i)^2=\sum\lambda_i(\theta-\overline\nu)^2+\sum\lambda_i(\hat\nu_i-\overline\nu)^2$.
```python
from sympy import *

l = 4
theta = symbols('theta', real=True)
lam = symbols('lam1:5', positive=True)
nu = symbols('nu1:5', real=True)

nubar = Add(*[lam[i]*nu[i] for i in range(l)]) / Add(*[lam[i] for i in range(l)])

lhs = Add(*[lam[i]*(theta - nu[i])**2 for i in range(l)])
rhs = (Add(*[lam[i] for i in range(l)])*(theta - nubar)**2
       + Add(*[lam[i]*(nu[i] - nubar)**2 for i in range(l)]))

assert simplify(expand(lhs - rhs)) == 0
```

CLAIM c2: The exponential kernel in $\theta$ is proportional to a $N(\overline\nu,\ \delta^2/\sum\lambda_i)$ density.
```python
from sympy import *

theta, nubar = symbols('theta nubar', real=True)
delta2, L = symbols('delta2 L', positive=True)   # L = sum of lambda_i

kernel = exp(-(L/(2*delta2))*(theta - nubar)**2)

# N(nubar, delta2/L) density
var = delta2 / L
dens = 1/sqrt(2*pi*var) * exp(-(theta - nubar)**2/(2*var))

ratio = simplify(dens / kernel)
# ratio must not depend on theta (i.e. proportionality in theta)
assert simplify(diff(ratio, theta)) == 0
# and the normalizing constant is exactly sqrt(L/(2*pi*delta2))
assert simplify(ratio - sqrt(L/(2*pi*delta2))) == 0
```

CLAIM c5: Under $\gamma=\sigma^2$ the half-Cauchy becomes $f(\gamma)=\frac{1}{\pi(1+\gamma)\sqrt{\gamma}}$, a proper density on $(0,\infty)$.
```python
from sympy import *

gamma = symbols('gamma', positive=True)
sigma = symbols('sigma', positive=True)

f_sigma = 2/(pi*(1 + sigma**2))

# sigma = sqrt(gamma), |d sigma / d gamma| = 1/(2 sqrt(gamma))
jac = Abs(diff(sqrt(gamma), gamma))
assert simplify(jac - 1/(2*sqrt(gamma))) == 0

f_gamma = simplify(f_sigma.subs(sigma, sqrt(gamma)) * jac)
booklet = 1/(pi*(1 + gamma)*sqrt(gamma))

assert simplify(f_gamma - booklet) == 0
assert simplify(integrate(booklet, (gamma, 0, oo)) - 1) == 0
```

CLAIM c7: The transformed density is $\pi(\phi)=\frac{1}{\pi\sqrt{\phi(1-\phi)}}$ on $(0,1)$, i.e. exactly the Beta(1/2, 1/2) density.
```python
from sympy import *

phi = symbols('phi', positive=True)
g = symbols('g', positive=True)                 # g = gamma = sigma^2 (do not shadow sympy's gamma)

f_gamma = 1/(pi*(1 + g)*sqrt(g))                # density of sigma^2 (claim c5)
s2 = (1 - phi)/phi                              # sigma^2 in terms of phi
jac = Abs(diff(s2, phi))                        # = 1/phi**2

pi_phi = simplify(f_gamma.subs(g, s2) * jac)
booklet = 1/(pi*sqrt(phi*(1 - phi)))

# restrict to the stated support 0 < phi < 1 so sqrt((1-phi)/phi) splits
assert simplify((pi_phi - booklet).rewrite(Pow).subs(phi, Rational(1, 3))) == 0
assert simplify((pi_phi - booklet).subs(phi, Rational(7, 10))) == 0
assert simplify((pi_phi - booklet).subs(phi, Rational(1, 100))) == 0

# and it is the Beta(1/2,1/2) density (use the gamma form: B(1/2,1/2) = pi,
# which SymPy's unevaluated beta() will not reduce on its own)
a = b = Rational(1, 2)
B = gamma(a)*gamma(b)/gamma(a + b)
assert simplify(B - pi) == 0
beta_pdf = phi**(a - 1)*(1 - phi)**(b - 1)/B
assert simplify(beta_pdf - booklet) == 0
assert simplify(integrate(booklet, (phi, 0, 1)) - 1) == 0
```
