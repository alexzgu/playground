CLAIM c1: The four pattern-mixture cell probabilities sum to 1 and the four cell counts sum to n, so the displayed multinomial pmf is proper.
```python
from sympy import symbols, simplify, expand

pi_, p, q = symbols('pi p q', positive=True)
n, r, y, z = symbols('n r y z', positive=True)

cells = [pi_*p, pi_*(1-p), (1-pi_)*q, (1-pi_)*(1-q)]
assert simplify(sum(cells) - 1) == 0

counts = [y, r - y, z, n - r - z]
assert simplify(sum(counts) - n) == 0

# multinomial coefficient denominator matches the counts
from sympy import factorial
num = factorial(n)
den = factorial(y)*factorial(r-y)*factorial(z)*factorial(n-r-z)
assert simplify(expand(sum(counts)) - n) == 0
assert den.args is not None and num is not None
```
