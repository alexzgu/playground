**8.2.1 Bayes Factor**

**Definition.**

$$\text{Prior odds} := P(M_1)/P(M_2)$$

$$\text{Posterior odds} := P(M_1 \mid x)/P(M_2 \mid x)$$

*(The booklet writes $x$ with an under-tilde to indicate a vector.)*

$$\text{Bayes factor} := \frac{\text{Posterior odds}}{\text{Prior odds}}$$

Interpretation: Evidence provided by the data for $M_1$ beyond that provided by the prior.

`Remark (1):` Evidence for $M_1$: $M_1$ in numerator.

`Remark (2):` There is symmetry in $M_1$ and $M_2$.

Measure of strength of evidence for $M_1$

*[margin note, left of this table: "Rule of thumb"]*

| Bayes Factor (BF) | log(B.F.) | Evidence |
|---|---|---|
| $1 \le \text{BF} < 3$ | $0 - 1.099$ | little |
| $3 \le \text{BF} < 20$ | $1.099 - 2.996$ | positive |
| $20 \le \text{BF} < 150$ | $2.996 - 5.011$ | strong |
| $\text{BF} \ge 150$ | $\ge 5.011$ | very strong |

> ⏳ VERIFY[c2]: the log(B.F.) cut points $1.099, 2.996, 5.011$ are the natural logs of the BF cut points $3, 20, 150$, rounded to three decimals.

*Kass and Raftery (1995, JASA)*

CLAIM c1: exp(log(ratio)) reproduces each printed ratio to the precision printed
```python
import math

# (log_ratio, printed_ratio, sig_figs_or_rule)
rows = [
    (0.0,   1.0),
    (4.67,  107.0),
    (11.53, 1e5),
    (20.61, 8.9e8),
    (31.88, 7.0e13),
    (45.36, 5.0e19),
]

def sig_round(x, n):
    if x == 0:
        return 0.0
    from math import floor, log10
    return round(x, -int(floor(log10(abs(x)))) + (n - 1))

# 22: exp(0) == 1 exactly
assert math.exp(rows[0][0]) == 1.0

# 23: exp(4.67) rounds to 107 (3 sig figs)
assert sig_round(math.exp(4.67), 3) == 107.0

# 24: exp(11.53) rounds to 1e5 (1 sig fig)
assert sig_round(math.exp(11.53), 1) == 1e5

# 25: exp(20.61) rounds to 8.9e8 (2 sig figs)
assert sig_round(math.exp(20.61), 2) == 8.9e8

# 26: exp(31.88) rounds to 7.0e13 (2 sig figs)
assert sig_round(math.exp(31.88), 2) == 7.0e13

# 27: exp(45.36) rounds to 5.0e19 (2 sig figs)
assert sig_round(math.exp(45.36), 2) == 5.0e19

# monotone increasing evidence as mu_1 increases (Remark 2)
lrs = [r[0] for r in rows]
assert all(b > a for a, b in zip(lrs, lrs[1:]))
```

CLAIM c2: the log-Bayes-factor cut points are ln(3), ln(20), ln(150) rounded to 3 decimals
```python
import math

assert round(math.log(3), 3) == 1.099
assert round(math.log(20), 3) == 2.996
assert round(math.log(150), 3) == 5.011

# and the printed lower endpoint of the first band is log(1) = 0
assert math.log(1) == 0.0

# bands are contiguous: each band's upper log endpoint is the next band's lower one
bands = [(1, 3), (3, 20), (20, 150)]
logs = [(round(math.log(a), 3), round(math.log(b), 3)) for a, b in bands]
assert all(logs[i][1] == logs[i + 1][0] for i in range(len(logs) - 1))
```
