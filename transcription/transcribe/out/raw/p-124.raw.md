CLAIM c1: With the Gibbs proposal $q(x,y)=\pi(y|x)$ (so $q(y,x)=\pi(x|y)$), the M-H acceptance ratio $\pi(y)\pi(x|y)/[\pi(x)\pi(y|x)]$ reduces to $\pi(x,y)/\pi(x,y)=1$, hence $\alpha(x,y)=1$.
```python
from sympy import *

x, y = symbols('x y', real=True)
r = Rational(1, 2)  # correlation, exact

# Standard bivariate normal joint density with correlation r
joint = (1/(2*pi*sqrt(1-r**2))) * exp(-(x**2 - 2*r*x*y + y**2)/(2*(1-r**2)))

# Marginals are standard normal
marg_x = exp(-x**2/2)/sqrt(2*pi)
marg_y = exp(-y**2/2)/sqrt(2*pi)

# Confirm the marginals really are the marginals of `joint`
assert simplify(integrate(joint, (y, -oo, oo)) - marg_x) == 0
assert simplify(integrate(joint, (x, -oo, oo)) - marg_y) == 0

# Conditionals: y|x ~ N(r*x, 1-r^2),  x|y ~ N(r*y, 1-r^2)
cond_y_given_x = exp(-(y - r*x)**2/(2*(1-r**2)))/sqrt(2*pi*(1-r**2))
cond_x_given_y = exp(-(x - r*y)**2/(2*(1-r**2)))/sqrt(2*pi*(1-r**2))
assert simplify(cond_y_given_x - joint/marg_x) == 0
assert simplify(cond_x_given_y - joint/marg_y) == 0

# Booklet: pi(x,y) = pi(y)pi(x|y) = pi(x)pi(y|x)
assert simplify(marg_y*cond_x_given_y - joint) == 0
assert simplify(marg_x*cond_y_given_x - joint) == 0

# Eq (11.2) -> (11.3) -> (11.4): the ratio is identically 1
ratio = (marg_y*cond_x_given_y)/(marg_x*cond_y_given_x)
assert simplify(ratio - 1) == 0
assert simplify(Min(1, ratio) - 1) == 0
```

CLAIM c2: For the random walk chain, if $z$ has pdf $g$ = the $Normal(0,\sigma^2)$ density and $y = x + z$, then $q(x,y) = g(x-y)$ coincides with the $Normal(x,\sigma^2)$ density in $y$; the proposal is also symmetric, $q(x,y)=q(y,x)$.
```python
from sympy import *

x, y, z = symbols('x y z', real=True)
sigma = symbols('sigma', positive=True)

g = lambda t: exp(-t**2/(2*sigma**2))/sqrt(2*pi*sigma**2)   # pdf of z ~ N(0, sigma^2)

q = g(x - y)                                                 # booklet: q(x,y) = g(x - y)
normal_x_sigma2 = exp(-(y - x)**2/(2*sigma**2))/sqrt(2*pi*sigma**2)   # N(x, sigma^2) at y

# q(x, y) is exactly the Normal(x, sigma^2) density in y  =>  y|x ~ Normal(x, sigma^2)
assert simplify(q - normal_x_sigma2) == 0

# It is a proper density in y
assert simplify(integrate(q, (y, -oo, oo)) - 1) == 0

# Symmetry of the random walk proposal: q(x,y) = q(y,x)
assert simplify(g(x - y) - g(y - x)) == 0

# And y = x + z with z ~ g indeed has the density q(x, .): convolution/change of variable
dens_of_y = g(z).subs(z, y - x)
assert simplify(dens_of_y - q) == 0
```
