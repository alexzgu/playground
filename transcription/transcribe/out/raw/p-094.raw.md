**9.2.3 Empirical Bayes Statistics**

$$E\left(\mu_i \mid \boldsymbol{y}\right) = \lambda_i \bar{y}_i + \left(1-\lambda_i\right)\hat{\theta}$$

$$\lambda_i = \frac{\delta^2}{\delta^2 + \sigma_i^2/n_i}$$

$$\bar{y} = \frac{\sum_{i=1}^{l} \lambda_i \bar{y}_i}{\sum_{i=1}^{l} \lambda_i}$$

$$\hat{\mu}_i = \lambda_i \bar{y}_i + \left(1-\lambda_i\right)\hat{\theta}$$

> ⏳ VERIFY[c1]: in the normal–normal model $\bar{Y}_i\mid\mu_i \sim N(\mu_i, \sigma_i^2/n_i)$, $\mu_i \sim N(\theta,\delta^2)$, the posterior mean of $\mu_i$ is $\lambda_i\bar{y}_i + (1-\lambda_i)\theta$ with $\lambda_i = \delta^2/(\delta^2+\sigma_i^2/n_i)$.

> ⏳ VERIFY[c2]: $\bar{y} = \sum_i \lambda_i\bar{y}_i / \sum_i \lambda_i$ is the precision-weighted mean of the $\bar{y}_i$ under the marginal law $\bar{Y}_i \sim N(\theta, \delta^2 + \sigma_i^2/n_i)$.

Because $\sigma_i^2$ and $\delta^2$ are really unknown, we substitute reasonable estimators for them in $\hat{\mu}_i$. Now we can proceed in one of two ways:

a. Continue as a Bayesian

$$\text{Bayes}\qquad \mu_i \mid \boldsymbol{y} \sim N\left(\hat{\mu}_i,\; \frac{\left(1-\lambda_i\right)^2 \nu^2 + \left(1-\lambda_i\right)\delta^2}{\hat{\delta}^2,\, \hat{\sigma}_i^2}\right)$$

> ⏳ VERIFY[c3]: the identity $\lambda_i\,\sigma_i^2/n_i = (1-\lambda_i)\,\delta^2$, i.e. the known-$\theta$ posterior variance equals the $(1-\lambda_i)\delta^2$ term appearing above.

b. Evaluate $\mu_i$ under the marginal densities of the $Y_{ij}$

$$\text{empirical Bayes}\qquad \boldsymbol{Y_i} \sim N\left(\frac{\theta 1_{n_i},\, \sigma_i^2 I_{n_i} + \delta^2 J_{n_i, n_i}}{\hat{\theta} = \bar{y},\, \hat{\delta}^2,\, \sigma_i^2}\right)$$

> ⏳ VERIFY[c4]: under $Y_{ij} = \mu_i + e_{ij}$ with $\mu_i \sim N(\theta,\delta^2)$ and $e_{ij}\overset{iid}{\sim}N(0,\sigma_i^2)$, the marginal mean of $\boldsymbol{Y_i}$ is $\theta 1_{n_i}$ and its covariance is $\sigma_i^2 I_{n_i} + \delta^2 J_{n_i,n_i}$.

**9.2.4 Full Bayesian Analysis**

The main result can be rewritten as

$$\mu \mid \boldsymbol{y}, \sigma^2, \delta^2 \text{has a l-variate normal density}$$

===CLAIMS===

CLAIM c1: Normal–normal posterior mean is the shrinkage estimator with weight λ_i = δ²/(δ² + σ_i²/n_i)
```python
from sympy import *

ybar_i, theta = symbols('ybar_i theta', real=True)
mu = symbols('mu', real=True)

# positive variances
d2, s2, nn = symbols('d2 s2 nn', positive=True)

V = s2/nn  # sampling variance of ybar_i given mu

# work directly with the log posterior kernel (a quadratic in mu):
# log post = -(ybar-mu)^2/(2V) - (mu-theta)^2/(2 d2) + const
logpost = -(ybar_i - mu)**2/(2*V) - (mu - theta)**2/(2*d2)

# posterior mean = argmax of the quadratic -> solve d/dmu = 0 (linear in mu)
mstar = solve(Eq(diff(logpost, mu), 0), mu)[0]

lam = d2/(d2 + s2/nn)
assert simplify(mstar - (lam*ybar_i + (1-lam)*theta)) == 0

# posterior is Gaussian: -logpost = a*mu^2 + ... with variance 1/(2a) = lam * s2/nn
quad = expand(-logpost)
a = quad.coeff(mu, 2)
post_var = simplify(1/(2*a))
assert simplify(post_var - lam*s2/nn) == 0
```

CLAIM c2: ȳ = Σλ_iȳ_i / Σλ_i is exactly the precision-weighted mean under Ȳ_i ~ N(θ, δ²+σ_i²/n_i)
```python
from sympy import *

d2 = symbols('d2', positive=True)
l = 3
s = symbols('s1:4', positive=True)   # sigma_i^2
n = symbols('n1:4', positive=True)
yb = symbols('yb1:4', real=True)
th = symbols('th', real=True)

lam = [d2/(d2 + s[i]/n[i]) for i in range(l)]
prec = [1/(d2 + s[i]/n[i]) for i in range(l)]

# MLE/WLS of theta under Ybar_i ~ N(theta, d2 + s_i/n_i)
obj = sum(prec[i]*(yb[i]-th)**2 for i in range(l))
th_hat = solve(Eq(diff(obj, th), 0), th)[0]

booklet = Sum(0,(th,0,0)).doit() + sum(lam[i]*yb[i] for i in range(l))/sum(lam[i] for i in range(l))

assert simplify(th_hat - booklet) == 0
```

CLAIM c4: Marginal Y_i has mean θ·1 and covariance σ_i² I + δ² J
```python
from sympy import *

d2, s2 = symbols('d2 s2', positive=True)
th = symbols('th', real=True)
n = 4

# Y = mu*1 + e, mu ~ N(th, d2), e ~ N(0, s2 I), independent
one = ones(n, 1)
I = eye(n)
J = ones(n, n)

# Mean
mean = th*one
assert simplify(mean - th*one) == zeros(n, 1)

# Cov(Y) = Var(mu)*1 1' + Cov(e) = d2*J + s2*I
Cov = d2*(one*one.T) + s2*I
target = s2*I + d2*J
assert simplify(Cov - target) == zeros(n, n)

# consistency with c1: implied marginal var of ybar_i = d2 + s2/n
avg = (one.T*Cov*one)[0, 0]/n**2
assert simplify(avg - (d2 + s2/n)) == 0
```
