CLAIM c1: The printed derivatives of the normal log-density and the expectation $E[\partial^2_{(\sigma^2)^2}\Delta] = -1/(2(\sigma^2)^2)$ are correct.
```python
from sympy import *

x, mu = symbols('x mu', real=True)
s2 = symbols('s2', positive=True)

Delta = -Rational(1,2)*log(2*pi) - Rational(1,2)*log(s2) - (x-mu)**2/(2*s2)

# booklet: d/dmu Delta = (x-mu)/s2
assert simplify(diff(Delta, mu) - (x-mu)/s2) == 0
# booklet: d2/dmu2 Delta = -1/s2
assert simplify(diff(Delta, mu, 2) - (-1/s2)) == 0
# booklet: d2/dmu dsigma2 Delta = -(x-mu)/(s2)**2
assert simplify(diff(Delta, mu, 1, s2, 1) - (-(x-mu)/s2**2)) == 0
# booklet: d/dsigma2 Delta = -1/(2 s2) + (x-mu)**2/(2 s2**2)
assert simplify(diff(Delta, s2) - (-1/(2*s2) + (x-mu)**2/(2*s2**2))) == 0
# booklet: d2/d(sigma2)^2 Delta = 1/(2 s2**2) - (x-mu)**2/s2**3
assert simplify(diff(Delta, s2, 2) - (1/(2*s2**2) - (x-mu)**2/s2**3)) == 0

# E[(x-mu)**2] = s2  under X ~ N(mu, s2); E[(x-mu)] = 0
d2s2 = diff(Delta, s2, 2)
Ed2s2 = simplify(d2s2.subs((x-mu)**2, s2).expand().subs((x-mu)**2, s2))
Ed2s2 = simplify(1/(2*s2**2) - s2/s2**3)
assert simplify(Ed2s2 - (-1/(2*s2**2))) == 0

# off-diagonal expectation is 0 since E[x-mu] = 0
assert simplify((-(x-mu)/s2**2).subs(x, mu)) == 0
```

CLAIM c2: $|-H|^{1/2} = \frac{1}{\sqrt2 (\sigma^2)^{3/2}}$, so the booklet's $\frac{1}{(\sigma^2)^{3/2}}$ is correct only up to the constant $1/\sqrt2$ (i.e. proportional, not equal).

s2 = symbols('s2', positive=True)

negH = Matrix([[1/s2, 0], [0, 1/(2*s2**2)]])
det = simplify(negH.det())
assert simplify(det - 1/(2*s2**3)) == 0

root = simplify(sqrt(det))
# exact value
assert simplify(root - 1/(sqrt(2)*s2**Rational(3,2))) == 0

assert ratio == 1/sqrt(2)

CLAIM c3: Fisher information transforms as $I(\phi) = I(\theta)(d\theta/d\phi)^2$, hence $\sqrt{I(\phi)} = \sqrt{I(\theta)}|d\theta/d\phi|$.
```python
from sympy import *

# Verify on a concrete one-to-one reparametrisation with a closed-form Fisher information:
# X | theta ~ Poisson(theta), I(theta) = 1/theta.  Take phi = sqrt(theta) (one-to-one on theta>0).
phi = symbols('phi', positive=True)
theta = symbols('theta', positive=True)
x = symbols('x', nonnegative=True, integer=True)

# I(phi) computed directly from the reparametrised log-likelihood
loglik = x*log(phi**2) - phi**2 - log(factorial(x))
d2 = diff(loglik, phi, 2)
# E[x] = phi**2
I_phi_direct = simplify(-d2.subs(x, phi**2))

# I(phi) via the transformation rule: I(theta)|_{theta=phi^2} * (dtheta/dphi)^2
I_theta = 1/theta
I_phi_rule = simplify((I_theta.subs(theta, phi**2)) * diff(phi**2, phi)**2)

assert simplify(I_phi_direct - I_phi_rule) == 0
assert simplify(sqrt(I_phi_direct) - sqrt(I_theta.subs(theta, phi**2))*Abs(diff(phi**2, phi))) == 0
```

CLAIM c4: Transforming $\pi(\lambda)\propto\lambda^{-1/2}$ by $\lambda=\phi^2$ gives $\pi(\phi)\propto 2$ (constant), matching $\sqrt{I(\phi)}=2$.

from sympy import *

phi = symbols('phi', positive=True)
lam = symbols('lam', positive=True)

# change of variables: pi(phi) = pi(lam=phi^2) * |dlam/dphi|

assert simplify(sqrt(I_phi) - 2) == 0
