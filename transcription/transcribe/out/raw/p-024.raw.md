**3.1 Motivating Monte Carlo Methods (integration)**

$\int_R g(x)f(x)dx = E[g(x)] < \infty$, where $f(x)$ is the density on $R$
$\int g(\theta)\pi(\theta \mid x)d\theta$ is the same thing. *(The booklet writes $x$ with an under-tilde to indicate a vector.)*

If we take $X_1, ..., X_M \overset{iid}{\sim} f(x)$

$\bar{X} = \frac{\sum_{i=1}^{M} X_i}{M} \overset{a.s.}{\longrightarrow} E(X) = \int x f(x)dx < \infty$ – Strong law of large numbers.

So for large $M : \theta = \int x f(x) dx \approx \frac{\sum_{i=1}^{M} X_i}{M}$

So $\hat{\theta} = \frac{\sum_{i=1}^{M} X_i}{M}$ is a simulation consistent estimator of $\theta$.

In the more general problem:

$\frac{\sum_{i=1}^{M} g(X_i)}{M} \overset{a.s.}{\longrightarrow} \int g(x) f(x) dx$

So $\frac{\sum_{i=1}^{M} g(X_i)}{M}$ is a *simulation consistent estimator* of $\int g(x)f(x)dx$. *[hand-drawn star in the left margin marking this line]* *[margin notes, right: "$Y_i = g(X_i)$"; "$\frac{1}{M}\sum_{i=1}^{M} Y_i = \bar{Y}$"]*

CLAIM c1: The posterior P[θ=0|Y1=1] = (2/10)(1/3) / [(2/10)(1/3) + (1/10)(2/3)] equals 1/2, hence equals P(θ=1|Y1=1).

```python
from sympy import Rational, simplify

p_y_given_0 = Rational(2, 10)
p_y_given_1 = Rational(1, 10)
prior_0 = Rational(1, 3)
prior_1 = Rational(2, 3)

assert simplify(prior_0 + prior_1 - 1) == 0

num = p_y_given_0 * prior_0
den = p_y_given_0 * prior_0 + p_y_given_1 * prior_1
post_0 = simplify(num / den)

assert post_0 == Rational(1, 2)

post_1 = simplify(p_y_given_1 * prior_1 / den)
assert post_1 == Rational(1, 2)
assert simplify(post_0 - post_1) == 0
```

CLAIM c4: For X ~ N(mu=25, sigma^2=9), E(X^2) = Var(X) + (E X)^2 = 9 + 625 = 634, matching the exact Gaussian integral.
```python
from sympy import symbols, Rational, sqrt, pi, exp, integrate, oo, simplify, Integer

x = symbols('x', real=True)
mu = Integer(25)
sigma2 = Integer(9)

f = 1 / sqrt(2 * pi * sigma2) * exp(-(x - mu)**2 / (2 * sigma2))

first = simplify(integrate(x * f, (x, -oo, oo)))
assert first == Integer(25)

second = simplify(integrate(x**2 * f, (x, -oo, oo)))
assert second == Integer(634)

assert sigma2 + mu**2 == 634
assert mu**2 == 625
```
