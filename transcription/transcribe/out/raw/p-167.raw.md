*[margin note: "Block (Gibbs) sampler"]*

With many more parameters we can use blocking. That is, $\boldsymbol{\theta}_1$, $\boldsymbol{\theta}_2$, $\boldsymbol{\theta}_3$ can each be several parameters. This will improve mixing very much. *(The booklet writes $\theta$ with under-tildes to indicate vectors.)*

The priors, which we commonly used, will contribute to poor mixing as well. In complicated problems we may use 'shrinkage' or 'Cauchy' priors. These will vary with the parameters and whether the parameter is on the real line or a finite interval.

For $\theta$ in $(-\infty,\infty)$, (as a location-parameter)

$$\pi(\theta) = \frac{1}{\pi(1+\theta^2)},\; -\infty < \theta < \infty,\; (Cauchy)$$

$$\text{or preferable } \pi(\theta) = \frac{1}{b\pi\left(1+\left(\frac{\theta-a}{b}\right)^2\right)},\; -\infty < \theta < \infty$$

> ⏳ VERIFY[c1]: both stated Cauchy densities are proper (integrate to 1 over the real line), the second being the location–scale Cauchy with location $a$ and scale $b$.

where $a$ and $b$ are specified (e.g., Gelman, Jakulin, Pittau, and Su, AAS 2008).

For $\sigma > 0$ (as a scale parameter), one may use

$$\pi(\sigma) = \frac{2}{c\pi\left(1+\frac{\sigma^2}{c}\right)},\; \sigma > 0,\; (half\ Cauchy)$$

where $c$ is to be specified. [Note that the prior $\pi(\sigma) \propto \frac{1}{\sigma}$ is no good (Gelman, BA 2006)]. If we take $\phi = \sigma^2$ in $\pi(\sigma) = \frac{2}{c\pi\left(1+\frac{\sigma^2}{c}\right)}$, $\sigma > 0$, we get

$$\pi(\phi) = \frac{1}{c\pi\sqrt{\phi}\left(1+\frac{\phi}{c}\right)},\; \phi > 0$$

> ⏳ VERIFY[c2]: with $\phi=\sigma^2$, the Jacobian transformation of $\pi(\sigma)=\frac{2}{c\pi(1+\sigma^2/c)}$ yields $\pi(\phi)=\frac{1}{c\pi\sqrt{\phi}(1+\phi/c)}$.

That is,

$$\pi(\sigma^2) = \frac{1}{c\pi\sqrt{\sigma^2}\left(1+\frac{\sigma^2}{c}\right)},\; \sigma^2 > 0$$

Shrinkage priors are also used. The idea is as follows.

$$y\mid p \sim Ber(p),\; p \sim Beta(\mu\tau, (1-\mu)\tau)$$

So,

$$p\mid y \sim Beta(y+\mu\tau, (1-y)+(1-\mu)\tau)$$

> ⏳ VERIFY[c3]: the Bernoulli–Beta conjugate update gives posterior $Beta(y+\mu\tau,\,(1-y)+(1-\mu)\tau)$.

and

$$E(p\mid y) = \lambda y + (1-\lambda)\mu,\; where\; \lambda = \frac{1}{1+\tau}$$

> ⏳ VERIFY[c4]: the posterior mean equals $\lambda y + (1-\lambda)\mu$ with $\lambda = 1/(1+\tau)$.

and $\lambda$ is called a shrinkage parameter. So, taking $\lambda \sim U(0,1)$ we have

$$\pi(\tau) = \frac{1}{(1+\tau)^2},\; \tau > 0$$

> ⏳ VERIFY[c5]: if $\lambda = 1/(1+\tau) \sim U(0,1)$ then the induced density of $\tau$ on $(0,\infty)$ is $1/(1+\tau)^2$, and it is proper.

CLAIM c1: Both stated Cauchy densities integrate to 1 over $(-\infty,\infty)$.
```python
from sympy import *

theta, a = symbols('theta a', real=True)
b = symbols('b', positive=True)

# standard Cauchy as printed
std = 1/(pi*(1 + theta**2))
assert integrate(std, (theta, -oo, oo)) == 1

# "or preferable" location-scale Cauchy as printed
ls = 1/(b*pi*(1 + ((theta - a)/b)**2))
assert simplify(integrate(ls, (theta, -oo, oo))) == 1
```

CLAIM c2: With $\phi=\sigma^2$, the change of variables on the half-Cauchy $\pi(\sigma)=\frac{2}{c\pi(1+\sigma^2/c)}$ gives $\pi(\phi)=\frac{1}{c\pi\sqrt{\phi}(1+\phi/c)}$.
```python
from sympy import *

sigma, phi, c = symbols('sigma phi c', positive=True)

pi_sigma = 2/(c*pi*(1 + sigma**2/c))

# phi = sigma^2  =>  sigma = sqrt(phi),  |d sigma / d phi| = 1/(2 sqrt(phi))
sigma_of_phi = sqrt(phi)
jac = diff(sigma_of_phi, phi)
assert simplify(jac - 1/(2*sqrt(phi))) == 0

pi_phi_derived = pi_sigma.subs(sigma, sigma_of_phi) * jac
pi_phi_booklet = 1/(c*pi*sqrt(phi)*(1 + phi/c))
assert simplify(pi_phi_derived - pi_phi_booklet) == 0
```

CLAIM c3: For $y\mid p\sim Ber(p)$ and $p\sim Beta(\mu\tau,(1-\mu)\tau)$, the posterior is $Beta(y+\mu\tau,(1-y)+(1-\mu)\tau)$.
```python
from sympy import *

p = symbols('p', positive=True)
mu = symbols('mu', positive=True)
tau = symbols('tau', positive=True)

# check the kernel identity for both possible data values y in {0,1}
for y in (0, 1):
    lik = p**y * (1 - p)**(1 - y)
    prior_kernel = p**(mu*tau - 1) * (1 - p)**((1 - mu)*tau - 1)
    post_kernel = expand(powsimp(lik * prior_kernel, force=True))

    a_post = y + mu*tau
    b_post = (1 - y) + (1 - mu)*tau
    booklet_kernel = p**(a_post - 1) * (1 - p)**(b_post - 1)

    assert simplify(post_kernel - booklet_kernel) == 0

# and the posterior is a proper Beta: normalizing constant is finite/positive
y = symbols('y')
a_post = y + mu*tau
b_post = (1 - y) + (1 - mu)*tau
assert simplify(a_post + b_post - (1 + tau)) == 0   # total mass parameter
```

CLAIM c4: The posterior mean is $E(p\mid y)=\lambda y+(1-\lambda)\mu$ with $\lambda=1/(1+\tau)$.
```python
from sympy import *

y = symbols('y', real=True)
mu, tau = symbols('mu tau', positive=True)

a_post = y + mu*tau
b_post = (1 - y) + (1 - mu)*tau

post_mean = a_post/(a_post + b_post)          # Beta mean = a/(a+b)
lam = 1/(1 + tau)
booklet = lam*y + (1 - lam)*mu

assert simplify(post_mean - booklet) == 0
```

CLAIM c5: If $\lambda=1/(1+\tau)\sim U(0,1)$, the induced density of $\tau$ is $\pi(\tau)=1/(1+\tau)^2$ on $\tau>0$, and it integrates to 1.
```python
from sympy import *

tau = symbols('tau', positive=True)

lam = 1/(1 + tau)
# U(0,1) density is 1 on (0,1); tau > 0 maps onto lam in (0,1), monotone decreasing
jac = Abs(diff(lam, tau))
pi_tau_derived = simplify(1 * jac)

pi_tau_booklet = 1/(1 + tau)**2
assert simplify(pi_tau_derived - pi_tau_booklet) == 0

# properness
assert integrate(pi_tau_booklet, (tau, 0, oo)) == 1
```
