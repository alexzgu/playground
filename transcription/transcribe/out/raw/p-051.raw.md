**5.5 Numerical Integrations**

Gaussian quadrature
Orthogonal polynomial

$$\int h(x)f(x)dx \approx \sum_{k=1}^{n} w_k f(x_k)$$

$$w_1, ..., w_n \text{ are weights}$$

$$x_1, ..., x_n \text{ are roots, where } n \text{ is the number of roots.}$$

A specific type of integral gives rise to a specific orthogonal polynomial.
Typically choose: $n = 25, 50, 75, 100$.
1) Legendre polynomial

$$\int_a^b f(x)dx \approx \sum_{k=1}^{n} w_k f(x_k)$$

> ⏳ VERIFY[c1]: the Gauss rule for weight $h(x)=1$ on $[a,b]$ (Legendre) — nodes at the roots of the monic orthogonal polynomial, weights fixed by exactness on degrees $0..n-1$ — is exact for all polynomials of degree $\le 2n-1$.

2) Lagnere [sic: Laguerre] polynomial

$$\int_0^\infty x^\alpha e^{-x} f(x)dx \approx \sum_{k=1}^{n} w_k f(x_k)$$

> ⏳ VERIFY[c2]: the Gauss rule for weight $h(x)=x^{\alpha}e^{-x}$ on $[0,\infty)$ (generalized Laguerre) is exact for all polynomials of degree $\le 2n-1$.

3) Hermite polynomial

$$\int_{-\infty}^{\infty} e^{-x^2} f(x)dx \approx \sum_{k=1}^{n} w_k f(x_k)$$

> ⏳ VERIFY[c3]: the Gauss rule for weight $h(x)=e^{-x^2}$ on $(-\infty,\infty)$ (Hermite) is exact for all polynomials of degree $\le 2n-1$.

4) Jacobi polynomial

$$\int_{-1}^{1} (1-x)^\alpha (1+x)^\beta f(x)dx \approx \sum_{k=1}^{n} w_k f(x_k)$$

> ⏳ VERIFY[c4]: the Gauss rule for weight $h(x)=(1-x)^{\alpha}(1+x)^{\beta}$ on $[-1,1]$ (Jacobi) is exact for all polynomials of degree $\le 2n-1$.

Better approximations can be obtained using adaptive quadrature.
Double integrals can be obtained:

$$\int \int f(x,y)dxdy = \sum_{k_2=1}^{n_2} w_{k_2} \sum_{k_1=1}^{n_1} w_{k_1} f(x_{k_1}, x_{k_2})$$

*(The booklet writes `=` rather than `\approx` here, unlike the four rules above; and both arguments of $f$ are written as $x$'s — $f(x_{k_1}, x_{k_2})$ — where the second node set belongs to the $y$ variable.)*

> ⏳ VERIFY[c5]: the tensor-product rule $\sum_{k_2} w_{k_2} \sum_{k_1} w_{k_1} f(x_{k_1}, y_{k_2})$ reproduces $\int\int f(x,y)\,dx\,dy$ exactly when $f$ is a polynomial of degree $\le 2n_i-1$ in each variable separately.

CLAIM c1: Gauss-Legendre (weight 1 on [a,b]), n=2, is exact for polynomials of degree ≤ 3.
```python
from sympy import *

x = symbols('x')

def gauss_nodes_weights(moments, n):
    """Nodes = roots of the monic orthogonal polynomial built from the moments.
    Weights fixed by exactness on degrees 0..n-1 ONLY (so exactness on
    degrees n..2n-1 is a real assertion, not an assumption)."""
    c = symbols('c0:%d' % n)
    p = x**n + sum(c[i]*x**i for i in range(n))
    # <p, x^k> = 0 for k = 0..n-1, using the moment sequence
    eqs = [moments[n+k] + sum(c[i]*moments[i+k] for i in range(n)) for k in range(n)]
    csol = solve(eqs, list(c), dict=True)[0]
    p = p.subs(csol)
    nodes = [simplify(r) for r in solve(Poly(p, x), x)]
    assert len(nodes) == n
    w = symbols('w0:%d' % n)
    weqs = [sum(w[k]*nodes[k]**j for k in range(n)) - moments[j] for j in range(n)]
    wsol = solve(weqs, list(w), dict=True)[0]
    weights = [simplify(wsol[wi]) for wi in w]
    return nodes, weights

n = 2
a, b = Integer(-1), Integer(3)          # concrete [a,b]
mom = [Rational(b**(j+1) - a**(j+1), j+1) for j in range(2*n)]
nodes, weights = gauss_nodes_weights(mom, n)

# assertion: exact on degrees n..2n-1 (i.e. 2 and 3), which were never imposed
for j in range(n, 2*n):
    resid = simplify(sum(weights[k]*nodes[k]**j for k in range(n)) - mom[j])
    assert resid == 0, (j, resid)

# and hence on every polynomial of degree <= 2n-1
f = 5 - 2*x + 7*x**2 - 3*x**3
lhs = integrate(f, (x, a, b))
rhs = simplify(sum(weights[k]*f.subs(x, nodes[k]) for k in range(n)))
assert simplify(lhs - rhs) == 0

# sanity: the rule is NOT exact at degree 2n = 4 (the bound is sharp)
assert simplify(sum(weights[k]*nodes[k]**4 for k in range(n)) - Rational(b**5 - a**5, 5)) != 0
```

CLAIM c5: the double-integral tensor-product rule is exact for f polynomial of degree ≤ 2n_i-1 in each variable.
```python
from sympy import *

x, y = symbols('x y')

def gauss_nodes_weights(moments, n, v):
    c = symbols('c0:%d' % n)
    p = v**n + sum(c[i]*v**i for i in range(n))
    eqs = [moments[n+k] + sum(c[i]*moments[i+k] for i in range(n)) for k in range(n)]
    csol = solve(eqs, list(c), dict=True)[0]
    p = p.subs(csol)
    nodes = [simplify(r) for r in solve(Poly(p, v), v)]
    w = symbols('w0:%d' % n)
    weqs = [sum(w[k]*nodes[k]**j for k in range(n)) - moments[j] for j in range(n)]
    wsol = solve(weqs, list(w), dict=True)[0]
    return nodes, [simplify(wsol[wi]) for wi in w]

# n1 = 2 nodes in x on [0,1];  n2 = 2 nodes in y on [0,1]
n1 = n2 = 2
mom1 = [Rational(1, j+1) for j in range(2*n1)]
mom2 = [Rational(1, j+1) for j in range(2*n2)]
xs, wx = gauss_nodes_weights(mom1, n1, x)
ys, wy = gauss_nodes_weights(mom2, n2, y)

# degree <= 3 in x and <= 3 in y separately (a full tensor-degree test)
f = (1 + 2*x - x**2 + 3*x**3) * (2 - y + 4*y**2 + y**3) + 5*x**3*y**3 - x*y**2

lhs = integrate(integrate(f, (x, 0, 1)), (y, 0, 1))
rhs = sum(wy[k2] * sum(wx[k1] * f.subs({x: xs[k1], y: ys[k2]}) for k1 in range(n1))
          for k2 in range(n2))
assert simplify(lhs - rhs) == 0

# sharp: a degree-4 term in x breaks it
g = x**4
lhs_g = integrate(integrate(g, (x, 0, 1)), (y, 0, 1))
rhs_g = sum(wy[k2] * sum(wx[k1] * g.subs({x: xs[k1], y: ys[k2]}) for k1 in range(n1))
            for k2 in range(n2))
assert simplify(lhs_g - rhs_g) != 0
```
