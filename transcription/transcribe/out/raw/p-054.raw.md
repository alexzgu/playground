**5.9 Monte Carlo Integration**

Consider the following problem:

$$E[h(x)] = \int_{-\infty}^{\infty} h(x) f(x)\, dx$$

Where $f(x)$ is a density function. We assume that $E[h(x)]$ is finite, otherwise the computation is meaningless.

Suppose we have a large random sample of size $M$ from $f(x)$. That is

$$X_1, ..., X_M \;\underset{\sim}{iid}\; f(x).$$

*(The booklet sets the "$\sim$" underneath "iid" here rather than as $\overset{iid}{\sim}$; transcribed as printed.)*

Then, a Monte Carlo estimator of $E[h(x)]$ is

$$\hat{h}_M = M^{-1} \sum_{i=1}^{M} h(x_i)$$

Observe that by the SLLN,

$$\hat{h}_M \overset{a.s.}{\longrightarrow} E[h(x)]$$

Thus, if $M$ is chosen large enough,

$$E[h(x)] \approx \hat{h}_M$$

Now, if $E[h^2(x)] < \infty,$ then

$$V_M = Var(\hat{h}_M) = \frac{1}{M} \int_{-\infty}^{\infty} [h(x) - E[h(x)]]^2 f(x)\, dx$$

> ⏳ VERIFY[c1]: for an iid sample, $Var(\hat{h}_M) = \frac{1}{M}\int [h(x)-E[h(x)]]^2 f(x)\,dx$, i.e. $Var(h(X))/M$.

Thus, an estimator of the variance $V_M$ is $\dfrac{s_M^2}{M}$, where $s_M^2 = \displaystyle\sum_{i=1}^{M} \frac{[h(x_i) - \hat{h}_M]^2}{M-1}$

For large$M$, using the CLT, *[sic: no space between "large" and "$M$"]*

$$\frac{\hat{h}_M - E[h(x)]}{s/\sqrt{M}} \overset{d}{\to} \text{Normal}(0,1) \;\text{ (see later).}$$

> ⏳ VERIFY[c2]: the standardization is correct — if $Var(\hat h_M)=\sigma^2/M$ with $\sigma^2 = Var(h(X))$, then $(\hat h_M - E[h(x)])\big/(\sigma/\sqrt{M})$ has mean 0 and variance 1.

Note that $s/\sqrt{M}$ is the Monte Carlo error in $\hat{h}_M$. An approximate 95% confidence interval for $E[h(x)]$ is

$$\hat{h}_M \pm 2s/\sqrt{M}$$

> ⏳ VERIFY[c3]: $\pm 2$ standard errors gives approximately 95% coverage under the normal limit, i.e. $2\Phi(2)-1 = 0.95$ to two decimal places.

Example

$$x|\theta \;\sim\; N(\theta, 1), \theta \;\sim\; Cauchy(0,1)$$

===CLAIMS===

CLAIM c1: For an iid sample $X_1,\dots,X_M\sim f$, $Var(M^{-1}\sum h(X_i)) = \frac{1}{M}\int_{-\infty}^{\infty}[h(x)-E[h(x)]]^2 f(x)\,dx$.
```python
from sympy import *

# General algebra: iid => Var(mean of M terms) = Var(h(X))/M
M, sigma2 = symbols('M sigma2', positive=True)
var_mean = M * sigma2 / M**2          # variances of iid terms add
assert simplify(var_mean - sigma2 / M) == 0

# Concrete decisive instance: f = Exponential(1), h(x) = x**2, M = 3
x = symbols('x', positive=True)
f = exp(-x)
h = x**2

Eh  = integrate(h * f, (x, 0, oo))
Eh2 = integrate(h**2 * f, (x, 0, oo))
var_h = simplify(Eh2 - Eh**2)

# booklet's RHS with M = 3
rhs = Rational(1, 3) * integrate((h - Eh)**2 * f, (x, 0, oo))

# direct variance of the average of 3 iid copies
lhs = var_h / 3

assert simplify(lhs - rhs) == 0
assert simplify(rhs - Rational(1, 3) * var_h) == 0
assert var_h == 20          # E[X^4]-E[X^2]^2 = 24 - 4
```

CLAIM c2: If $Var(\hat h_M) = \sigma^2/M$ where $\sigma^2 = Var(h(X))$, then $(\hat h_M - E[h(x)])/(\sigma/\sqrt M)$ has mean 0 and variance 1.
```python
from sympy import *

M, sigma = symbols('M sigma', positive=True)
mu = symbols('mu', real=True)

# hat h_M has mean mu and variance sigma**2 / M
mean_hat = mu
var_hat = sigma**2 / M

# standardized quantity Z = (hat h_M - mu) / (sigma / sqrt(M))
c = 1 / (sigma / sqrt(M))            # scaling constant
mean_Z = c * (mean_hat - mu)
var_Z = c**2 * var_hat

assert simplify(mean_Z) == 0
assert simplify(var_Z - 1) == 0
```

CLAIM c3: $\hat h_M \pm 2s/\sqrt M$ is an approximate 95% interval, i.e. the normal coverage of $\pm 2$ standard errors equals 0.95 to two decimal places.
```python
from sympy import *

z = Integer(2)
# coverage = 2*Phi(2) - 1 = erf(2/sqrt(2))
coverage = erf(z / sqrt(2))
assert simplify(coverage - (2 * (Rational(1,2) * (1 + erf(z / sqrt(2))))- 1)) == 0

val = N(coverage, 20)
# rounds to 0.95 at two decimal places
assert Rational(round(val * 100), 100) == Rational(95, 100)
# and it is strictly between 0.94 and 0.96
assert val > Rational(94, 100) and val < Rational(96, 100)
```
