**12.2 SIR algorithm**

$\pi(\theta\mid x)$: in a multiparameter problem, it may be difficult to draw samples from $\pi(\theta\mid x)$.

*(The booklet writes $\theta$ and $x$ with under-tildes to indicate vectors.)*

Approximate $\pi(\theta\mid x)$ by $\pi_a(\theta\mid x)$, where it is easy to draw samples from $\pi_a(\theta\mid x)$.

$$\pi(\theta_1,\theta_2\mid x) = \pi(\theta_2\mid \theta_1, x)\,\pi(\theta_1\mid x)$$

$$\pi_a(\theta_1,\theta_2\mid x) = \pi_a(\theta_2\mid \theta_1, x)\,\pi(\theta_1\mid x)$$

> ⏳ VERIFY[c1]: the chain-rule factorization $\pi(\theta_1,\theta_2\mid x)=\pi(\theta_2\mid\theta_1,x)\pi(\theta_1\mid x)$ holds for a joint density.

$$\frac{\pi(\theta\mid x)}{\pi_a(\theta\mid x)} \le H < \infty$$

$$\pi_a(\theta\mid x) = \begin{cases} \text{importance function or} \\ \text{proporal [sic] density or} \\ \text{candidate generating density} \end{cases}$$

===CLAIMS===

CLAIM c1: The chain rule $\pi(\theta_1,\theta_2\mid x)=\pi(\theta_2\mid\theta_1,x)\,\pi(\theta_1\mid x)$ holds — verified on an exact bivariate example where the conditional and marginal are computed from the joint by definition.
```python
from sympy import *

t1, t2 = symbols('theta_1 theta_2', positive=True)

# Exact joint density on (0,1)x(0,1) (x suppressed: everything is conditional on x).
# The unnormalized kernel integrates to 7/4, so divide by 7/4 to get a true density.
joint = (4*t1*t2 + Rational(1, 2)*(t1 + t2) + Rational(1, 4)) / Rational(7, 4)

# It is a valid density on the unit square.
assert integrate(integrate(joint, (t2, 0, 1)), (t1, 0, 1)) == 1
assert simplify(joint.subs({t1: Rational(1,3), t2: Rational(2,5)})) > 0

# Marginal of theta_1, and conditional of theta_2 given theta_1, by definition.
marg1 = integrate(joint, (t2, 0, 1))
cond21 = simplify(joint / marg1)

# The conditional integrates to 1 for every theta_1 in (0,1).
assert simplify(integrate(cond21, (t2, 0, 1)) - 1) == 0

# Chain rule: joint == conditional * marginal.
assert simplify(cond21 * marg1 - joint) == 0
```

CLAIM c2: The normalized SIR weights $w_h = \frac{\pi(\theta_h\mid x)/\pi_a(\theta_h\mid x)}{\sum_{k=1}^{M}\pi(\theta_k\mid x)/\pi_a(\theta_k\mid x)}$ are proportional to the importance ratios $\pi(\theta_h\mid x)/\pi_a(\theta_h\mid x)$ and sum to one.
```python
from sympy import *

M = 5
h = symbols('h')

# Symbolic positive importance ratios r_h = pi(theta_h|x) / pi_a(theta_h|x).
r = symbols('r1:6', positive=True)

denom = Add(*r)
w = [r[k] / denom for k in range(M)]

# The weights sum to exactly 1.
assert simplify(Add(*w) - 1) == 0

# w_h is proportional to r_h: the ratio w_h / r_h is the same constant for all h.
const = simplify(w[0] / r[0])
for k in range(M):
    assert simplify(w[k] / r[k] - const) == 0
assert simplify(const - 1/denom) == 0

# Consistency with the stated bound pi/pi_a <= H < infinity: ratios bounded by H
# forces every weight to be at least 1/(M*H) when ratios are positive.
H = symbols('H', positive=True)
vals = {r[k]: H for k in range(M)}
assert simplify(w[0].subs(vals) - Rational(1, M)) == 0

# Numeric spot check against the printed formula.
ratios = [Rational(1, 2), Rational(3, 2), Rational(1, 4), Rational(2, 1), Rational(3, 4)]
s = sum(ratios)
wn = [x / s for x in ratios]
assert sum(wn) == 1
assert wn[3] == Rational(2, 1) / s == Rational(8, 20)
```
