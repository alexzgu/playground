Note: This is not a standard generalized linear model. First two stages in the model are conjugate; inference about the $\lambda_i$ is straight forward.

*[the words "not a standard" are struck through/underlined by hand, emphasizing the negation]*

Joint posterior density

$$\pi(\lambda,\alpha,\beta\mid d) \propto \prod_{i=1}^{\ell}\left\{\frac{(n\lambda_i)^{d_i}e^{-n_i\lambda_i}}{d_i!}\;\frac{(\alpha e^{-x_i'\beta})^{\alpha}\lambda_i^{\alpha-1}e^{-\lambda_i\alpha e^{-x_i'\beta}}}{\Gamma(\alpha)}\right\}\times\frac{1}{(1+\alpha)^2}$$

*(The booklet writes $\lambda$, $\beta$, $d$, $x_i$, $\nu$ with under-tildes to indicate vectors.)*

Fitting the model(Metropolis-Hastings sampler)

$$\lambda_i\mid\alpha,\beta,d \overset{ind}{\sim} \text{Gamma}(d_i+\alpha,\; n_i+\alpha e^{-x_i'\beta})$$

*[handwritten mark over the "$\overset{ind}{\sim}$" symbol, reading "$\gamma^{*}$"(?), with a hand-drawn curl looping back to the conditioning list; apparently emphasizing the conditional independence of the $\lambda_i$]*

> ⏳ VERIFY[c1]: the $\lambda_i$-kernel of the joint posterior is exactly the Gamma$(d_i+\alpha,\;n_i+\alpha e^{-x_i'\beta})$ kernel.

$$\pi(\alpha,\beta\mid\lambda,d) \propto \left[\prod_{i=1}^{\ell}\left\{\frac{(\alpha e^{-x_i'\beta})^{\alpha}\lambda_i^{\alpha-1}e^{-\lambda_i\alpha e^{-x_i'\beta}}}{\Gamma(\alpha)}\right\}\right]\times\frac{1}{(1+\alpha)^2}$$

Construction of proposal density $\pi_a(\alpha,\beta\mid\lambda,d)$.

$$\Delta(\alpha,\beta) = \ell\alpha log\alpha - \ell log\Gamma(\alpha) - 2log(1+\alpha) - \alpha\Big\{\sum_{i=1}^{\ell}x_i'\Big\}\beta + (\alpha-1)\sum_{i=1}^{\ell}log\lambda_i - \alpha\sum_{i=1}^{\ell}\lambda_i e^{-x_i'\beta}$$

> ⏳ VERIFY[c2]: $\Delta(\alpha,\beta)$ equals $\log$ of the displayed $\pi(\alpha,\beta\mid\lambda,d)$ kernel.

Two steps:

a. Optimize $\Delta(\alpha,\beta)$ (Nelder-Mead algorithm); mode is as at $(\hat\alpha,\hat\beta)$.

b. Obtain the Hessian matrix at $(\hat\alpha,\hat\beta)$, denoted by $H$.

Approximate posterior density

$$\begin{bmatrix}\alpha\\\beta\end{bmatrix}\Bigg|\lambda,d \sim Normal\left\{\begin{bmatrix}\alpha\\\beta\end{bmatrix},\begin{bmatrix}\sigma^2 & \nu'\\ \nu & \Delta\end{bmatrix}\right\}$$

where

$$\begin{bmatrix}\sigma^2 & \nu'\\ \nu & \Delta\end{bmatrix} = k(-H^{-1})$$

with $k$ a tuning constant.

$$\pi_a(\alpha,\beta\mid\lambda,d) = g_a(\alpha\mid\beta,\lambda,d)\,h_a(\beta\mid\lambda,d)$$

where $g_a(\alpha\mid\beta,\lambda,d)$ and $h_a(\beta\mid\lambda,d)$ are to be determined.

$$\beta\mid\lambda,d \sim \text{Normal}\{\hat\beta,\Delta\} : h_a(\beta\mid\lambda,d)$$

$$\alpha\mid\beta,\lambda,d \sim \text{Normal}\{\hat\alpha+\nu'\Delta^{-1}(\beta-\hat\beta),\;\sigma^2-\nu'\Delta^{-1}\nu\}$$

> ⏳ VERIFY[c3]: for a joint Normal with mean $(\hat\alpha,\hat\beta)$ and covariance $\begin{bmatrix}\sigma^2 & \nu'\\ \nu & \Delta\end{bmatrix}$, the conditional of $\alpha$ given $\beta$ has mean $\hat\alpha+\nu'\Delta^{-1}(\beta-\hat\beta)$ and variance $\sigma^2-\nu'\Delta^{-1}\nu$.

CLAIM c1: The $\lambda_i$-dependent factors of the joint posterior form the kernel of Gamma$(d_i+\alpha,\ n_i+\alpha e^{-x_i'\beta})$ (shape/rate).
```python
from sympy import *

lam, n_i, d_i, alpha, eta = symbols('lam n_i d_i alpha eta', positive=True)
# eta stands for exp(-x_i' beta) > 0

# lambda_i factors from the joint posterior kernel
joint_kernel = lam**d_i * exp(-n_i*lam) * lam**(alpha-1) * exp(-lam*alpha*eta)

# Gamma(shape = d_i + alpha, rate = n_i + alpha*eta) kernel
shape = d_i + alpha
rate = n_i + alpha*eta
gamma_kernel = lam**(shape-1) * exp(-rate*lam)

assert simplify(joint_kernel - gamma_kernel) == 0

# and the normalized Gamma density integrates to 1 with these parameters
dens = rate**shape / gamma(shape) * lam**(shape-1) * exp(-rate*lam)
val = integrate(dens.subs({d_i: 2, alpha: 3, n_i: Rational(1,2), eta: Rational(1,3)}),
                (lam, 0, oo))
assert simplify(val - 1) == 0
```

CLAIM c2: $\Delta(\alpha,\beta) = \log\pi(\alpha,\beta\mid\lambda,d)$ kernel, i.e. the log of $\prod_i (\alpha e^{-x_i'\beta})^\alpha\lambda_i^{\alpha-1}e^{-\lambda_i\alpha e^{-x_i'\beta}}/\Gamma(\alpha)\times(1+\alpha)^{-2}$.
```python
from sympy import *

alpha = symbols('alpha', positive=True)
b = symbols('b', real=True)          # scalar beta
ell = 3
x = [Rational(1,2), Rational(-1,3), Rational(2,5)]   # x_i' (scalars)
lam = [Rational(3,2), Rational(1,4), Rational(7,3)]  # lambda_i > 0

# log of the stated kernel, term by term
logkernel = -2*log(1+alpha)
for i in range(ell):
    eta = exp(-x[i]*b)
    logkernel += alpha*log(alpha*eta) + (alpha-1)*log(lam[i]) - lam[i]*alpha*eta - log(gamma(alpha))

# Delta(alpha, beta) as printed on the page
Delta = (ell*alpha*log(alpha) - ell*log(gamma(alpha)) - 2*log(1+alpha)
         - alpha*sum(x)*b
         + (alpha-1)*sum(log(l) for l in lam)
         - alpha*sum(lam[i]*exp(-x[i]*b) for i in range(ell)))

assert simplify(expand_log(logkernel - Delta, force=True)) == 0
```

CLAIM c3: Partitioned-Normal conditional: $\alpha\mid\beta \sim N(\hat\alpha+\nu'\Delta^{-1}(\beta-\hat\beta),\ \sigma^2-\nu'\Delta^{-1}\nu)$.
```python
from sympy import *

# beta is 2-dimensional; joint covariance [[sigma^2, nu'], [nu, Delta]]
s2 = Rational(5, 2)
nu = Matrix([Rational(1, 2), Rational(-1, 3)])
Dl = Matrix([[2, Rational(1, 4)], [Rational(1, 4), 3]])
Sigma = Matrix(3, 3, lambda i, j: 0)
Sigma[0, 0] = s2
Sigma[0, 1:3] = nu.T
Sigma[1:3, 0] = nu
Sigma[1:3, 1:3] = Dl
assert Sigma.is_positive_definite

ah = Rational(1, 5)
bh = Matrix([Rational(2, 7), Rational(-1, 2)])
a, b1, b2 = symbols('a b1 b2', real=True)
bvec = Matrix([b1, b2])

# conditional density from the joint: exponent of N_3, completed in `a`
z = Matrix([a - ah, b1 - bh[0], b2 - bh[1]])
P = Sigma.inv()
q = expand((z.T * P * z)[0, 0])

# coefficient of a^2 gives 1/var; complete the square for the mean
c2 = Poly(q, a).coeff_monomial(a**2)
c1 = Poly(q, a).coeff_monomial(a)
var_cond = simplify(1 / c2)
mean_cond = simplify(-c1 / (2 * c2))

var_claim = simplify(s2 - (nu.T * Dl.inv() * nu)[0, 0])
mean_claim = simplify(ah + (nu.T * Dl.inv() * (bvec - bh))[0, 0])

assert simplify(var_cond - var_claim) == 0
assert simplify(expand(mean_cond - mean_claim)) == 0
```
