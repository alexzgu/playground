**Digression – Sufficient Statistics**

$X_1, \dots, X_n \mid \theta \;\underline{iid}\; f(x \mid \theta)$

$T(\underset{\sim}{X})$ is a sufficient statistic.

$f(\underset{\sim}{x} \mid T(\underset{\sim}{x}), \theta)$ does not depend on $\theta$

Factorization Theorem:

If $f(\underset{\sim}{x} \mid \theta) = g(T(\underset{\sim}{x}) \mid \theta) h(\underset{\sim}{x})$, where $g(T(\underset{\sim}{x}) \mid \theta)$ depends on $T(\underset{\sim}{x})$ and $\theta$, $h(\underset{\sim}{x})$ does not depend on $\theta$.

Then $T(\underset{\sim}{x})$ is a sufficient statistic. (True both ways.)

**Example**

$X_1, \dots, X_n \mid \mu \;\underline{iid}\; N(\mu, \sigma^2)$, $\sigma^2$ is known.

$$
\begin{aligned}
f(\underset{\sim}{x} \mid \mu) &= \left(\frac{1}{2\pi\sigma^2}\right)^{\frac{n}{2}} exp\left\{ -\frac{1}{2\sigma^2} \sum_{i=1}^{n} (x_i - \mu)^2 \right\} \\
&= \left(\frac{1}{2\pi\sigma^2}\right)^{\frac{n}{2}} exp\left\{ -\frac{1}{2\sigma^2}\left[ (n-1)s^2 + n(\bar{x} - \mu)^2 \right] \right\} \\
&= e^{-\frac{n}{2\sigma^2}(\bar{x} - \mu)^2} \left(\frac{1}{2\pi\sigma^2}\right)^{\frac{n}{2}} e^{-\frac{(n-1)s^2}{2\sigma^2}}
\end{aligned}
$$

> ⏳ VERIFY[c4]: $\sum_{i=1}^{n}(x_i-\mu)^2 = (n-1)s^2 + n(\bar{x}-\mu)^2$, where $s^2 = \frac{1}{n-1}\sum_i (x_i-\bar{x})^2$.

> ⏳ VERIFY[c5]: The third line equals the second, i.e. the Gaussian likelihood factors as $g(\bar{x},\mu)\,h(x)$ with the stated $g$ and $h$.

Here $g(\bar{x}, \mu) = e^{-\frac{n}{2\sigma^2}(\bar{x}-\mu)^2}$ and $h(x) = \left(\frac{1}{2\pi\sigma^2}\right)^{\frac{n}{2}} e^{-\frac{(n-1)s^2}{2\sigma^2}}$.

So $\bar{X}$ is a sufficient statistic.

If both $(\mu, \sigma^2)$ are unknown.

$$f(\underset{\sim}{x} \mid \mu, \sigma^2) = f(\underset{\sim}{x} \mid \mu) = g(\bar{x}, s^2, \mu, \sigma^2) h(x)$$

So $(\bar{X}, s^2)$ are joint sufficient for $(\mu, \sigma^2)$.

**3.2.1 How to construct conjugate priors?**

Idea: Look at the likelihood function as a distribution of the parameters.

$$\pi(\theta \mid \underset{\sim}{x}) = \frac{L(\theta \mid \underset{\sim}{x})}{\int L(\theta \mid \underset{\sim}{x}) d\theta} \;\text{— —fiducial inference.}$$

CLAIM c1: Beta(α,β) prior with n₁ Bernoulli trials and s₁ successes yields a Beta(s₁+α, n₁−s₁+β) posterior.
```python
from sympy import *

th = Symbol('theta', positive=True)
# exact rational parameters
alpha, beta_, n1, s1 = Rational(2), Rational(3), 5, 2
B = lambda a, b: gamma(a)*gamma(b)/gamma(a + b)   # Beta function, evaluates exactly

prior = th**(alpha-1) * (1-th)**(beta_-1) / B(alpha, beta_)
lik = th**s1 * (1-th)**(n1-s1)          # Bernoulli likelihood kernel
marg = integrate(prior*lik, (th, 0, 1))
post = simplify(prior*lik/marg)

a_post, b_post = s1 + alpha, n1 - s1 + beta_
claimed = th**(a_post-1) * (1-th)**(b_post-1) / B(a_post, b_post)

assert simplify(post - claimed) == 0
assert simplify(integrate(claimed, (th, 0, 1)) - 1) == 0
```

CLAIM c2: Sequentially updating the stage-one posterior with a second batch (n₂ trials, s₂ successes) gives Beta(s₁+s₂+α, n₁−s₁+n₂−s₂+β).
```python
from sympy import *

th = Symbol('theta', positive=True)
alpha, beta_ = Rational(2), Rational(3)
n1, s1, n2, s2 = 5, 2, 4, 3
B = lambda a, b: gamma(a)*gamma(b)/gamma(a + b)   # Beta function, evaluates exactly

# stage-one posterior acts as the stage-two prior
a1, b1 = s1 + alpha, n1 - s1 + beta_
prior2 = th**(a1-1) * (1-th)**(b1-1) / B(a1, b1)
lik2 = th**s2 * (1-th)**(n2-s2)
post2 = simplify(prior2*lik2 / integrate(prior2*lik2, (th, 0, 1)))

a2, b2 = s1 + s2 + alpha, n1 - s1 + n2 - s2 + beta_
claimed = th**(a2-1) * (1-th)**(b2-1) / B(a2, b2)

assert simplify(post2 - claimed) == 0
```

CLAIM c3: Pooling all n₁+n₂ observations against the original Beta(α,β) prior gives the same posterior as the two-stage update.
```python
from sympy import *

th = Symbol('theta', positive=True)
alpha, beta_ = Rational(2), Rational(3)
n1, s1, n2, s2 = 5, 2, 4, 3

# one-shot: pooled likelihood against the ORIGINAL prior
prior = th**(alpha-1) * (1-th)**(beta_-1) / beta(alpha, beta_)
lik_pooled = th**(s1+s2) * (1-th)**((n1-s1)+(n2-s2))
post_pooled = simplify(prior*lik_pooled / integrate(prior*lik_pooled, (th, 0, 1)))

# two-stage
a1, b1 = s1 + alpha, n1 - s1 + beta_
prior2 = th**(a1-1) * (1-th)**(b1-1) / beta(a1, b1)
lik2 = th**s2 * (1-th)**(n2-s2)
post_seq = simplify(prior2*lik2 / integrate(prior2*lik2, (th, 0, 1)))

assert simplify(post_pooled - post_seq) == 0
```

CLAIM c4: The sum-of-squares decomposition Σ(xᵢ−μ)² = (n−1)s² + n(x̄−μ)².

from sympy import *

for n in (2, 3, 4, 5):
    xs = symbols('x1:%d' % (n+1), real=True)
    mu = Symbol('mu', real=True)
    xbar = Add(*xs)/n
    s2 = Add(*[(x - xbar)**2 for x in xs]) / (n - 1)

    rhs = (n - 1)*s2 + n*(xbar - mu)**2
