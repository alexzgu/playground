CLAIM c1: Likelihood times prior $1/\sigma^2$ gives kernel $(1/\sigma^2)^{n/2+1}\exp\{-\frac{1}{2\sigma^2}[(n-1)s^2+n(\bar y-\mu)^2]\}$; equivalently $\sum(y_i-\mu)^2=(n-1)s^2+n(\bar y-\mu)^2$.
```python
from sympy import *

mu, sig2 = symbols('mu sigma2', positive=True)

for n in (2, 3, 4, 5):
    ys = symbols('y1:%d' % (n + 1), real=True)
    ybar = Rational(1, n) * sum(ys)
    s2 = Rational(1, n - 1) * sum((y - ybar) ** 2 for y in ys)
    # sum of squares identity
    assert simplify(sum((y - mu) ** 2 for y in ys) - ((n - 1) * s2 + n * (ybar - mu) ** 2)) == 0

    # full kernel: N likelihood * prior 1/sigma^2
    lik = prod([(2 * pi * sig2) ** Rational(-1, 2) * exp(-(y - mu) ** 2 / (2 * sig2)) for y in ys])
    post = lik * (1 / sig2)
    kernel = (1 / sig2) ** (Rational(n, 2) + 1) * exp(-((n - 1) * s2 + n * (ybar - mu) ** 2) / (2 * sig2))
    ratio = simplify(post / kernel)
    # ratio must be a constant free of mu, sigma2 and the data
    assert ratio.free_symbols == set(), ratio
    assert simplify(ratio - (2 * pi) ** Rational(-n, 2)) == 0
```

CLAIM c2: As a function of $\sigma^2$ the joint kernel is $IG(n/2, B/2)$ with $B=(n-1)s^2+n(\bar y-\mu)^2$, and $IG(n/2,B/2)$ coincides with the scaled inverse chi-square $I\chi^2(n,B)$.
```python
from sympy import *

x, B, nu = symbols('x B nu', positive=True)  # x = sigma^2, B = (n-1)s^2 + n(ybar-mu)^2
n = nu

joint_kernel = (1 / x) ** (n / 2 + 1) * exp(-B / (2 * x))

# Inverse-Gamma(a, b) pdf
a, b = n / 2, B / 2
ig = b ** a / gamma(a) * x ** (-a - 1) * exp(-b / x)

ratio = simplify(joint_kernel / ig)
assert x not in ratio.free_symbols          # kernel matches IG up to a constant in x
assert simplify(ratio - gamma(n / 2) / (B / 2) ** (n / 2)) == 0

# Scaled inverse chi-square Ichi2(nu, nu*tau^2 = B):
# pdf = (nu*tau^2/2)^{nu/2}/Gamma(nu/2) * x^{-(1+nu/2)} * exp(-nu*tau^2/(2x))
ichi2 = (B / 2) ** (nu / 2) / gamma(nu / 2) * x ** (-(1 + nu / 2)) * exp(-B / (2 * x))
assert simplify(ichi2 - ig) == 0
```

CLAIM c3: $\exp\{-\frac{n}{2\sigma^2}(\mu-\bar y)^2\}$ is the kernel of $N(\bar y, \sigma^2/n)$.
```python
from sympy import *

mu, sig2, nn = symbols('mu sigma2 n', positive=True)
ybar = Symbol('ybar', real=True)

kernel = exp(-nn / (2 * sig2) * (mu - ybar) ** 2)
var = sig2 / nn
normal = 1 / sqrt(2 * pi * var) * exp(-(mu - ybar) ** 2 / (2 * var))

ratio = simplify(normal / kernel)
assert mu not in ratio.free_symbols
assert simplify(ratio - sqrt(nn / (2 * pi * sig2))) == 0

# and it integrates to 1 over mu with that constant
assert simplify(integrate(normal, (mu, -oo, oo)) - 1) == 0
```
