CLAIM c1: $\chi^2_v$ density equals the $\text{Gamma}(v/2,\ \text{rate}\ 1/2)$ density.

```python
from sympy import *

v, x = symbols('v x', positive=True)

chi2 = x**(v/2 - 1) * exp(-x/2) / (2**(v/2) * gamma(v/2))
# Gamma(shape = v/2, rate = 1/2):  rate**shape / Gamma(shape) * x**(shape-1) * exp(-rate*x)
gam = Rational(1, 2)**(v/2) / gamma(v/2) * x**(v/2 - 1) * exp(-x/2)

assert simplify(chi2 - gam) == 0
```

CLAIM c2: the normal/chi-square scale mixture $\theta\mid\delta^2\sim N(\hat\theta,\delta^2\hat\sigma^2)$, $v/\delta^2\sim\chi^2_v$ has marginal equal to the printed $t_v$ density $\frac{\Gamma(\frac{v+1}{2})}{\Gamma(\frac{v}{2})\sqrt{\pi v}\,\hat\sigma}\left[1+\frac{1}{v}\left(\frac{\theta-\hat\theta}{\hat\sigma}\right)^2\right]^{-\frac{v+1}{2}}$ (exact check at $v=3$, $\hat\theta=0$, $\hat\sigma=1$).
```python
from sympy import *

u = symbols('u', positive=True)
th = symbols('theta', real=True)

v = Integer(3)

# theta | u ~ N(0, v/u)   since delta^2 = v/u and sigmahat = 1
normal = sqrt(u / (2 * pi * v)) * exp(-u * th**2 / (2 * v))
# u ~ chi^2_v
chi = u**(v/2 - 1) * exp(-u/2) / (2**(v/2) * gamma(v/2))

mix = integrate(normal * chi, (u, 0, oo))

t_density = gamma((v + 1) / 2) / (gamma(v / 2) * sqrt(pi * v)) * (1 + th**2 / v)**(-(v + 1) / 2)

assert simplify(mix - t_density) == 0
```

CLAIM c3: the multivariate normal/chi-square scale mixture $\theta\mid\delta^2\sim N_p(\hat\theta,\delta^2\hat\Sigma)$, $v/\delta^2\sim\chi^2_v$ has marginal equal to the printed multivariate $t$ density $\frac{\Gamma(\frac{v+p}{2})}{\Gamma(\frac{v}{2})|\pi v\hat\Sigma|^{1/2}}\left[1+\frac{Q}{v}\right]^{-\frac{v+p}{2}}$ with $Q=(\theta-\hat\theta)'\hat\Sigma^{-1}(\theta-\hat\theta)$ (exact check at $p=2$, $v=3$, $\hat\Sigma=I_2$).
```python
from sympy import *

u = symbols('u', positive=True)
q = symbols('q', positive=True)  # the quadratic form Q = (theta-thetahat)' Sigmahat^{-1} (theta-thetahat)

v = Integer(3)
p = Integer(2)
Sigma = eye(2)

# theta | u ~ N_p(thetahat, (v/u) * Sigma):  density at Q = q
normal_p = (u / (2 * pi * v))**(p / 2) * det(Sigma)**Rational(-1, 2) * exp(-u * q / (2 * v))
chi = u**(v/2 - 1) * exp(-u/2) / (2**(v/2) * gamma(v/2))

mix = integrate(normal_p * chi, (u, 0, oo))

# printed density:  |pi*v*Sigmahat|^(1/2) = det(pi*v*Sigma)**(1/2)
const = gamma((v + p) / 2) / (gamma(v / 2) * sqrt(det(pi * v * Sigma)))
printed = const * (1 + q / v)**(-(v + p) / 2)

assert simplify(mix - printed) == 0

# the printed normalizing constant is the right one: integrate over R^2 in polar
# coordinates (Sigma = I so Q = r^2, area element r dr dtheta -> factor 2*pi*r)
r = symbols('r', positive=True)
total = integrate(2 * pi * r * printed.subs(q, r**2), (r, 0, oo))
assert simplify(total - 1) == 0

# numeric cross-check of the normalization
from scipy.integrate import dblquad
import math
c = float(const)
f = lambda y, x: c * (1 + (x*x + y*y) / 3.0)**(-2.5)
val, err = dblquad(f, -40, 40, lambda x: -40, lambda x: 40)
assert abs(val - 1) < 1e-3   # heavy tails; finite box captures mass to ~1e-3
```
