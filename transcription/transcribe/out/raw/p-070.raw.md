# Chapter 8 — BAYESIAN INFERENCE

**8.1 Interval Estimation**

It is usually difficult to present the entire posterior density. Typically a measure of center and a measure of spread (e.g. posterior mean and standard deviation) are used. But this is somewhat non-Bayesian at least for asymmetric posterior densities. Why?

Suppose the posterior density for $\theta$ is the following:

$$\theta|y \quad \sim \quad \mathcal{N}\big(\mathbb{E}(\theta|y), (\,\text{std}(\theta|y))^2\big),$$

then a very good 95% *credible* interval for $\theta$ is

$$\mathbb{E}(\theta|y) \pm 1.96\,\text{std}(\theta|y).$$

> ⏳ VERIFY[c1]: For $\theta\mid y\sim\mathcal N(\mu,\sigma^2)$, the interval $\mu\pm1.96\sigma$ has posterior probability content $0.95$ to the printed precision.

For Basyesians [sic] the word *credible* replaces the word *confidence*. We talk about a 95% *credible interval* instead of a 95% *confidence interval*.

`Note:` Throughout we assume that all posterior densities are continuous.

**8.1.1 Credible Interval**

**Definition.** Let $0 \le \alpha \le 1$, typically $\alpha = 0.05$, an interval $(a,b)$ is called a $100(1-\alpha)\%$ *credible interval* if its posterior probability content is $1-\alpha$, i.e.

$$\int_a^b p(\theta|y)d\theta = 1 - \alpha.$$

See Figure 1.

`Remark (1):` Credible intervals are not unique.
`Remark (2):` Credible intervals are easy to construct.

Typically we use credible interval with equal tail probabilities to their left and their right. For a $100(1-\alpha)\%$ credible interval, there is $\frac{\alpha}{2}100\%$ probability in each tail.
See Figure 2.

There are two ways to construct credible intervals.

===CLAIMS===

CLAIM c1: For a normal posterior, $\mu \pm 1.96\,\sigma$ carries probability 0.95 to the printed precision (i.e. $\Phi(1.96)-\Phi(-1.96)=0.9500$ to 4 decimals), and 1.96 is the correct rounded 0.975 quantile.
```python
from sympy import Rational, erf, sqrt, nsimplify, N, Abs

# Standard normal CDF via erf; interval content for mu +- 1.96 sigma is scale-free.
z = Rational(196, 100)
content = erf(z / sqrt(2))          # Phi(z) - Phi(-z)

# The booklet asserts 95% content at the printed precision (2 dp on 1.96 -> 4 dp on 0.9500).
assert round(float(N(content, 20)), 4) == 0.9500

# 1.96 must be the 0.975 normal quantile rounded to 2 decimals:
# check erf-based content brackets 0.95 tighter than either neighbor 1.95 / 1.97.
c_lo = erf(Rational(195, 100) / sqrt(2))
c_hi = erf(Rational(197, 100) / sqrt(2))
target = Rational(95, 100)
assert Abs(N(content - target)) < Abs(N(c_lo - target))
assert Abs(N(content - target)) < Abs(N(c_hi - target))

# And the exact content is strictly greater than 0.95 (1.96 slightly over-covers), a decisive sign check.
assert N(content - target) > 0
```

CLAIM c2: The definition's normalization is consistent: for the normal posterior of the page, $\int_a^b p(\theta\mid y)\,d\theta = 1-\alpha$ with $\alpha=0.05$, $a=\mu-1.96\sigma$, $b=\mu+1.96\sigma$, and the equal-tail split puts $\alpha/2$ in each tail.
```python
from sympy import symbols, exp, sqrt, pi, integrate, oo, Rational, simplify, N

theta = symbols('theta', real=True)
mu = symbols('mu', real=True)
sigma = symbols('sigma', positive=True)

p = exp(-(theta - mu)**2 / (2*sigma**2)) / (sigma*sqrt(2*pi))

# Density integrates to 1 over the whole line (so 1-alpha content is well posed).
assert simplify(integrate(p, (theta, -oo, oo)) - 1) == 0

# Symmetric tails: each tail beyond mu +- 1.96 sigma has equal mass.
z = Rational(196, 100)
a = mu - z*sigma
b = mu + z*sigma
left  = integrate(p, (theta, -oo, a))
right = integrate(p, (theta, b, oo))
assert simplify(left - right) == 0

# Central content equals 1 - (left + right), and rounds to 1 - alpha = 0.95.
center = integrate(p, (theta, a, b))
assert simplify(center - (1 - left - right)) == 0
assert round(float(N(center.subs({mu: 0, sigma: 1}), 20)), 4) == 0.9500

# Each tail carries alpha/2 = 0.025 to the printed precision.
assert round(float(N(left.subs({mu: 0, sigma: 1}), 20)), 3) == 0.025
```
