**15.3.3 Normal-means Model**

$$y_{ij}|\mu_i,\sigma^2 \overset{ind}{\sim} N(\mu_i,\sigma^2),\; i=1,\cdots,l,\; j=1,\cdots,n_i,$$
$$\mu_i|\theta,\delta^2 \overset{ind}{\sim} N(\theta,\delta^2)$$
$$\pi(\theta,\sigma^2,\delta^2),\;\text{improper prior;}$$
$$\text{may lead to improper posterior}$$

Note that $y_{ij}|\theta,\sigma^2,\delta^2 \sim N(\theta,\sigma^2+\delta^2)$.

> ⏳ VERIFY[c1]: marginalizing $\mu_i$ out of $N(\mu_i,\sigma^2)$ against $\mu_i\sim N(\theta,\delta^2)$ gives exactly $N(\theta,\sigma^2+\delta^2)$.

So $Cov(y_{ij},y_{ij'}|\theta,\sigma^2,\delta^2)=\delta^2$, $Cor(y_{ij},y_{ij'}|\theta,\sigma^2,\delta^2)=\frac{\delta^2}{\delta^2+\sigma^2}=\rho$, $0<\rho<1$

> ⏳ VERIFY[c2]: for $j\neq j'$, $Cov(y_{ij},y_{ij'})=\delta^2$ and $Cor(y_{ij},y_{ij'})=\delta^2/(\delta^2+\sigma^2)$.

and $\delta^2=\frac{\rho}{1-\rho}\sigma^2$

> ⏳ VERIFY[c3]: $\rho=\delta^2/(\delta^2+\sigma^2)$ is equivalent to $\delta^2=\frac{\rho}{1-\rho}\sigma^2$, and $\delta^2>0 \iff 0<\rho<1$.

CLAIM c1: Marginalizing $\mu \sim N(\theta,\delta^2)$ out of $y|\mu \sim N(\mu,\sigma^2)$ yields $N(\theta,\sigma^2+\delta^2)$.
```python
from sympy import *

y, mu = symbols('y mu', real=True)

def npdf(x, m, v):
    return exp(-(x - m)**2 / (2*v)) / sqrt(2*pi*v)

# fully symbolic Gaussian convolution may not close; use exact Rational parameters
for th, s2, d2 in [(Rational(1), Rational(2), Rational(3)),
                   (Rational(-5, 2), Rational(1, 3), Rational(7, 4)),
                   (Rational(0), Rational(5), Rational(1, 2))]:
    marginal = integrate(npdf(y, mu, s2) * npdf(mu, th, d2), (mu, -oo, oo))
    target = npdf(y, th, s2 + d2)
    assert simplify(marginal - target) == 0

# and the claimed variance is the sum, not e.g. a weighted combination
th, s2, d2 = Rational(1), Rational(2), Rational(3)
marginal = integrate(npdf(y, mu, s2) * npdf(mu, th, d2), (mu, -oo, oo))
mean = integrate(y * marginal, (y, -oo, oo))
var = integrate((y - mean)**2 * marginal, (y, -oo, oo))
assert simplify(mean - th) == 0
assert simplify(var - (s2 + d2)) == 0
```

CLAIM c2: For $j\neq j'$, $Cov(y_{ij},y_{ij'})=\delta^2$ and $Cor(y_{ij},y_{ij'})=\delta^2/(\delta^2+\sigma^2)$.
```python
from sympy import *

mu = symbols('mu', real=True)
theta = symbols('theta', real=True)
sig2, del2 = symbols('sigma2 delta2', positive=True)

def npdf(x, m, v):
    return exp(-(x - m)**2 / (2*v)) / sqrt(2*pi*v)

# y_ij and y_ij' are conditionally independent given mu_i, each with mean mu_i.
# Hence Cov(y_ij, y_ij') = E[E[y|mu]E[y'|mu]] - theta^2 = E[mu^2] - theta^2 = Var(mu) = delta^2.
Emu2 = integrate(mu**2 * npdf(mu, theta, del2), (mu, -oo, oo))
Emu = integrate(mu * npdf(mu, theta, del2), (mu, -oo, oo))
assert simplify(Emu - theta) == 0
cov = simplify(Emu2 - Emu**2)
assert simplify(cov - del2) == 0

# Var(y_ij) = sigma^2 + delta^2 (claim c1), so the correlation is:
var = sig2 + del2
cor = cov / sqrt(var * var)
assert simplify(cor - del2/(del2 + sig2)) == 0
```

CLAIM c3: $\rho=\delta^2/(\delta^2+\sigma^2)$ $\iff$ $\delta^2=\frac{\rho}{1-\rho}\sigma^2$, with $\delta^2>0 \iff 0<\rho<1$.
```python
from sympy import *

sig2, del2 = symbols('sigma2 delta2', positive=True)
rho = symbols('rho', positive=True)

# solve rho = del2/(del2+sig2) for del2
sols = solve(Eq(rho, del2/(del2 + sig2)), del2)
assert len(sols) == 1
assert simplify(sols[0] - rho/(1 - rho)*sig2) == 0

# round trip: substituting the booklet's delta^2 back recovers rho
back = simplify((rho/(1 - rho)*sig2) / ((rho/(1 - rho)*sig2) + sig2))
assert simplify(back - rho) == 0

# range: for sigma^2 > 0, delta^2 = rho/(1-rho)*sigma^2 is positive and finite iff 0 < rho < 1
r = Symbol('r', real=True)
expr = r/(1 - r)
assert expr.subs(r, Rational(1, 2)) > 0
assert expr.subs(r, Rational(99, 100)) > 0
assert expr.subs(r, Rational(3, 2)) < 0          # rho > 1 gives delta^2 < 0
assert expr.subs(r, Rational(-1, 2)) < 0         # rho < 0 gives delta^2 < 0
assert limit(expr, r, 1, '-') == oo              # rho -> 1 pushes delta^2 to infinity
assert expr.subs(r, 0) == 0                      # rho = 0 gives delta^2 = 0
```
