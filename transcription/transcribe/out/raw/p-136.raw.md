**12.1 Bayesian predictive $p$-value ($BPP$)**

We start with a discrepancy function

1. $\chi^2$ type

$$T(y, \theta) = \sum_{i=1}^{n} \frac{[y_i - E(y_i \mid \theta)]^2}{Var(y_i \mid \theta)}.$$

*(The booklet writes $y$ and $\theta$ with under-tildes to indicate vectors.)*

2. -2 times log likelihood

$$T(y, \theta) = -2\log(P(y \mid \theta)).$$

3. Power divergence measure
   Needs to make cells. Define $O_i$ observed cell counts. $e_i$ expected cell counts.

*[Figure: hand-drawn in the right margin beside this item — a rectangle ruled into a row of cells, with a second partial row beneath, sketching the partition of the sample space into cells.]*

$$T(y, \theta) = \frac{2}{\tau(\tau+1)} \sum_{i=1}^{k} O_i \left[ \left( \frac{O_i}{e_i} \right)^{\tau} - 1 \right]$$

where $\tau = \frac{2}{3}$ and $e_i$ are functions of $\theta$. (Aside $\sum_{i=1}^{n} \frac{(O_i - e_i)^2}{e_i} \sim \chi^2_{n-1}$)

> ⏳ VERIFY[c1]: with $\tau = \tfrac{2}{3}$ the leading constant $\frac{2}{\tau(\tau+1)}$ equals $\tfrac{9}{5}$, so $T(y,\theta) = \frac{9}{5}\sum_i O_i[(O_i/e_i)^{2/3} - 1]$.

> ⏳ VERIFY[c2]: for multinomial counts with $k$ cells, Pearson's statistic $\sum_i (O_i-e_i)^2/e_i$ has exact mean $k-1$, matching the stated $\chi^2_{n-1}$ degrees of freedom.

Suppose I can repeatedly sample my model,

$$T(y^{rep}, \theta) = \sum_{i=1}^{n} \frac{[y_i^{rep} - E(y_i \mid \theta)]^2}{Var(y_i \mid \theta)}$$

Define

$$BPP = P\left[ T(y^{rep}, \theta) \geq T(y^{obs}, \theta) \mid y^{obs} \right]$$

$$\int \int_{y^{rep}, \theta} P\left[ T(y^{rep}\theta) \geq T(y^{obs}\theta) \mid y^{obs}, y^{rep}, \theta \right] \pi(y^{rep}, \theta \mid y^{obs}) dy^{rep} d\theta$$

$\text{[sic: the commas in } T(y^{rep}\theta) \text{ and } T(y^{obs}\theta) \text{ are omitted in the booklet]}$

$$\pi(y^{rep}, \theta \mid y^{obs}) = \pi(y^{rep} \mid \theta, y^{obs}) \pi(\theta \mid y^{obs})$$

CLAIM c1: With τ = 2/3, the power-divergence leading constant 2/(τ(τ+1)) equals 9/5.
```python
from sympy import *

tau = Rational(2, 3)
const = 2 / (tau * (tau + 1))
assert simplify(const - Rational(9, 5)) == 0

# and the full statistic matches the 9/5 form for arbitrary O_i, e_i
O1, O2, e1, e2 = symbols('O1 O2 e1 e2', positive=True)
Os, es = [O1, O2], [e1, e2]
T_book = 2 / (tau * (tau + 1)) * sum(O * ((O / e) ** tau - 1) for O, e in zip(Os, es))
T_alt = Rational(9, 5) * sum(O * (O / e) ** Rational(2, 3) - O for O, e in zip(Os, es))
assert simplify(T_book - T_alt) == 0
```

CLAIM c2: For multinomial(N, p) counts with k cells, E[Σ (O_i − e_i)²/e_i] = k − 1 exactly, matching the stated χ²_{k−1} degrees of freedom.
```python
from sympy import *

N = symbols('N', positive=True)
p1, p2 = symbols('p1 p2', positive=True)
p3 = 1 - p1 - p2          # k = 3 cells, probabilities sum to 1
ps = [p1, p2, p3]

# O_i ~ Binomial(N, p_i) marginally: E[(O_i - N p_i)^2] = Var(O_i) = N p_i (1 - p_i)
expected = sum(N * p * (1 - p) / (N * p) for p in ps)
assert simplify(expected - (len(ps) - 1)) == 0
```

CLAIM c3: π(y_rep, θ | y_obs) = π(y_rep | θ, y_obs) · π(θ | y_obs) for a concrete discrete joint distribution.
```python
from sympy import *

# joint over (y_rep, theta) given a fixed y_obs; exact rationals, sums to 1
joint = {(0, 0): Rational(1, 8), (0, 1): Rational(1, 4),
         (1, 0): Rational(3, 8), (1, 1): Rational(1, 4)}
assert simplify(sum(joint.values()) - 1) == 0

thetas = {0, 1}
marg_theta = {t: sum(v for (y, tt), v in joint.items() if tt == t) for t in thetas}

for (y, t), pj in joint.items():
    cond = pj / marg_theta[t]                 # pi(y_rep | theta, y_obs)
    assert simplify(cond * marg_theta[t] - pj) == 0
```
