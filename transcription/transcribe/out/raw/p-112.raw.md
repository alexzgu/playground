### PDF page 112 (booklet page 107)

Observe that (2) and (3) do not have simple forms(i.e., we can not draw samples from them easily).

It is easy to use the grid method in (2) but (3) is more difficult because $\tau$ is unbounded.

How to draw a sample from the cpd of $\tau$?

$$\pi(\tau\mid p,\mu,y) \propto \frac{a^{\mu\tau-1}b^{(1-\mu)\tau-1}}{\left(B(\mu\tau,(1-\mu)\tau)\right)^{\ell}} \times \frac{1}{(1+\tau)^2},\ \tau>0$$

*(The booklet writes $p$ and $y$ with under-tildes to indicate vectors.)*

Make the transformation

$$\phi = \frac{\tau}{\tau+1}\ \text{ to give }\ \tau = \frac{\phi}{1-\phi},\ 0<\phi<1$$

> ⏳ VERIFY[c1]: $\phi=\tau/(\tau+1)$ inverts to $\tau=\phi/(1-\phi)$ on $0<\phi<1$.

Now

$$\frac{d\tau}{d\phi} = \frac{1}{(1-\phi)^2}$$

> ⏳ VERIFY[c2]: $\dfrac{d}{d\phi}\dfrac{\phi}{1-\phi} = \dfrac{1}{(1-\phi)^2}$.

So

$$\pi(\phi\mid p,\mu,y) \propto \frac{a^{\mu\tau-1}b^{(1-\mu)\tau-1}}{\left(B(\mu\tau,(1-\mu)\tau)\right)^{\ell}}\Big|_{\tau=\frac{\phi}{1-\phi}},\ 0<\phi<1$$

> ⏳ VERIFY[c3]: the Jacobian $d\tau/d\phi$ exactly cancels the factor $1/(1+\tau)^2$, so no extra factor of $\phi$ survives in $\pi(\phi\mid p,\mu,y)$.

The grid method can now be performed in a manner similar to that for $\pi(\mu\mid p,\tau,y)$.

**Rao-Blackwellized estimators**

There are no Rao-Blackwellized estimators for $\mu$ and $\tau$, but these are essentially nuisance parameters.

A Rao-Blackwellized estimators [sic] for $f(p_i\mid y)$ is

$$\widehat{f(p_i\mid y)} = M^{-1}\sum_{h=1}^{M} f\left(p_i\mid \mu^{(h)},\tau^{(h)},y\right)$$

where

$$f(p_i\mid\mu,\tau,y) = \frac{p_i^{\,y_i+\mu\tau-1}(1-p_i)^{\,n_i-y_i+(1-\mu)\tau-1}}{B\left(y_i+\mu\tau,\ n_i-y_i+(1-\mu)\tau\right)},\ 0<p_i<1,\ i=1,...,\ell$$

$$\text{Let } \lambda_i = \frac{n_i}{n_i+\tau},\ \hat{p}_i = \frac{y_i}{n_i},\ i=1,...,\ell$$

> ⏳ VERIFY[c4]: $f(p_i\mid\mu,\tau,y)$ is a proper Beta$(y_i+\mu\tau,\ n_i-y_i+(1-\mu)\tau)$ density (integrates to 1 on $(0,1)$).

Then, a Rao-Blackwellized estimator of $E(p_i\mid y)$ is

$$\widehat{E(p_i\mid y)} = M^{-1}\sum_{h=1}^{M}\left\{\lambda_i^{(h)}\hat{p}_i + (1-\lambda_i^{(h)})\mu^{(h)}\right\}$$

> ⏳ VERIFY[c5]: the Beta mean satisfies $\dfrac{y_i+\mu\tau}{n_i+\tau} = \lambda_i\hat{p}_i + (1-\lambda_i)\mu$ with $\lambda_i=n_i/(n_i+\tau)$, $\hat p_i=y_i/n_i$.

and

$$\widehat{Var(p_i\mid y)} = M^{-1}\sum_{h=1}^{M}\frac{y_i+\mu^{(h)}\tau^{(h)}}{(n_i+\tau^{(h)})^2(n_i+\tau^{(h)}+1)} + \frac{1}{M-1}\sum_{h=1}^{M}\left(t_i^{(h)}-\bar{t}_i\right)^2,$$

$\text{[sic: the numerator of the first sum should be } (y_i+\mu^{(h)}\tau^{(h)})\left(n_i-y_i+(1-\mu^{(h)})\tau^{(h)}\right)\text{ — the second Beta parameter is dropped in the printed formula]}$

> ⏳ VERIFY[c6]: the conditional Beta variance is $\dfrac{(y_i+\mu\tau)\left(n_i-y_i+(1-\mu)\tau\right)}{(n_i+\tau)^2(n_i+\tau+1)}$, and the printed numerator $y_i+\mu\tau$ alone does not equal it.

where

$$t_i^{(h)} = \lambda_i^{(h)}\hat{p}_i + (1-\lambda_i^{(h)})\mu^{(h)}\ \text{ and }\ \bar{t}_i = M^{-1}\sum_{h=1}^{M}t_i^{(h)}$$

CLAIM c1: The transformation $\phi=\tau/(\tau+1)$ inverts to $\tau=\phi/(1-\phi)$ for $0<\phi<1$, mapping onto $\tau>0$.
```python
from sympy import *

tau = symbols('tau', positive=True)
phi = symbols('phi', positive=True)

phi_of_tau = tau/(tau + 1)
tau_of_phi = phi/(1 - phi)

# round trip both ways
assert simplify(phi_of_tau.subs(tau, tau_of_phi) - phi) == 0
assert simplify(tau_of_phi.subs(phi, phi_of_tau) - tau) == 0

# the map sends tau > 0 onto 0 < phi < 1
assert limit(phi_of_tau, tau, 0, '+') == 0
assert limit(phi_of_tau, tau, oo) == 1
assert simplify(diff(phi_of_tau, tau)) == 1/(tau + 1)**2  # strictly increasing
```

CLAIM c2: $\dfrac{d\tau}{d\phi} = \dfrac{1}{(1-\phi)^2}$ where $\tau=\phi/(1-\phi)$.
```python
from sympy import *

phi = symbols('phi', positive=True)
tau_of_phi = phi/(1 - phi)

assert simplify(diff(tau_of_phi, phi) - 1/(1 - phi)**2) == 0
```

CLAIM c3: Under $\tau=\phi/(1-\phi)$ the Jacobian exactly cancels $1/(1+\tau)^2$, so $\pi(\phi\mid\cdot)$ is the $\tau$-kernel with $\tau$ substituted and no leftover $\phi$ factor.
```python
from sympy import *

phi = symbols('phi', positive=True)
tau, mu, a, b, ell = symbols('tau mu a b ell', positive=True)

tau_of_phi = phi/(1 - phi)

# pi(tau | .) propto kernel(tau) * 1/(1+tau)^2
kernel = a**(mu*tau - 1) * b**((1 - mu)*tau - 1) / beta(mu*tau, (1 - mu)*tau)**ell
pi_tau = kernel / (1 + tau)**2

# change of variables: pi(phi) = pi(tau(phi)) * |d tau / d phi|
pi_phi = pi_tau.subs(tau, tau_of_phi) * diff(tau_of_phi, phi)

# the booklet's claimed result: just the kernel evaluated at tau = phi/(1-phi)
claimed = kernel.subs(tau, tau_of_phi)

assert simplify(pi_phi - claimed) == 0

# the cancellation itself, stated bare
assert simplify((1/(1 + tau_of_phi)**2) * diff(tau_of_phi, phi) - 1) == 0
```

CLAIM c4: $f(p_i\mid\mu,\tau,y)$ is a normalized Beta$(y_i+\mu\tau,\ n_i-y_i+(1-\mu)\tau)$ density.
```python
from sympy import *

p = symbols('p', positive=True)

# symbolic shape parameters, then a decisive exact numeric substitution
def dens(alpha, bet):
    B = beta(alpha, bet).rewrite(gamma)   # force evaluation of the Beta function
    return p**(alpha - 1) * (1 - p)**(bet - 1) / B

# exact rational instance: y=2, n=5, mu=1/3, tau=6  ->  alpha=4, bet=7
y_i, n_i, mu, tau = Integer(2), Integer(5), Rational(1, 3), Integer(6)
alpha = y_i + mu*tau
bet = n_i - y_i + (1 - mu)*tau
assert alpha == 4 and bet == 7
assert simplify(integrate(dens(alpha, bet), (p, 0, 1)) - 1) == 0

# a second instance with non-integer shapes: y=1, n=4, mu=1/2, tau=3
y_i, n_i, mu, tau = Integer(1), Integer(4), Rational(1, 2), Integer(3)
alpha = y_i + mu*tau
bet = n_i - y_i + (1 - mu)*tau
assert alpha == Rational(5, 2) and bet == Rational(9, 2)
assert simplify(integrate(dens(alpha, bet), (p, 0, 1)) - 1) == 0

# shapes always sum to n_i + tau
n, t, m = symbols('n t m', positive=True)
yy = symbols('yy', positive=True)
assert simplify((yy + m*t) + (n - yy + (1 - m)*t) - (n + t)) == 0
```

CLAIM c5: The Beta mean $(y_i+\mu\tau)/(n_i+\tau)$ equals $\lambda_i\hat p_i+(1-\lambda_i)\mu$ with $\lambda_i=n_i/(n_i+\tau)$, $\hat p_i=y_i/n_i$.
```python
from sympy import *

n_i, tau = symbols('n_i tau', positive=True)
y_i, mu = symbols('y_i mu', positive=True)

alpha = y_i + mu*tau
bet = n_i - y_i + (1 - mu)*tau
beta_mean = alpha/(alpha + bet)

lam = n_i/(n_i + tau)
p_hat = y_i/n_i
shrinkage = lam*p_hat + (1 - lam)*mu

assert simplify(beta_mean - (y_i + mu*tau)/(n_i + tau)) == 0
assert simplify(beta_mean - shrinkage) == 0
```

CLAIM c6: The conditional Beta variance is $(y_i+\mu\tau)(n_i-y_i+(1-\mu)\tau)/[(n_i+\tau)^2(n_i+\tau+1)]$; the printed numerator $y_i+\mu\tau$ alone is not equal to it.
```python
from sympy import *

n_i, tau = symbols('n_i tau', positive=True)
y_i, mu = symbols('y_i mu', positive=True)

alpha = y_i + mu*tau
bet = n_i - y_i + (1 - mu)*tau
s = alpha + bet  # = n_i + tau

beta_var = alpha*bet/(s**2*(s + 1))

correct = (y_i + mu*tau)*(n_i - y_i + (1 - mu)*tau)/((n_i + tau)**2*(n_i + tau + 1))
assert simplify(beta_var - correct) == 0

# the formula as printed on the page drops the second Beta parameter
printed = (y_i + mu*tau)/((n_i + tau)**2*(n_i + tau + 1))

# decisive exact counterexample: y=2, n=5, mu=1/3, tau=6 -> alpha=4, bet=7, s=11
sub = {y_i: Integer(2), n_i: Integer(5), mu: Rational(1, 3), tau: Integer(6)}
assert correct.subs(sub) == Rational(4*7, 11**2*12) == Rational(7, 363)
assert printed.subs(sub) == Rational(4, 11**2*12) == Rational(1, 363)
assert simplify(printed.subs(sub) - correct.subs(sub)) != 0

# the printed version equals the correct one only if bet == 1, which is not generally true
assert simplify(correct - printed) != 0
```
