For a dyad with households i and j, there is a pair of dyad effects with a prior with another
covariance matrix. But this matrix is funny. Take a close look and you‚Äôll see that there is only
one standard deviation parameter, œÉd. Why? Because the labels in each dyad are arbitrary.
It isn‚Äôt meaningful which household comes first or second. So both parameters must have
the same variance. But we do want to estimate their correlation, and that is what œÅd will do
for us. If œÅd is positive, then when one household gives more within a dyad, so too does the
other. If œÅd is negative, then when one households gives more, the other gives less. If œÅd is
instead near zero, then there is no pattern within dyads.
