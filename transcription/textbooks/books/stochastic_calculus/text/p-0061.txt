It has been found that it is useful to make some additional technical
assumptions. First, we assume that F is complete under P, that is,
it contains all the nulls sets. A set A′ ⊂Ωis a null set (for P) if
A′ ⊂A for some A F with P(A) = 0. This is convenient for then
one can prove that an event has probability zero by showing that it
is contained in an event of probability zero. If F is not complete, one
can complete it by considering the collection of subsets A A′ where
A F and A′ is a null set. In the case of Lebesgue measure on the
Borel subsets of R, the completion is the set of Lebesgue measurable
sets.
